# 计算涡旋背景上Dirac算符的零模数（Callias指数）

## 零、你的角色

你是数学物理学家，精通指标定理（Atiyah-Singer、Callias、APS）。本问题是纯数学物理计算：给定一个流形和涡旋背景场，求其上Dirac算符的零模数。

## 一、物理背景（仅用于动机，不用于计算）

我们有一个SCVC（自旋-1复矢量凝聚体）介质。介质中存在涡旋环解——沿一环线的|Ψ|→0，相位绕整数圈。涡旋环在3D空间+M_vac内部空间中存在。

**目标**：计算这个涡旋背景上Dirac算符的束缚零模数。如果理论正确，应该是**3**（对应三代费米子）。

## 二、数学设定

### 流形

全空间 = ℝ³（物理空间）× M_vac（内部空间）
M_vac = (S²×S¹)/Z₂

但涡旋背景主要在ℝ³中：涡旋环是ℝ³中的1维闭环。在涡旋核心附近，横向截面是ℝ²（或紧化后是S²，切除涡旋核心后变为D²边界S¹）。

**有效简化**：在涡旋环的横向截面上，Dirac算符约化为2D涡旋背景上的2D Dirac算符。这是标准的涡旋零模计算。

### Dirac算符

3+1维（或3维欧几里得+内部空间）中的Dirac算符，在涡旋背景场Φ(x)下：

```
D_Φ = D_free + Γ(Φ)
```

其中D_free是自由Dirac算符，Γ(Φ)是涡旋场与旋量的汤川-like耦合。

**关键**：涡旋相位绕数N=1时，Callias指数定理预言束缚零模的存在。

### Callias指数定理（3+1维）

在奇数维空间中，Dirac算符的指数由**无穷远处的谱不对称性**决定：

```
index(D_Φ) = lim_{M→∞} Tr( D_Φ / √(D_Φ² + M²) )
```

对于涡旋背景，这退化为：在涡旋核心外的包围面（S²）上，计算一个2D Dirac算符的谱不对称度η。

## 三、具体计算任务

### 任务1：横向截面的2D Dirac算符

在涡旋环的横向截面上（局部ℝ²，紧化为S²），涡旋背景给出一个**磁单极-like**的规范场配置：

- 涡旋绕数N → 磁通量Φ = 2πN
- 磁场B穿过截面的积分 = 2πN

Dirac算符在S²上约化为：

```
D_S² = σ·(i∇ + A)
```

其中A是涡旋诱导的有效U(1)规范场，对应磁单极场。

### 任务2：计算Callias指数

对于这个S²上的Dirac算符：
1. **磁单极的Landau能级**：在单极场背景上，S²上的旋量本征态形成Landau能级。最低Landau能级（LLL）的简并度 = 2j+1，其中j与单极强度有关。
2. **绕数N=1**：单极强度 = 1/2（最小非零），LLL简并度 = 2×(1/2)+1 = 2。但这只是S²上的一个截面。
3. **涡旋环是闭合曲线**：整个环上有无限多个截面，但周期边界条件量子化后，只有有限个模式。涡旋环的周长~r_e，量子化条件给出模式数。

### 任务3：M_vac内部空间的贡献

M_vac = (S²×S¹)/Z₂的拓扑：
- S²的Euler示性数χ=2
- S¹的基本群π₁=ℤ
- Z₂商群修改了S¹的周期

涡旋在M_vac中的嵌入方式决定内部空间有多少截面贡献。

### 任务4：总零模数

总零模数 =（横向截面的LLL简并度）×（沿涡旋环的量子化模式数）×（M_vac内部空间的简并度）

**期望**：若有理由证明这三个因子的乘积=3，则三代费米子从Callias指数自然推出。
