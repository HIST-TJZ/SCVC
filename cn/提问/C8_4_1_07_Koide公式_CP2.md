# 计算任务：Koide公式在CP²框架中的几何起源

## 背景
Koide公式：
$$\frac{m_e + m_\mu + m_\tau}{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2} = \frac{2}{3}$$
实验值：0.66666 ± 0.00002，精度十万分之一。

旧弹性介质理论用S₃代际对称性（120°分离）解释该公式。SCVC+CP²的等距群是SU(3)，S₃不再显然。但SU(3)的Weyl群恰好是S₃——S₃是SU(3)的**离散子群**。

## 任务
从CP²上SU(3)→Weyl群S₃的自然约化，推导带电轻子的Koide公式。

## 关键观察
1. SU(3)的Weyl群是S₃（三个权重的置换群）
2. 涡旋破缺SU(3)→某子群H，H与S₃的关系？
3. 3个零模波函数在SU(3)权重空间中占据3个不同权重——这些权重的几何关系是否恰好是120°？
4. 如果Yukawa耦合正比于权重投影，$\sqrt{m_i} \propto$ 权重向量的某些分量，则：
   $$(\sqrt{m_1} + \sqrt{m_2} + \sqrt{m_3})^2 = 3(m_1+m_2+m_3)/2$$
   自然成立

## 需要计算
1. SU(3)的3个最低权重态的显式形式（对于3表示）
2. 权重向量在CP²上的几何：是否形成120°对称配置？
3. 涡旋剖面对权重简并的解除：是否保留S₃对称性？
4. 如果$\sqrt{m_i} \propto |\langle w_i|\Phi\rangle|$，推导比值结构

## 精度问题
- 旧Koide公式精度10⁻⁵——带电轻子
- 是否对夸克也成立？（需要不同Φ）
- 中微子是否有类似公式？

## 输出格式
- SU(3)权重向量→3个轻子质量的显式映射
- Koide公式的推导（从几何到代数）
- 夸克的类比公式及预测
- 需要0个或1个额外参数
