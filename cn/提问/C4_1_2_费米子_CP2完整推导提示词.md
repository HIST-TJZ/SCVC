## 二、核心问题链

### 问题1：手征费米子——9D→4D的指标定理

**这是费米子问题的根。** KK约化天然产生矢量费米子（左右对称）。SM需要手征费米子。

CP²是Kähler流形，可以承载背景U(1)通量。9D Dirac算符：
```
D̸_9 = D̸_4 ⊗ 1 + γ₅ ⊗ D̸_CP²×S¹
```

若D̸_CP²×S¹有非零指标（由背景通量F诱发），4D零模就只有一种手征性。

**Atiyah-Singer指标定理在CP²上**：
```
index(D̸_CP²) = ∫_CP² Â(TCP²) ∧ ch(F)
```

对于U(1)通量（来自S¹的Wilson线或CP²上的磁单极），计算指标：
- 通量强度n=1时，零模数=？
- 不同SU(3)表示（3, 3̄, 1, 8等）的指标各是多少？
- 能否自然产生3代？

**已知结果（请验证）**：CP²上旋量场耦合U(1)磁单极（强度k），最低Landau能级简并度为 (k+1)(k+2)/2。对k=1，有3个零模→3代。

**任务**：验证这个结果，确认它是否给出所需的手征费米子谱。

### 问题2：三代费米子

如果问题1确认指标定理给出3代，这是否同时解释了：
- 为什么是3代？（通量强度k=1的最小非平凡配置）
- 为什么3代有质量层级？（不同代的零模波函数在CP²上的局域位置不同→与涡旋场的重叠积分不同→Yukawa耦合不同）
- 为什么只有3代没有第4代？（k=2给出6代，但通量量子化可能只允许k=1）

### 问题3：费米子质量谱

电子质量已从几何推出（m_e ≈ 9.06×10⁻³¹ kg，偏差0.5%）。

**μ子和τ子**：如果3代来自CP²上不同位置的零模波函数，它们的质量由波函数与涡旋场的重叠积分决定：
```
m_i ∝ ∫_CP² ψ_i* Φ_vortex ψ_i
```
其中ψ_i是第i个零模波函数，Φ_vortex是涡旋场剖面。

**夸克质量**：夸克是SU(3)三重态。同样机制——零模波函数与涡旋的重叠积分，但波函数在SU(3)表示空间中取值。

**能否从CP²的几何和涡旋拓扑出发，给出9个费米子质量的定量预测？**
- 旧弹性理论用Koide公式+S₃对称性，3-4参数→9质量
- SCVC+CP²中，参数能否来自几何（CP²的Kähler模、通量强度、涡旋剖面）？

### 问题4：中微子质量

中微子极小质量的自然解释：跷跷板机制。
```
m_ν ∼ m_D²/M_R
```
其中m_D ∼ 轻子质量（Dirac项），M_R ∼ 重右手中微子质量 ∼ M_KK ∼ 10¹⁷⁻¹⁸ GeV。

**这给出的m_ν量级是否正确？**（观测m_ν ∼ 0.01-0.1 eV）

### 问题5：CKM混合与CP破坏

CKM矩阵来自Yukawa矩阵的对角化。若Yukawa耦合由零模波函数在CP²上的重叠积分决定，CKM角应由波函数的几何重叠决定。

**能否从CP²上零模波函数的几何重叠计算CKM角？** 旧理论有δ_CP=60°的预言——SCVC+CP²中是否保留？

### 问题6：夸克禁闭

旧弹性理论中，禁闭来自三环涡旋的Borromean互锁——三个涡旋环拓扑纠缠，无法分离。

**SCVC+CP²中，SU(3)胶子来自CP²的Killing矢量。夸克三重态如何形成？禁闭是涡旋拓扑的结果还是SU(3)规范动力学的标准结果？**
