# RG跑动验证：sin²θ_W从KK标度到M_Z

## 零、你的角色

粒子物理/RG流专家。本任务是标准模型计算：给定高能标度处的规范耦合，用RG方程跑到低能，检查是否与观测一致。

## 一、高能初始条件（来自SCVC框架的KK约化）

在KK标度 μ_KK ≈ 5.2×10¹⁷ GeV（≈M₇附近）：

```
α_U(1)  = 4(ℓ_Pl/R₁)²  ≈ 1/137.04
α_SO(3) = 6(ℓ_Pl/R)²   ≈ 1/8.93
```

对应规范耦合：
```
g₁ = √(4πα_U(1)) ≈ 0.303   (U(1), 来自S¹等距)
g₂ = √(4πα_SO(3)) ≈ 1.19   (SU(2), 来自S²等距)
```

Weinberg角在KK标度：
```
sin²θ_W(μ_KK) = g₁²/(g₁²+g₂²) ≈ 0.061
```

**问题：这个值能从μ_KK≈5.2×10¹⁷ GeV通过SM的RG跑动演化到M_Z≈91.2 GeV，达到观测值sin²θ_W(M_Z)≈0.231吗？**

## 二、计算任务

### 2.1 单圈RG方程（标准模型）

SU(2)×U(1)规范耦合的单圈β函数：

```
dg₁/d(ln μ) =  b₁·g₁³/(16π²)
dg₂/d(ln μ) =  b₂·g₂³/(16π²)
```

SM的β函数系数：
```
b₁ = 41/6  (U(1), 标准归一化)
b₂ = -19/6 (SU(2))
```

**注意U(1)归一化**：标准SM用SU(5)归一化，g_Y = √(3/5)g₁。请明确你的约定并保持自洽。

### 2.2 跑动计算

从μ_KK = 5.2×10¹⁷ GeV积分到μ = M_Z = 91.2 GeV。

输出：
1. g₁(M_Z), g₂(M_Z)的值
2. sin²θ_W(M_Z) = g₁²/(g₁²+g₂²)（或等价地用g_Y）
3. α(M_Z) = g₁²g₂²/(4π(g₁²+g₂²))（电磁耦合在M_Z的值，应为~1/128）
