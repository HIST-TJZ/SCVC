# α消去：电子质量公式的纯几何形式

## 零、前置发现（已验证，可直接使用）

两个独立计算给出同一个几何关系：

```
KK约化：       α = 4(ℓ_Pl/R₁)²
NCG谱作用量：  α = 2G_NΛ²

联立 → R₁ = √2/Λ ≈ 23.4 ℓ_Pl
```

**这不是拟合。** 两条完全不同的数学路线（经典KK vs 量子热核展开）独立收敛到同一个R₁值。

## 一、核心计算：α从电子质量公式中消去

### 旧公式（含α）

```
m_e = (π/2)^(1/3) × α × ℏ^(2/3) G_N^(-1/3) c^(-1/3) H₀^(1/3)
```

### 代入 α = 4(ℓ_Pl/R₁)² = 4ℏG_N/(c³R₁²)

```
m_e = (π/2)^(1/3) × 4ℏG_N/(c³R₁²) × ℏ^(2/3) G_N^(-1/3) c^(-1/3) H₀^(1/3)
    = 4(π/2)^(1/3) × ℏ^(5/3) G_N^(2/3) c^(-10/3) H₀^(1/3) / R₁²
```

### 数值验证

```
R₁ = 23.41 ℓ_Pl = 3.78×10⁻³⁴ m（从 α = 4(ℓ_Pl/R₁)² 反推）

m_e(旧，含α)  = 9.061×10⁻³¹ kg
m_e(新，含R₁) = 9.061×10⁻³¹ kg  ← 完全一致
m_e(实测)     = 9.109×10⁻³¹ kg
偏差 = 0.53%（来自原始质量公式本身的偏差）
```

## 二、物理含义

### α不再是"神秘的无量纲数"

```
旧视角：α ≈ 1/137 —— 为什么是这个数？不知道。
新视角：α = 4(ℓ_Pl/R₁)²，R₁ ≈ 23.4 ℓ_Pl —— S¹半径是普朗克长度的23.4倍。
```

**α测量的是普朗克尺度和内部空间尺度的比值平方。** 就像引力常数G_N测量的是普朗克质量，α测量的是S¹紧致化半径。

### 电荷不是基本量

```
e²/(4πε₀) = 4G_Nℏc/R₁²
```

电荷平方正比于引力常数×量子作用量×光速÷S¹半径平方。**电荷是内部几何的涌现量。**

### 完整输入参数

```
v32输入：c, ℏ, G_N, α（4个，含一个"魔法数字"）
v33几何版：c, ℏ, G_N, R₁（4个，全是长度/时间/质量量纲的量）
          或等价地：c, ℏ, G_N, Λ（截断标度）
```

α的"魔法"消失了——被替换为R₁这个有明确几何意义的长度。
