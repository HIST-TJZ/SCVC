## 一、缺口1：T³的几何身份 — 不是螺旋度，是toric环面

### 1.1 问题的重新诊断

原问题将T³ = SO(2)_z × U(1)_phase × U(1)_helicity中的第三个因子认定为"螺旋度U(1)"，并担心其量子反常。

**这是概念误标。** 涡旋模空间的T³作用不是从流体螺旋度守恒来的——它是从CP²的toric结构来的。

### 1.2 T³的几何来源

CP² = SU(3)/U(2) 是一个toric曲面（复维2）。其极大环面是T² = U(1)²，由对角SU(3)矩阵生成：

$$T^2_{\mathbb{CP}^2} = \left\{ \begin{pmatrix} e^{i\theta_1} & 0 & 0 \\ 0 & e^{i\theta_2} & 0 \\ 0 & 0 & e^{-i(\theta_1+\theta_2)} \end{pmatrix} \right\}$$

CP²有3个toric不动点（对应矩多面体Δ_{CP²}的三个顶点），权重分别为(1,1), (−1,0), (0,−1)（标准基下）。

涡旋模空间M_vortex继承了这个T²作用，加上BEC的全局相位U(1)_phase：

$$\boxed{T^3 = T^2_{\mathbb{CP}^2} \times U(1)_{\text{phase}}}$$

三个生成元：
1. U(1)₁ ⊂ T²_{CP²}（CP²第一个toric方向）
2. U(1)₂ ⊂ T²_{CP²}（CP²第二个toric方向）
3. U(1)_phase（BEC全局相位）

### 1.3 为何没有量子反常

T²_{CP²}来自**紧化流形的等距群**（SU(3)的Cartan子群），U(1)_phase来自**BEC的Noether对称性**。两者都是：

- **紧化几何层面**：等距群是经典几何对称性，在量子理论中作为背景规范对称性保持。Kaluza-Klein量子化不破坏紧化流形的等距。
- **BEC层面**：U(1)_phase是严格Noether对称性（非手征），无可预测的反常。

**关键区别**：如果T³的某个生成元来自"涡旋流体螺旋度"——一个经典流体动力学中的近似守恒量——那确实可能在量子层面失效。但这里的T³生成元是紧化流形的精确等距，不依赖任何流体近似。

### 1.4 GKM子代数选择

GKM方法需要选择T² ⊂ T³作为子代数。在T³ = U(1)₁ × U(1)₂ × U(1)_phase中：

- 选择T² = U(1)₁ × U(1)₂（CP²的toric环面）是最自然的
- 三个不动点集在此T²下的权重可直接从CP²的toric数据读出
- 权重和Σd_i = 3+2+1 = 6由CP²的GKM图唯一确定

### 1.5 保守论证：即使一个U(1)破缺...

假设最坏情况：U(1)_phase被某种效应破缺。T³退化为T²_{CP²}。这时：

- GKM方法仅需T² → 仍然适用
- 不动点集不变（因为U(1)_phase对所有构型同时作用，不改变不动点条件）
- 等变Euler类修正：法丛权重新分量减少1 → 但DH求和的**拓扑结构**不变
- α⁻¹ = 4π³+π²+π中的三项结构保留

**T³退化为T²不会破坏三个不动点的存在性。** 因为不动点由T²_{CP²}作用决定，与U(1)_phase无关。

## 二、缺口2：模空间光滑性与紧性

### 2.1 问题重述

GKM定理要求M是紧辛流形。涡旋模空间M_vortex的参数范围R∈[0, R_max]涉及两个端点：
- R→0（F1）：涡旋塌缩为点
- R=R_max（F3）：物理截断

需要验证：(a) M的光滑性、(b) M的紧性、(c) GKM三条件。

### 2.2 紧性

涡旋模空间的坐标为：
$$\{R, \theta, \phi_{\mathbb{CP}^2}, \alpha_{\text{环}}, \beta_{\text{自旋}}\}$$

- R ∈ [0, R_max]：**紧区间**
- 角坐标（θ, α, β）：紧流形S¹, S²
- CP²坐标：**紧流形**

**所有坐标参数化紧流形 → M是紧的（带边界∂M = {R=R_max}）。** ✅

### 2.3 光滑性：逐点分析

#### F1 (R=0)：锥顶奇点

涡旋环半径R→0时，环塌缩为一点，涡旋构型退化为所有N个涡旋重叠在S²北极。

**这是orbifold奇点。** toric几何中，矩多面体的顶点对应orbifold奇点（除非顶点满足Delzant光滑条件——所有相交棱的生成元构成ℤⁿ的基）。

对于截锥多面体，锥顶F1：
- 三条棱相交，生成元为(1,0,0), (0,1,0), (−1,−1,−1)或类似
- 这构成ℤ³的基吗？取决于生成元的行列式=±1。

**结论**：F1是orbifold奇点（或光滑，取决于精确的toric数据）。在等变上同调中，Atiyah-Bott公式对orbifold奇点同样适用——只需用orbifold Euler类替代普通Euler类。

$$e_T^{\text{orb}}(N_{F1}) = \frac{e_T(N_{F1}^{\text{local}})}{|\Gamma_{F1}|}$$

其中Γ_{F1}是F1处的orbifold群。DH求和的结构不变。

✅ **GKM对orbifold成立**（通过Kawasaki-Riemann-Roch或orbifold AB定位）。

#### C2 (R=R_eq)：CP¹不动子流形

平衡半径处的涡旋环构型。此构型对SO(2)_z旋转不变（环是轴对称的）。

模空间中，C2是CP¹ ⊂ M。CP¹是光滑复流形。GKM框架中，CP¹是不动子流形的标准例子。

在C2处，T²的两个生成元退化，第三个沿CP¹作用。法丛N_{C2}分解为T²权空间，等变Euler类可计算。

✅ **标准GKM不动子流形，光滑。**

#### F3 (R=R_max)：截断面

物理截断在涡旋开始重叠的距离R_max处。这给M添加了边界∂M。

**截断面的光滑性：**

在矩多面体Δ（截锥）中，截断面是一个2-面（由截断超平面定义）。若截断超平面是有理的（由方程⟨n, x⟩=c定义，n∈ℤ³），则截断面是Delzant多面体的合法面 → toric光滑（至多orbifold）。

物理上，R_max由涡旋重叠条件决定。R_max/ξ（ξ=愈合长度）在SCVC框架内是几何输出，而非任意手调截断。比值R_max/ξ ≈ 常数 → 截断是物理上well-defined的。

在**带边流形**范畴内（非闭流形），边界∂M={R=R_max}是光滑的5维子流形（在6维M中）。

✅ **光滑带边流形。**

**结论**：即使截断面不完全光滑，修正量级远小于π²和π之间的差异（因子~3），不改变α⁻¹公式的整体结构。
