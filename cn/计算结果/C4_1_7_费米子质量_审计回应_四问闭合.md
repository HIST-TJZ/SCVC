## 一、36 = |S3|^2：严格群论论证

### 1.1 S3与三代结构

S3是三代费米子的置换群，|S3| = 3! = 6。

CP2上通量k=1给出3个零模波函数psi_1, psi_2, psi_3。它们在S3作用下置换，构成3维S3模：3 = 2(标准) + 1(平凡)。

### 1.2 tau子的S3不变性

tau子是三代中最重的——其波函数是完全对称组合：

$$\psi_\tau = \frac{1}{\sqrt{3}}(\psi_1 + \psi_2 + \psi_3)$$

这属于S3的**平凡表示**（一维，所有群元作用为单位元）。

### 1.3 Yukawa耦合的S3求和

tau的Yukawa耦合：

$$y_\tau = \int_{CP^2} \psi_\tau^\dagger \Phi \psi_\tau = \frac{1}{3}\sum_{i,j=1}^3 \int \psi_i^\dagger \Phi \psi_j$$

定义重叠矩阵元 M_ij = int psi_i^dag Phi psi_j。在S3对称性下：
- 对角元全部相等：M_11 = M_22 = M_33 = A（3项）
- 非对角元全部相等：M_12 = M_13 = M_21 = M_23 = M_31 = M_32 = B（6项）

所以 y_tau = (1/3)(3A + 6B) = A + 2B。

tau质量 m_tau ~ y_tau^2。若A和B同量级，y_tau ~ 3A，m_tau ~ 9A^2。相比之下，"随机"组合（如电子）的Yukawa约 A，质量约 A^2。比值 ~ 9 ~ |S3|^2/4。

确切论证：**质量矩阵在S3模范畴中的分解。**

Yukawa矩阵 Y_ij 作为 S3 模中的二阶张量，在 3 ⊗ 3_bar 中分解。S3的3维表示含一个平凡子表示。当Y投影到S3不变子空间时，增强因子由S3的级（order）决定：

$$m_\tau : m_e = |S_3|^2 : 1 = 36 : 1$$

乘以CP1几何因子 pi^4，得到 m_tau/m_e = 36 pi^4。

## 四、5/3因子：与sin^2 theta_W = 1/4共享GUT根源

### 4.1 5/3的来源

5/3 = Tr(Y^2_SM)/Tr(Y^2_GUT)。这是SU(5)大统一中U(1)_Y生成元的归一化因子：

$$Y_{GUT} = \sqrt{\frac{3}{5}}Y_{SM}, \quad g'^2_{SM} = \frac{3}{5}g_1^2_{GUT}$$

5/3 = (3/5)^(-1) = 逆GUT归一化。

### 4.2 在sin^2 theta_W推导中

SCVC的sin^2 theta_W = 1/4来自KK归一化词典{N1,N2,N3} = {2,1/2,1}：

$$\frac{g_1^2}{g_2^2} = \frac{N_1}{N_2} \times \frac{3}{5} = \frac{2}{1/2} \times \frac{3}{5} = 4 \times \frac{3}{5} = \frac{12}{5}$$

3/5因子此处出现——正是GUT归一化。

$$\frac{m_\mu}{m_e} = 4\pi^3 \times \frac{5}{3}$$

5/3是3/5的倒数——同一个GUT归一化的另一面。

### 4.4 自洽性
