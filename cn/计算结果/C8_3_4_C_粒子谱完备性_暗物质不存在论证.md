M_vac = (S²×S¹)/Z₂，其中 Z₂ 作用是 (x, θ) ↦ (-x, θ+π)，x ∈ S²，θ ∈ S¹。

```

## 三、"不存在"论证的核心结构

### 3.1 论证的精髓

标准模型的问题：它不"预言"粒子谱——粒子谱是实验输入。标准模型只说"如果存在质量为m、自旋为s、表示R的粒子，则它与其他粒子的相互作用是……"——**但不解释为什么这些粒子存在而不是其他粒子。**

SCVC的不同：粒子谱**不是输入**。粒子谱由M_vac的拓扑完全决定。我们不需要问"有没有可能还有我们没有放的暗物质粒子"——因为我们没有"放"任何粒子。**所有的粒子都来自同一个源头：M_vac的同伦群。**

因此，暗物质粒子的存在性问题等价于：**M_vac的同伦群是否包含一个之前被遗漏的、对应稳定电中性粒子的拓扑扇区？**

答案：**不包含。** 如第二章所示，所有π₁、π₂、π₃扇区要么不稳定，要么带电/带磁荷。

### 3.2 为什么π₁不产生稳定暗物质：深层理由

π₁(M_vac) 的非平凡性来自 S¹ 纤维。S¹ 上的涡旋线分类由绕数 N 给出。

在 SCVC 中，**绕数 N 通过 Atiyah-Singer 指标定理与费米子零模耦合**：
```
n_F(N) = |N| · C(M_vac)
```
其中 C(M_vac) 是 M_vac 上 Dirac 算子的拓扑不变量（某种陈数或 η-不变量）。

因此：
- **N ≠ 0 → n_F ≠ 0 → 携带 SM 费米子量子数 → 带电或带色荷 → 不是暗的**
- **N = 0 → n_F = 0 → 无费米子核心 → 无量子力学稳定机制 → 不稳定**

**这个二分法是根本性的：在SCVC中，"中性"和"稳定"是互斥的。** 中性来自无费米子零模，无费米子零模意味着无稳定机制。这是框架的内在结构——不是可调参数。

### 3.3 为什么没有"额外隐藏扇区"

有人可能反驳：也许M_vac还有我们没有发现的子结构，产生额外的拓扑扇区？

这种反驳在标准模型中是合理的——标准模型是拉格朗日量层面的构造，你可以随时"加一个新场"。但在SCVC中，几何是固定的：**M_vac = (S²×S¹)/Z₂。它的同伦群是计算出来的，不是假设的。**

除非我们修改M_vac本身的拓扑（等价于修改整个框架），否则不可能出现新的拓扑扇区。

修改M_vac的拓扑意味着修改规范群、粒子谱、耦合常数——即放弃已有的所有成功预言（α⁻¹、sin²θ_W、三代费米子……）。

## 附录A：同伦群计算细节

### A.1 π₀((S²×S¹)/Z₂)

Z₂作用 (x,θ) ↦ (-x, θ+π) 没有不动点（x = -x 在 S² 上无解）。轨道空间 (S²×S¹)/Z₂ 连通。→ π₀ = 0。

### A.2 π₁((S²×S¹)/Z₂)

S²×S¹ → (S²×S¹)/Z₂ 是二重正则覆盖，deck群为 Z₂。覆盖的代数给出正合列：

```
1 → p_*(π₁(S²×S¹)) → π₁((S²×S¹)/Z₂) → Z₂ → 1
```

其中 π₁(S²×S¹) ≅ ℤ（生成元为 S¹ 上的回路 a）。Z₂ 生成元 g(x,θ) = (-x,θ+π) 在 π₁ 上的作用：g_*(a) = a（g 映射回路到自身，仅在 S² 因子上翻转基点）。因此 p_*(ℤ) ≅ ℤ 是 π₁(X) 的正规子群，指数为2。

因此：
```
π₁((S²×S¹)/Z₂) ≅ ℤ  或  π₁ ≅ ℤ × Z₂  或  π₁ ≅ ℤ ⋊ Z₂（非分裂扩张）
```

保守地，π₁至少包含 ℤ 自由部分 → 涡旋线存在。

### A.3 π₂((S²×S¹)/Z₂)

对有限覆盖 p: Y → X，高阶同伦群满足：π_n(X) ≅ π_n(Y)（n ≥ 2 时有限覆盖诱导同构）。

因此 π₂((S²×S¹)/Z₂) ≅ π₂(S²×S¹) = π₂(S²) ⊕ π₂(S¹) = ℤ ⊕ 0 = ℤ。

### A.4 π₃((S²×S¹)/Z₂)

同理，π₃((S²×S¹)/Z₂) ≅ π₃(S²×S¹) = π₃(S²) ⊕ π₃(S¹)。

π₃(S²) ≅ ℤ（Hopf不变量）。π₃(S¹) = 0。→ π₃ ≅ ℤ。

## 附录B：N↔n_F二分法的数学基础

在SCVC中，涡旋环携带的背景规范场通量为：

```

```

其中 Φ₀ 是基本通量量子。在通量为 Φ_N 的背景上，Dirac算子的零模数目由Atiyah-Singer指标定理给出：

```
ind(D̸_Φ_N) = N · ∫_{M_vac} Â(M_vac) ∧ ch(F₀)
```

其中 ch(F₀) 是基本通量丛的陈特征标。积分为整数（由M_vac拓扑保证），记为 C_M。因此：

```
n_F(N) - n_F(-N) = N · C_M

对于 M_vac = (S²×S¹)/Z₂ 上的最小耦合旋量，C_M = ±1（符号取决于手征约定）。因此 n_F(N) = |N| 个手征零模（每个手征有对应的反手征零模）。总零模数 = 2|N|。

**推论**：N = 0 ⇔ n_F = 0，N ≠ 0 ⇔ n_F ≠ 0。这是精确的当且仅当关系。
