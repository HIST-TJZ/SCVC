## 一、CP²体积模量质量

### 1.1 Kähler势与动能项

CP²的体积模量 T = χ + i a，其中 χ = Vol(CP²)/ℓ₉⁴。

4D N=1超引力框架下（CP²紧致化）：
```
K = −3 ln(T + T̄) = −3 ln(2χ)
K_TT̄ = ∂_T∂_T̄ K = 3/(4χ²)

动能项: L_kin = (3/(4χ²))[(∂χ)² + (∂a)²]

正则场: φ_χ = √(3/2) M̄_Pl ln χ
      φ_a = √(3/2) M̄_Pl · a/χ
```

### 1.2 质量计算

```
m²_φ = V''(χ_min) × (dχ/dφ)² = V''(χ_min) × (2χ²_min / 3M̄²_Pl)
```

**情况A：自然稳定化（χ_min ∼ 1, V'' ∼ M⁴_KK）**

这是"无微调"预期——CP²被稳定在KK标度：
```
m_φ = √(2/3) · M²_KK / M̄_Pl ≈ 8.4×10¹⁶ GeV ≈ 8.4×10¹³ TeV
```
**→ >> TeV。不可能用做暗物质。**

**情况B：浅极小值（Casimir + Flux近似抵消）**

若V(χ)极小值异常平坦：V'' = ε·M⁴_KK，ε≪1：
```
m²_φ = ε · (2/3) M⁴_KK / M̄²_Pl
ε = (3/2)(m²_φ M̄²_Pl / M⁴_KK)
```

为达到暗物质质量窗口所需的微调参数：

| 目标m_φ | ε | 自然性评估 |
|:---|:---:|:---|
| 10⁻²² eV (fuzzy DM) | 10⁻⁷⁴ | 荒谬 |
| 10⁻⁶ eV (QCD轴子标度) | 10⁻⁵⁸ | 极端微调 |
| 1 keV (温DM) | 10⁻³⁸ | 极端微调 |
| 1 MeV | 10⁻²⁸ | 极端微调 |
| 1 GeV | 10⁻¹⁴ | 严重微调 |
| 1 TeV | 10⁻² | ~可接受 |
| 10¹⁶ GeV (自然) | 1 | 自然 |

**结论：体积模量的自然质量是10¹⁶ GeV。任何在暗物质窗口的质量都需要指数级微调。**

### 1.3 为什么这么重？

体积模量不具有平移对称性或任何保护轻质量的对称性。它的"平坦方向"来自经典7D理论中标度不变性，但量子修正（Casimir + flux）以M_KK标度进入。没有机制自然产生浅极小值——需要Casimir和flux的精确抵消，这是一个未受任何对称性保护的巧合。

## 二、涡旋CP¹取向模量

### 2.1 身份澄清

涡旋模空间M中的CP¹因子来自**涡旋环的内部取向自由度**（与CP²内CP¹的嵌入不同，后者不存在于刚性CP²中）。度规是Fubini-Study：

```
K = ln(1 + |z|²),  ω_FS = i dz∧dz̄/(1+|z|²)²
辛体积: ∫ ω_FS = π
```

### 2.2 质量

- **若为精确NG玻色子**：m = 0 → 暗辐射，被ΔN_eff约束排除
- **若对称性破缺**：m ∼ Λ²_breaking / f_CP1。衰变常数 f_CP1 ∼ M_vortex ∼ M_KK ∼ 5×10¹⁷ GeV

| 破缺标度Λ | 达到1 keV所需的瞬子作用量S |
|:---|:---|
| Λ_QCD (200 MeV) | m ≪ 1 keV——过于轻 |
| TeV | S ∼ 110 |
| M_SUSY (10⁴ GeV) | S ∼ 100 |
| M_GUT (2×10¹⁶ GeV) | S ∼ 96 |

**→ 可调到暗物质窗口，但S ∼ 100没有独立预测。完全自由参数。**

```
T_osc = √(m M̄_Pl / (3√(π²g_*/90)))
Ω_φ h² = (½m²φ²_i / s(T_osc)) × s₀ / (ρ_crit/h²)
```

## 五、如果质量仍>>TeV — 隐藏扇区与替代方案

### 5.1 为什么"自然质量>>TeV"是框架的结构性特征

CP²模场（体积模量、轴子）没有对称性保护轻质量：

| 模场 | 保护对称性？ | 自然质量 |
|:---|:---|:---|
| 体积模量 χ | 仅经典标度不变性 → 量子破缺 | ~M_KK ~ 10¹⁷ GeV |
| 轴子 a=Im(T) | 平移对称性 → 仅受非微扰破缺 | 取决于S_inst |
| 涡旋CP¹取向 | 若为NG玻色子 → 零质量 | 0 或 取决于破缺 |
| KK模式 | 无 | ≥ M_KK ~ 10¹⁷ GeV |

**这不是CP²的特有问题——这是所有Planck/KK标度紧致化中模场的普遍问题。**弦论中同样存在"模量质量问题"（Kähler modulus stabilization problem），需要非微扰机制（如gaugino condensation）产生模量势能。CP²框架的困难是同一问题的具体体现。

### 5.2 可能的出路

**A. 放弃CP²模场 → 接受"需要额外输入"**

最诚实的路径。SM粒子以外的暗物质需要新的动力学，框架不提供。这不丢人——弦论、超对称、额外维也都没有自然解决暗物质。暗物质本质是超越SM的独立板块。

**B. 隐藏扇区**

若存在一个独立的隐藏规范群（不在CP²几何中，而是完全新的扇区）：
- 隐藏群在某个低标度Λ_hid ∼ 100 TeV处禁闭
- 最轻隐藏强子 = 暗物质候选（类"hidden valley"/"quirky DM"）
- 质量自然在TeV–PeV范围
- 不需要微调模量质量

代价：完全新的场内容 + 独立的动力学标度 → 暗物质板块与几何框架解耦

**C. 极早期产生机制**

对超重DM（m ∼ 10¹⁶ GeV），可绕过错位机制：
- **引力产生**：暴胀后的非绝热过程中，引力子交换产生DM粒子 → Ω ∝ (T_RH/M̄_Pl)³(m/T_RH)。需要T_RH ∼ m/20 ∼ 10¹⁵ GeV以得到正确遗迹密度
- **暴胀子衰变**：若体积模量=暴胀子，其衰变产物中包含稳定残余 → 完全取决于衰变分支比

**D. 非热产生**

若DM从未达到热平衡：
- Moduli-dominated宇宙：模量振荡主导能量密度，衰变时产生DM
- 正确遗迹密度需要m_mod ∼ 10–100 TeV和合适的分支比
