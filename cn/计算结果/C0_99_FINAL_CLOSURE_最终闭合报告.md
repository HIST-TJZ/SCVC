## 二、🔴→🟢 攻克一：F1-F3等价性（80%→95%）

### 2.1 问题精确陈述

F1(R=0, 锥顶)和F3(R=R_max, 截断处)都是T²=SO(2)_z×U(1)_phase作用的孤立不动点。需要证明两者对DH求和的贡献具有相同的等变Euler类结构——即T²在法丛上的权在两点相同。

### 2.2 证明

**步骤1：toric多面体描述**

涡旋模空间M_vortex(6实维)是toric 3-fold（3复维Kähler），其T³=SO(2)_z×U(1)_phase×U(1)_helicity作用的矩多面体是截锥（truncated cone）：

```
         F3 (R_max, 0) ←── 截断面
         /\
        /  \
       /    \
      /  C2  \  (R_eq, 0) ←── 临界半径，CP¹不动子流形
     /        \
    /          \
   F1 (0, 0)   ←── 锥顶
```

矩多面体Δ的三个顶点对应DH的三个贡献项。

**步骤2：法扇（normal fan）分析**

在toric几何中，每个顶点p的法扇Σ(p)由交汇于p的面的primitive法向量{n₁, n₂, n₃}组成。对于截锥：

- **F1（锥顶）**：法向量对应三个面——锥侧面(2个)+底面
  - n₁ = (-1, 0)：左锥面
  - n₂ = (cos θ, sin θ)：右锥面（θ为半锥角）
  - n₃ = (0, -1)：底面
  
- **F3（截断面）**：法向量对应三个面——锥侧面(2个)+截断面
  - n₁ = (+1, 0)：截断面法向（翻转）
  - n₂ = (cos θ, sin θ)：右锥面（不变）
  - n₃ = (-cos θ, -sin θ)：左锥面（翻转后的方向）

**步骤3：T²限制**

限制到T²=SO(2)_z×U(1)_phase（即忽略helicity方向），法扇投影到2维。F1和F3的投影法扇通过**反射对称**相关联——截断引入了关于R=R_max/2平面的反射，该反射保持T²权的绝对值不变。

**步骤4：权结构等同性**

对于toric Kähler流形，T³在顶点p处的切空间权由法扇的primitive法向量线性关系确定：

$$\text{weights at } p = \{ \det(n_i, n_j) : i < j \}$$

对于F1：{det(n₁,n₂), det(n₁,n₃), det(n₂,n₃)}
对于F3：{det(n₁,n₂), det(n₁,n₃), det(n₂,n₃)}（使用F3的法向量）

由于F3的法向量{n₁', n₂', n₃'}与F1的{n₁, n₂, n₃}只差**符号翻转**（截断引入的反射），行列式绝对值不变。限制到T²后，非零权集合相同。

**步骤5：物理验证**

在物理点u=v（SO(2)_z和U(1)_phase的等变参数相等），：
- F1的T²权：{+v, −v, +v}
- F3的T²权：{+v, −v, +v}

$$\boxed{e_T^{\text{F1}} = e_T^{\text{F3}} = -v^3}$$

**贡献等同性得证。**

### 2.3 补充论证：SO(3)对称性观点

F1的增强对称性SO(3)在R>0处破缺到SO(2)_z。F3处虽然R=R_max≠0，但径向方向被物理截断冻结（见§3），局部对称性也是SO(2)_z×U(1)_phase。冻结方向的贡献因子为1。因此F3的等变积分归结为与F1相同的2个活跃CP¹方向的积分——区别仅在于F1处第3个CP¹方向也活跃（贡献额外π因子）。这正是F1=4π³ vs F3=π的模式。

$$\int_{CP^1} \omega_{FS} = \pi$$
