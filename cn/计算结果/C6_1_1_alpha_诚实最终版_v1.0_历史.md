## 零、一句话

**α ≈ 1/137.036 是SCVC框架唯一的无量纲输入常数。** 框架的完整输入为4个常数：c, ℏ, G_N, α。其余一切物理量（电子质量、费米子质量谱、电弱混合角、τ延迟……）均由此4个输入+框架结构导出。

这使SM的~19个自由参数降至**1个无量纲参数**——削减95%。

### 1.2 两个数值巧合

尽管无法从第一原理推导，以下π组合碰巧接近1/137：

**(2/π)^11 × (π/3) = 1/137.184...**（偏离实测1049 ppm）
- (2/π) = Vol(S¹)/Vol(SO(3))，与涡旋环11D模空间的SO(3)自由度相关
- 11 = 涡旋环的DOF数（7个严格零模 + 3个S³纤维 + 1个赝呼吸模）
- π/3 = Hopf纤维化 → Z₂商群 → S₃代际划分的几何推论
- 状态：🟡 **开放猜想**——DOF因子2/π的普适性未证明

**4π³ + π² + π = 137.036...**（偏离实测2.2 ppm）
- 这一度被认为是最精确的π组合
- de-equivariantization路线试图从T²-哈密顿辛流形的不动点数据推导这三个项
- 2026-07-21严格判定：**所需6维紧辛流形数学上不存在**（ABBV局部化Σ≠0）
- 状态：🔴 **数值巧合，非推导**

### 1.3 Connes缝合的贡献（与α间接相关）

**已证明**：M_vac = (S²×S¹)/Z₂ 天然是spin流形（w₂=0），承载天然Dirac算符。

这意味着：如果走Connes NCG的谱作用量路线，M_vac可以替代Connes手工选取的有限空间F。谱作用量的热核展开中，a₄系数决定规范耦合g——从而决定α的**量级**。这是当前剩下的唯一可行的"α从几何"路线。

但预期：NCG谱作用量能给出α~1/100到1/200的范围（定性正确），不太可能精确命中137.036——Connes自己的框架也没有做到这一点。

## 二、α在框架中的实际角色

### 2.1 它是被放进来的，但放进来的方式很省

α出现的位置：

**电子质量公式**：
```
m_e = α × (π/2)^(1/3) × ℏ^(2/3) × G_N^(-1/3) × c^(-1/3) × H₀^(1/3)
```
这里的α是显式乘入的。m_e ∝ α^1 是框架的**选择**（非量纲强制）。

**r_e/a 关系**：
```
r_e / a = (2/π)^(1/3)
```
这个关系不依赖α——因为质量公式中α的指数被选为1，使得它在r_e/a中抵消。这是一个**恒等式重排**，不是独立验证。

### 2.2 什么地方不需要α

以下结果完全独立于α的数值：
- M_vac几何及spin性质
- 三代费米子的拓扑起源
- S₃代际对称性 → Koide质量公式结构
- 电弱混合角sin²θ_W（从M_vac等距群KK约化）
- τ延迟 ~0.4ms（从涡旋动力学）
- 弦引力/夸克禁闭标度

## 四、开放猜想（留给未来）

### 猜想1：(2/π)^11 × (π/3) 可能不是巧合

- 11 = 涡旋环DOF数。7个严格零模已确认，4个额外DOF有物理动机。
- 如果所有11个DOF的路径积分归一化各自贡献2/π，结果自动接近1/137。
- **判据**：需要证明那些现在标🔴的DOF（ℝ³平移、S¹相位）确实贡献2/π。这可能需要涡旋环的显式数值解。

### 猜想2：NCG谱作用量可能给出更精确的α

- 用M_vac替代Connes的F，计算Dirac算符的Seeley-deWitt系数。
- 如果a₄系数恰好给出α≈1/137，则α可以从NCG推导。
- **判据**：需要完成M_vac上Dirac算符的热核展开计算。这是繁重但标准的微分几何任务。
