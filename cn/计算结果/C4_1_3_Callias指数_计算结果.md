## 零模数 = 1 （基于现有设定；若要得3需额外结构）

## 计算路径：A（纯拓扑Callias指数）+ 部分B（半解析补充）+ C（定性论证）

本计算综合使用三条路径：先严格走Callias指数定理的拓扑部分（路径A），对无法从拓扑确定的部分做半解析估计（路径B），最终对"为何是3"做定性讨论（路径C）。

---

## 关键步骤

### 步骤1：横向截面的2D Dirac零模（Jackiw-Rossi零模）

在涡旋环的横向截面上（局部ℝ²，紧化为S²），涡旋绕数N=1诱导一个有效U(1)规范场A_μ = ∂_μ(arg Φ)。磁通量：

Φ_B = ∮ A·dx = 2πN = 2π

这等价于S²上强度g=1/2的磁单极场（Dirac量子化条件：2g ∈ ℤ，旋量取最小非零值g=1/2）。

S²上带磁单极背景的2D Dirac算符：

D_S² = σ·(i∇ + A_monopole)

其Landau能级结构（Camporesi-Higuchi, 1996; 或参考Haldane的S²上量子Hall效应）：

- 本征值：E_l = ±(l + g + 1/2)/R，其中l = 0, 1, 2, ...
- 最低Landau能级(LLL)：l = 0，简并度 = 2g + 1

当g = 1/2时：LLL简并度 = 2×(1/2) + 1 = 2

但注意：这是S²上Dirac算符的正负手征合并的简并度。对于单个手征（Weyl）分支，LLL简并度为2g+1。在涡旋问题中，Jackiw-Rossi经典结果（Phys. Rev. D 23, 1965, 1981）给出：2+1维中绕数|N|=1的涡旋恰好束缚1个正规化零模（每旋量分量）。

关键区分：S²上的LLL简并度2g+1=2是指在紧化球面上存在两个简并的LLL态。但在ℝ²的开平面上（涡旋核心的横向截面），只有一个态是正规化的（另一个对应反涡旋，或发散）。所以：

**横向LLL简并度 = 1 （Jackiw-Rossi零模，绕数|N|=1）**

此为拓扑保护的结果，不依赖涡旋剖面。Callias指数定理保证：
index(D_2D) = N （涡旋绕数），在2+1维中精确给出|N|个零模。

---

### 步骤2：沿涡旋环的量子化

涡旋环周长L = 2πR（R为环半径）。在横向截面解出零模波函数ψ_transverse(r,φ)后，完整波函数分解为：

Ψ(x, s) = ψ_transverse(r, φ) ⊗ χ(s)

其中s是沿环的弧长参数，χ(s)满足环上的1D有效Dirac方程。由于横向零模的手征性，沿环的有效理论是1+1维手征费米子：

i∂_s χ(s) = E χ(s)

周期性边界条件：χ(s+L) = e^{iδ} χ(s)，其中δ是沿环的Berry相位（来自自旋联络和可能的Aharonov-Bohm通量）。

**情况A（无额外通量，旋量周期BC）：δ = 0**

k_n = 2πn/L，E_n = ±v_F k_n = ±v_F·(2πn/L)
零能条件E=0 ⇒ n=0 ⇒ 1个零模。

**情况B（有Berry相位δ ≠ 0）：**

k_n = (2πn + δ)/L，零能条件n = -δ/2π，通常不是整数 ⇒ 无精确零模（只有间隙最小的近零模）。

**结论**：在没有穿过涡旋环的磁通量且自旋联络相位为零的理想情况下，**沿环量子化模式数 = 1**。

但此数依赖于涡旋环的全局几何（是否与M_vac中的S¹有缠绕）和可能的额外通量，非纯拓扑保护，属于"需要涡旋显式解"的范畴。

---

### 步骤3：M_vac内部空间的贡献

M_vac = (S²×S¹)/ℤ₂ 是一个紧致3维流形。ℤ₂作用为对角作用：

(x, θ) ⟼ (-x, θ + π)

其中x ∈ S²（嵌入ℝ³），θ ∈ S¹。

#### 3.1 M_vac的拓扑

- X = (S²×S¹)/ℤ₂ 是ℝP²上的非平凡S¹丛（沿ℝP²不可缩回路的S¹纤维平移半周期）
- π₁(X) ≅ ℤ ⋊ ℤ₂（或ℤ × ℤ₂，取决于ℤ₂在S¹上的作用是否与回路对易）
- X是可定向的紧致3-流形，存在自旋结构

#### 3.2 M_vac上自由Dirac算符的零模

对紧致无边界奇数维流形M^odd上的Dirac算符：

Atiyah-Singer指标定理：index(D) = ∫_M Â(M) = 0 （奇数维Â亏格为零）

但这只要求dim ker D⁺ = dim ker D⁻，不禁止两者同时非零。

**关键问题**：M_vac是否存在调和旋量（harmonic spinors，即ker D ≠ 0）？

对乘积空间S²×S¹的标准自旋结构：
- S²上Dirac谱：E_l = ±(l+1)，l = 0,1,2,...，简并度2(l+1)。最低非零本征值=±1（l=0时）
- S¹上Dirac谱：E_n = n/R_S¹，n∈ℤ
- S²×S¹上Dirac算符D = D_S²⊗1 + γ₅⊗i∂_θ
  - 傅里叶展开∂_θ → in后：D_n = D_S²⊗1 + i n γ₅
  - D_n² = D_S²² + n²（因为γ₅与D_S²反対易且γ₅²=1）
  - 本征值E_n,l = ±√((l+1)² + n²)（以S²半径为单位）
  - 零模要求(l+1)² + n² = 0 ⇒ l = -1（不存在）且n=0 ⇒ **无零模**

ℤ₂商群的作用：要求旋量满足ψ(-x, θ+π) = ±ψ(x, θ)（±取决于自旋结构选择）。

**未扭曲自旋结构（+号）**：只有ℤ₂不变的S²×S¹上的态才降为X上的态。由于S²×S¹无零模，X也无零模。

**扭曲自旋结构（-号）**：ℤ₂反不变的态降为X上的态。同样，S²×S¹无零模，X也无零模。

**结论**：**M_vac内部空间简并度 = 1**（即M_vac不贡献额外的零模简并度）。自由Dirac算符在M_vac上有质量间隙，不产生零模。

**但是**：如果M_vac上存在背景规范场（例如涡旋也在M_vac中有绕数），则可能通过Callias-like机制产生额外零模。这是获取因子3的关键可能途径——见下文的"如何得到3"讨论。

#### 3.3 η-不变量的贡献

即使M_vac没有零模，其Dirac算符的η-不变量（谱不对称度）可以对全空间的Callias指数产生贡献：

η_D(M_vac) = lim_{s→0} Σ_λ sign(λ) |λ|^{-s}

对于(S²×S¹)/ℤ₂，η_D的计算需要明确度规和自旋结构，超出了纯拓扑的范围。对于标准度规，由于谱的对称性（±成对），η_D = 0（模可能的模2指标）。这不会改变零模数。

---

### 步骤4：总零模数

总零模数 = 横向LLL简并度 × 沿环量子化模式数 × M_vac内部简并度

| 因子 | 数值 | 类型 |
|------|------|------|
| 横向LLL简并度 | 1 | 拓扑保护（Callias, N=1） |
| 沿环量子化模式数 | 1 | 半拓扑（依赖Berry相位） |
| M_vac内部简并度 | 1 | 假定无内部规范场 |
| **总零模数** | **1** | |

---

## 简并度分解

- **横向LLL简并度 = 1**（来源：Jackiw-Rossi零模定理，Callias指数定理在2+1维给出index=N=1。严格拓扑保护。）
- **沿环量子化模式数 = 1**（来源：闭合环上k=0的驻波模式，周期性边界条件。但此数依赖Berry相位δ：若δ≠0则可能为0。）
- **内部空间简并度 = 1**（来源：M_vac=(S²×S¹)/ℤ₂上自由Dirac算符有质量间隙，无调和旋量。但若M_vac中有背景场则可能非1。）
- **总零模数 = 1×1×1 = 1**

---

## 不需要涡旋显式解的部分

1. **横向LLL零模的存在性**：纯拓扑。Callias指数定理保证绕数N的涡旋必然束缚|N|个正规化零模。不依赖涡旋的径向剖面（只要渐近|Φ|→常数，相位绕数=N）。
2. **零模数是整数**：拓扑保护，连续形变下不变。
3. **横向LLL简并度的下限**：至少1个零模。不能是0（除非涡旋被破坏）。

---

## 需要涡旋显式解才能确定的部分

1. **沿环Berry相位δ**：依赖于涡旋环的曲率、挠率、自旋联络沿环的积分，以及是否有磁通穿过环面。δ ≠ 0 可能消除零模或分裂为近零模对。
2. **M_vac中是否存在背景场**：如果涡旋在M_vac的S¹因子上也有绕数，M_vac的贡献可以是大于1的整数。
3. **涡旋环与M_vac的互相缠绕**：涡旋环若在M_vac的S¹方向上也有非平凡拓扑，总绕数可能是多个因子的乘积。

---

## 置信度：中

理由：

- ✅ **横向LLL = 1**：**高置信度**。这是Jackiw-Rossi和Callias指数定理的标准结果，有40+年的文献支撑。
- ⚠️ **沿环模式数 = 1**：**中置信度**。基本物理图像清晰（手征费米子在环上的驻波），但Berry相位δ的具体值未知。若δ=π，则零模消失。
- ⚠️ **M_vac内部 = 1**：**中置信度**。纯拓扑分析给出1，但M_vac上可能存在的背景场或边界条件改变此数。自由Dirac算符在紧致奇数维流形上无零模是generic结论。
- ⚠️ **总乘积 = 1**：**中置信度**。每个因子的不确定性累积。

## 如何得到3？（理论构造，非必然结论）

若目标是得到零模数=3（对应三代费米子），需要以下机制之一：

### 可能性1：涡旋绕数N=3
三个同轴的N=1涡旋环（或一个N=3的涡旋）。横向LLL简并度变为3。这是最简单粗暴的方式，但与"涡旋绕数N=1"的设定矛盾。

### 可能性2：M_vac上存在磁单极规范场
在M_vac的S²因子上放置强度g'=1的磁单极。此时M_vac上Dirac算符的最低Landau能级简并度=2g'+1=3。总零模数=1×1×3=3。这需要额外的物理输入（M_vac中为何有磁单极？），但框架内部自洽。

### 可能性3：涡旋环与M_vac的S¹有非平凡缠绕
若涡旋环在"ℝ³×M_vac"的S¹方向上也绕了一圈（即涡旋环在M_vac的S¹因子上有绕数w=3），有效绕数变为1×3=3，总零模数=3。这需要指定涡旋在内部空间的具体嵌入方式。

### 可能性4：三代来自横向LLL的2g+1=2与沿环的额外简并
g=1/2时LLL=2（S²紧化下的手征简并度），加上沿环的扭结边界条件产生1.5的"分数"贡献 → 2×1.5=3。但这个论证非常牵强，1.5不是一个自然的拓扑整数。

### 可能性5：M_vac = (S²×S¹)/ℤ₂的η-不变量=3
如果选择ℤ₂扭曲自旋结构且M_vac存在引力瞬子背景，η-不变量可以是非零整数。但这需要极特殊的几何设定，非generic结论。

---

## 总结

在给定的设定下（涡旋绕数N=1，M_vac无额外背景场），**拓扑强制的零模数是1**。要得到3，必须引入额外结构——最自然的候选是M_vac内部的背景规范场（如S²上的磁单极）或涡旋在M_vac中的非平凡缠绕。纯从ℝ³涡旋+自由M_vac出发，3不是一个"自然"的拓扑输出值。

这个结果与Atiyah-Singer指标定理的精神一致：零模数由底层几何和拓扑不变量决定，不能通过调整参数"调节"到任意整数。
