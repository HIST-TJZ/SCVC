| 不动点 | DH贡献 | 权重 |

## 二、几何公式的发现

### 2.1 μ子质量

$$\frac{m_\mu}{m_e} = 4\pi^3 \times \frac{5}{3}$$

**几何要素：**
- **4π³ = F1**：DH求和的最大项，来自锥顶(R=0)处SO(3)增强对称性的不动点
- **5/3**：GUT归一化因子 = Tr(Y²_SM)/Tr(Y²_GUT) = (10/3)/2

**物理含义**：μ子是"第二代"——对应F1的权重优势（F1贡献被F3部分抵消，但净效果由5/3群论因子确定）。

### 2.2 τ子质量

$$\frac{m_\tau}{m_e} = 36\pi^4 = (6\pi^2)^2$$

**几何要素：**
- **36 = 6² = (3!)²**：三代费米子的置换群S₃的阶的平方
- **π⁴ = (π²)² = C2²**：CP¹不动子流形DH贡献的平方，代表"临界涡旋环"
- **6π² = 3! × C2**：三代置换 × CP¹几何

**物理含义**：τ子是"第三代"——质量获得全部三代置换对称性(3!)和CP¹几何(C2)的平方增强。

### 2.3 电子质量（绝对标度）

电子质量已在框架中独立推导：

$$m_e = \alpha \times \left(\frac{\pi}{2}\right)^{1/3} \times H_0^{1/3}$$

使用α=1/(4π³+π²+π)和H₀，偏差<1%。电子是质量标度的"锚"。

### 2.4 平方根形式（最简洁）

$$\boxed{\sqrt{\frac{m_\mu}{m_e}} = 2\pi^{3/2}\sqrt{\frac{5}{3}}, \quad \sqrt{\frac{m_\tau}{m_e}} = 6\pi^2}$$

### 3.2 Koide自洽性

Koide公式在几何公式下的检验：

$$\frac{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2}{m_e + m_\mu + m_\tau} = \frac{(1+14.377+59.218)^2}{1+206.71+3507} = \frac{5564.4}{3714.4} = 1.4981$$

目标值：3/2 = 1.5000。偏差：**-0.13%**。

Koide不是精确成立——是**近似成立**（0.13%精度）。这与实验一致：Koide公式本身在实验精度内精确，但RG跑动会轻微破坏它。几何公式在M_KK标度给出Koide比值≈1.498，RG跑动后变为实验的1.500。
