# α_s RG 跑动：KK 标度 → M_Z 验证

## 背景

SCVC 框架中，强力 SU(3) 来自 CP² 的等距群。在 KK 标度 (M_KK ≈ 5.2×10¹⁷ GeV) 处，α_s 的初始值由 CP²×S¹ 的 KK 约化给出：

$$\alpha_s(M_{KK}) \approx \frac{32}{3\pi^3} \cdot \frac{1}{M_9^7 R_{CP}^6 R_{S^1}}$$

数值约为 **1/25 ~ 1/35**，中心值取 1/30。M₉/M₄ ≈ 0.35。

---

## 计算方法

### β 函数系数 (SM, MS-bar 方案)

| Nf | b₀ = 11 − 2Nf/3 | b₁ = 102 − 38Nf/3 |
|----|-----------------|-------------------|
| 6  | 7               | 26                |
| 5  | 23/3 ≈ 7.667    | 116/3 ≈ 38.667    |
| 4  | 25/3 ≈ 8.333    | 154/3 ≈ 51.333    |
| 3  | 9               | 64                |

### 夸克阈值 (MS-bar 质量)

| 夸克 | m(m) [GeV] |
|------|-----------|
| top  | 163.0     |
| bot  | 4.18      |
| chm  | 1.27      |

### 方法

- **1-圈**：解析 RG 方程，逐段跨越夸克阈值：`1/α(μ₂) = 1/α(μ₁) − Σ b₀ᵢ/(2π) ln(μᵢ/μᵢ₊₁)`
- **2-圈**：基于 1-圈结果的迭代修正，使用近似公式 `Δ(1/α) ≈ (b₁/(4πb₀)) ln(α_end/α_start)` 在各段累积
- **阈值匹配**：LO 连续匹配 `α_s^(Nf−1)(m_q) = α_s^(Nf)(m_q)`。NLO 修正 ~ (α_s/π)² ~ 0.1%，可忽略

### 能标关系

```
M_KK = 5.2×10¹⁷ GeV  (Nf=6)
  ↓
m_top = 163 GeV      (Nf=6 → 5，仅此一个阈值跨越)
  ↓
M_Z = 91.2 GeV       (Nf=5)
  ↓
m_bot = 4.18 GeV     (Nf=5 → 4，低于 M_Z，不影响本次计算)
  ↓
m_chm = 1.27 GeV     (Nf=4 → 3)
```

## 第二部分：SCVC α_s(M_KK) = 1/30 → M_Z

直接使用 SCVC 预言的中心值 **α_s(M_KK) = 1/30 = 0.033333** 向低能跑动：

**结果：LANDAU 极点！**

- Landau 极点位置：**μ ≈ 1.05 × 10⁶ GeV**
- M_Z = 91.2 GeV 远在此之下 → **无法到达 M_Z！**

物理原因：α_s(M_KK) = 1/30 比 SM 要求值 (1/49) 大约 1.6 倍。SM 是渐近自由的 (b₀ > 0)，向低能跑动时 α_s 增大。初始耦合过强导致 α_s 在 ~10⁶ GeV 处发散。

## 第六部分：三个关键物理问题

### Q1：框架预言能否在 ±5% 内命中 α_s(M_Z)？

**答：不能。**

SCVC 预言的 α_s(M_KK) 范围 (1/25 ~ 1/35) 全部在到达 M_Z 之前遇到 Landau 极点：

| α_s(M_KK) | Landau 极点位置 | 能否到达 M_Z？ |
|-----------|----------------|---------------|
| 1/25 = 0.040 | 9.3×10⁷ GeV | **否** |
| 1/30 = 0.033 | 1.1×10⁶ GeV | **否** |
| 1/35 = 0.029 | 1.2×10⁴ GeV | **否** |

SM 要求 α_s(M_KK) ≈ 1/49 = 0.0204。SCVC 预言值约大了 60%。

### Q2：是否需要假设 MSSM-like 阈值修正？

**答：MSSM 阈值修正方向正确，可能解决问题。**

MSSM 在 M_KK 附近引入超对称伙伴子（gauginos, Higgsinos 等），等效于增加有效味道数，从而**降低有效 b₀**：

- SM b₀(Nf=6) = **7**
- 全 MSSM b₀ ≈ **3**

用 b₀ = 3 在 Nf=6 区域做 1-圈估计：

$$\frac{1}{\alpha_s(M_Z)} = 30 - \frac{3}{2\pi}\ln\frac{M_{KK}}{m_t} - \frac{23/3}{2\pi}\ln\frac{m_t}{M_Z}$$

→ **α_s(M_Z) ≈ 0.0817**（偏差 −30.9%）

虽然偏小，但**避免了 Landau 极点**！这说明 MSSM 方向的修正可以解决耦合发散问题。结合 KK 标度处的额外阈值效应和对 b₁ 的修正，有可能完全协调 SCVC 预言与实验。

### Q3：Nf=6 的顶夸克阈值如何处理？

**答：M_Z 处 Nf=5，仅顶夸克阈值相关。**

- M_Z = 91.2 GeV < m_top = 163 GeV → M_Z 处为 **Nf=5**
- 从 M_KK 到 M_Z 仅跨越 **一个夸克阈值 (m_top)**
- LO 匹配：`α_s^(5)(m_top) = α_s^(6)(m_top)`（连续性条件）
- NLO 阈值修正量级为 (α_s/π)² ~ 0.1%，对于本次 ±5% 精度的检验**可忽略**
- b 和 c 夸克阈值在 M_Z 以下 → 不影响本次计算

## 第七部分：协调方案

### 若固定 α_s(M_KK) = 1/30 不变

则通过 1-圈 RG 方程反推：

$$\ln M_{KK} = \frac{2\pi}{b_0(6)}\left[\frac{1}{\alpha_s(M_{KK})} - \frac{1}{\alpha_s(M_Z)} - \frac{b_0(5)}{2\pi}\ln\frac{m_t}{M_Z}\right] + \ln m_t$$

→ **隐含 M_KK ≈ 2.1 × 10¹⁰ GeV**

比名义值 (5.2×10¹⁷ GeV) 低约 **7 个数量级**！

### 可能的协调路径

1. **MSSM-like 阈值修正**（b₀ 从 7→3）：方向正确，已证明可避免 Landau 极点
2. **非标准 RG 跑动**：额外维度可能修改有效 β 函数
3. **SCVC 对 α_s(M_KK) 的预言需要修正**：当前推导可能需要重新审查
4. **M_KK 标度可能低于名义值**：若 M_KK ~ 10¹⁰ GeV，则与实验一致
