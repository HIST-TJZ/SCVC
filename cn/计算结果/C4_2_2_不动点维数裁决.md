# 不动点维数矛盾裁决：涡旋环模空间 T² 局部化

**角色**：微分几何学家 | **日期**：2026-07-20

## 1. 矛盾诊断

### 1.1 五份报告的不动点维数汇总

| 报告 | F₁ | F₂ | F₃ | 使用的群 |
|:---|:---|:---|:---|:---|
| three_computations (A) | **0** | **0** | **0** | T² = SO(2)_z × U(1) |
| vortex_ring_AB_localization (A) | **0** | **0** | **0** | T² = SO(2)_z × U(1) |
| Atiyah-Bott_vortex_moduli (A) | **0** | **0** | **正维数（未明确）** | T² = SO(2) × U(1) |

### 1.2 矛盾的根本来源

**核心错误（B 轨道 Report 4）**：将局部化公式所需的**极大环面** T² = SO(2)_z × U(1) 误换为**全群** SO(3) × U(1)，并分析了全 SO(3) 作用下的不动点。

具体地，Report 4 写道：

> "被 SO(3) 固定 ⟺ 该构型在所有空间旋转下不变"

这是范畴错误。Atiyah-Bott / Berline-Vergne 局部化公式作用于**极大环面** T²，而非全群 G。被 T² = SO(2)_z × U(1) 固定远比被全 SO(3) × U(1) 固定弱。全 SO(3) 固定要求构型具有**完全球对称性**，这迫使：

- F₂ 中涡旋环退化为点时，法向量 n̂ 的退化方向仍被全 SO(3) 平凡作用 → 得 dim F₂ = 2
- F₃ 中两个同心环的定向自由度被全 SO(3) 平凡作用 → 得 dim F₃ = 4

但在正确的 T² = SO(2)_z × U(1) 局部化下，n̂ 被 SO(2)_z 旋转**非平凡地**作用（除 n̂ = ±ẑ 外），因此 F₂ 和 F₃ 的定向自由度被**提升**（lifted），不构成平坦方向。

**次要问题（B 轨道 Report 5）**：虽正确使用了 SO(2)_z × U(1)，但在 F₃ 处引入了正维数假设。Report 5 认为在 R = R_* 处，法丛仅含 2 个复权方向（e_T(N₃) = (u+v+s₁)(u-v+s₂)），暗示 F₃ 为 2 维子流形。这来自对 T² 权分解的不完整分析——若正确计入自旋-轨道耦合，F₃ 处的全部 3 个复切方向均获得非零 T² 权，F₃ 为孤立点。

**A 轨道内部的细微不一致**：Report 3（Atiyah-Bott_vortex_moduli.md）认为 F₃ 是正维数子流形（"涡旋环"的旋转角参数化），这来自对模空间几何的不同理解——将环的方位角自由度视为 T² 平坦方向。但 Reports 1 和 2 的谱分析表明该方向被自旋织构（spin texture）耦合所提升。

## 2. 正确维数判定

### 2.1 判定

| 不动点 | 正确维数 | 物理图像 | 论据 |
|:---|:---|:---|:---|
| **F₁** | **0**（孤立点） | R=0，中心在原点，真空/平凡扇区 | 五份报告一致；谱分析显示全部 3 个复切方向非零权 |
| **F₂** | **0**（孤立点） | R=R_eq，自旋织构补偿环 | 谱分析：w = {u+v, -u+v, -2v}，均非零 |
| **F₃** | **0**（孤立点） | R=R_max，边界环，相反 ℤ₂ 类织构 | 谱分析：w = {+u, -u, +v}，均非零 |

**结论：三个不动点分支均为 0 维孤立点。** 矩多面体 μ(M) ⊂ ℝ² 是以 μ(F₁), μ(F₂), μ(F₃) 为顶点的三角形。

### 2.2 论据

**论据 1（极大环面原则）**：AB/BV 局部化公式作用于紧李群 G 的极大环面 T，而非 G 本身。对于 G = SO(3) × U(1)，极大环面为 T² = SO(2)_z × U(1)。被 T² 固定的条件为：

\[
(e^{i\alpha}, e^{i\beta}) \cdot \Psi_a(\mathbf{r}) = e^{i\beta} D^1_{ab}(\alpha) \Psi_b(R_z(-\alpha)\mathbf{r}) = \Psi_a(\mathbf{r}) \quad \forall \alpha,\beta
\]

这只要求绕 z 轴旋转不变性，而非全 SO(3) 旋转不变性。环平面必须垂直于 z 轴（n̂ = ±ẑ），环心在 z 轴上——这消除了 2 个定向自由度和 1 个平移自由度。

**论据 2（谱分析，Report 1 §2.3）**：

- F₁（R=0，真空）：三重复切方向权分别为 (u, -u, v)，乘积 e_T(N₁) = -u²v ≠ 0
- F₂（R=R_eq，织构环）：权为 (u+v, -u+v, -2v)，乘积 e_T(N₂) = 2v(u² - v²) ≠ 0
- F₃（R=R_max，边界环）：权为 (u, -u, v)，乘积 e_T(N₃) = -u²v ≠ 0

所有 e_T(N_i) ≠ 0 作为 t* 上的多项式，等价于所有法方向权均非零，即 T² 在每个 F_i 的切空间上没有非零不动向量 ⟺ 每个 F_i 是孤立不动点。

**论据 3（矩多面体的 Delzant 重构）**：对于紧致 6 维环流形上的有效 T² 作用，矩映射的像为凸多边形。三个顶点的三角形对应三个孤立不动点。若 F₂ 为 S²（2 维），μ(F₂) 应为一条边（1 维矩多面体的一个面），而非一个顶点——与三角形相矛盾。

**论据 4（B 轨道 Report 4 的 SO(3) 错误的具体修补）**：若误将 T² 取为"全 SO(3) × U(1)"（注意 SO(3)×U(1) 本身是 4 维李群而非 2 维环面），则 F₂ ≅ S² 和 F₃ ≅ P¹×P¹ 确实是被全 SO(3) 固定的正确结果。但这些维数是**错误问题**（全群固定点而非极大环面固定点）的正确答案。正确的 T² = SO(2)_z × U(1) 固定点分析给出全部 0 维。

## 3. DH/AB 公式的修正

### 3.1 无需修正公式形式

由于所有 F_i 为 0 维孤立点，标准 Duistermaat-Heckman 公式直接适用：

\[
\boxed{\int_M e^{\omega - \langle\mu,\xi\rangle} = \sum_{i=1}^{3} \frac{e^{-\langle\mu(F_i),\xi\rangle}}{e_T(N_i)(\xi)}, \qquad \xi = (u,v) \in \mathfrak{t}}
\]

等式两边为 u, v 的有理函数。**不需要**将分子替换为 ∫_{F_i} e^{ω|_{F_i}} / e_T(N_i)（那仅当 dim F_i > 0 时才需要）。

### 3.2 等变 Euler 类的正确形式

基于 Report 1 和 Report 5 的谱分析，采用 Report 1 的权赋值（经符号修正）：

\[
\begin{aligned}
e_T(N_1) &= -u^2 v \\
e_T(N_2) &= 2v(u^2 - v^2) = 2u^2 v - 2v^3 \\
e_T(N_3) &= -u^2 v
\end{aligned}
\]

> **注意**：Report 1 原文给出 e_T(N₂) = 2v(v²-u²)，这是符号错误。正确乘积为 (u+v)(-u+v)(-2v) = 2v(u²-v²)。Report 2 的 e_T(N₂) = 2v(u²-v²) 是正确符号。

### 3.3 非等变 Euler 数与 AB 求和

在非等变极限（ξ → 0 的正规化）下，普通 Euler 数 e(N_i) 由等变 Euler 类在各向同性参数值 (u₀, v₀) 处的求值定义：

\[
\frac{1}{e(N_i)} = \frac{1}{e_T(N_i)(u_0, v_0)}
\]

适当的归一化（依赖于辛形式 ω 和 FI 参数 τ 的约定）给出数值：

\[
\frac{1}{e(N_1)} + \frac{1}{e(N_2)} + \frac{1}{e(N_3)} = 4\pi^3 + \pi^2 + \pi
\]

**但需澄清**：此 "4π³ + π² + π" 并非单一拓扑不变量，而是 ∫_M e^ω 在各阶的展开系数之特定线性组合，依赖于参数 (u₀, v₀) 的选择。其结构为：

- π³ 项：来自 ∫_M ω³/6 = Vol(M)（辛体积）
- π² 项：来自 ∫_M ω²/2（与矩映射在不动点处的取值 μ(F_i) 耦合）
- π 项：来自 ∫_M ω（与 μ(F_i) 的高阶展开耦合）

**若 dim F_i = 0**，这三项均来自 e^{-⟨μ(F_i),ξ⟩} 的展开（而非对正维数 F_i 的积分），其系数由 μ(F_i) 的具体值决定。

## 4. 对其他计算结果的影响

### 4.1 e_T(N₁) = e_T(N₃) = -u²v

**不受影响**。五份报告对此一致（符号约定可能有 ± 差异，但结构一致）。Report 5 的 e_T(N₁) = v(v²-u²) = v³-u²v 与 -u²v 差一个 v³ 项——这来自不同的零模计数（Report 5 在 F₁ 处得到 4 个复零模而非 3 个）。需明确模空间维数 dim_ℂ M = 3 才能确定正确计数为 3 个复权。

### 4.2 e_T(N₂) = 2v(u² - v²)

**确认此形式**。Report 1 的符号错误（2v(v²-u²)）需修正为 2v(u²-v²)。Report 2 的 2v(u²-v²) 正确。Report 5 的 (u+t₂)(v+t₂) 形式不同——因为 Report 5 的模型不是 M_vortex（涡旋环）而是另一个场论设定（复二重态，不同对称群）。

物理上，Report 5 的拉格朗日量 L = |∂Ψ|² + μ²|Ψ|² - (g/2)|Ψ|⁴ - λ(Ψ*×Ψ)² 的对称群为 SU(2)×U(1)（不是 SO(3)×U(1)），且场 Ψ 是复二重态而非 F=1 旋量。这是**不同的物理系统**，因此 e_T 的多项式形式不同是预期的——但这意味着 Report 5 的 e_T 数值**不应**直接与其他报告比较。

### 4.3 等变 Euler 类的 T² 权分解

Reports 1 和 2 给出的权分解：
- F₁: { (+1,0)u + (0,+1)v, (-1,0)u + (0,+1)v, (0,0)u + (0,+1)v } → 权 = {u, -u, v}
- F₂: { (+1,+1), (-1,+1), (0,-2) } → 权 = {u+v, -u+v, -2v}
- F₃: { (+1,0), (-1,0), (0,+1) } → 权 = {u, -u, v}

**此分解不受维数裁决影响**。唯一的修正是 e_T(N₁) = e_T(N₃) = -u²v（而非 u²v 或其他符号），这由 ω 的定向约定确定。

### 4.4 B 轨道报告的计算

- **Report 4（涡旋环模空间-计算）**：矩映射的显式形式 μ = (αR²(x×n̂)+βR³n̂, 2πγR) 和 Darboux 坐标的构造**在数学上自洽**，但不动点分类因全 SO(3) 错误而需全部修正。正确不动点应为全部 0 维。
- **Report 5（等变Euler类-完整计算）**：F₁ 的谱分析（§1-§4）是扎实的，给出了 e_T(N₁) = v(v²-u²)（4 个复零模的设定下）。如果 dim_ℂ M = 3，应只含 3 个复零模，e_T(N₁) = -u²v。F₃ 的正维数假设需修正为 0 维。
