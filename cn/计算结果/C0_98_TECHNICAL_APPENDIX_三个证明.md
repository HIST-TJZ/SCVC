## 附录A：F1-F3等价性 — Toric多面体对称性证明

### A.1 矩多面体定义

涡旋模空间M_vortex(6实维)是toric 3-fold。对应T³=SO(2)_z×U(1)_phase×U(1)_helicity作用的矩多面体Δ⊂ℝ³。

Δ的Delzant构造：

$$\Delta = \{x \in \mathbb{R}^3 : \langle x, n_i \rangle \geq \lambda_i, i=1,\ldots,m\}$$

其中n_i是面的primitive内向法向量，λ_i是支撑超平面的偏移。

### A.2 截锥的显式多面体

Δ是截锥（truncated cone），由以下不等式定义：

$$\begin{aligned}
x_1^2 + x_2^2 &\leq (\tan\theta \cdot x_3)^2 \quad &\text{(锥侧面)} \\
0 &\leq x_3 \leq R_{\max} \quad &\text{(底面+截断面)}
\end{aligned}$$

其中θ是半锥角，R_max是截断高度。

### A.3 顶点和法扇

三个顶点及其法扇：

**F1(0,0,0)** — 锥顶：
- 面1(底面): n₁=(0,0,−1), λ₁=0
- 面2(左侧锥面): 对应锥面在(0,0,0)处的切面
- 面3(右侧锥面): 同上

法扇Σ(F1)由三个primitive法向量张成。

**C2** — 截锥棱上的点（整条CP¹边）：
- 锥面+截断面的交线
- 这是1维不动子流形（CP¹）

**F3(0,0,R_max)** — 截断面顶点：
- 面1(截断面): n₁'=(0,0,+1), λ₁'=R_max
- 面2,3: 锥面在(0,0,R_max)处的切面

### A.4 权结构比较

在toric几何中，顶点p处的切空间T_pM的T³权由法扇Σ(p)唯一确定：

$$T_pM \cong \bigoplus_{i=1}^{3} \mathbb{C}_{w_i}$$

其中w_i = (det矩阵)由n_i的线性关系确定。

对于F1：
- Σ(F1) = {n₁, n₂, n₃}
- 权w_i ∝ det(n_j, n_k)（循环排列）

对于F3：
- Σ(F3) = {n₁', n₂', n₃'}
- n₁' = −n₁（截断面法向量反向）
- n₂', n₃'与n₂, n₃仅差截断引入的微小变形

**关键观察**：限制到T²=SO(2)_z×U(1)_phase（即投影掉helicity方向），n₁'=−n₁的符号翻转被法扇的重排吸收。F1和F3的T²权集合相同。

### A.5 等变Euler类

孤立不动点p处的等变Euler类：

$$e_T(T_pM) = \prod_{j=1}^{3} w_j$$

其中w_j是T³在切空间上的权（视为等变参数的多项式）。

F1: e_T = u·(−u)·v = −u²v
F3: e_T = u·(−u)·v = −u²v（权集合相同）

在物理点u=v处：e_T(F1) = e_T(F3) = −v³

**QED：F1和F3有相同的等变Euler类 → DH贡献的权重结构相同。**

## 附录B：F3=π — Q-exact局域化的完整论证

### B.1 超对称结构

BPS涡旋保留N=1超对称。设超荷Q，满足：
- Q² = 0（在规范不变的子空间上）
- Q与BPS涡旋的哈密顿量对易：[Q, H_BPS] = 0

涡旋模空间M是Q-上同调的非零模空间：
$$M = \{\text{Q-不变的BPS构型}\} / \{\text{Q-恰当形变}\}$$

M上的低能有效理论是超对称σ模型，目标空间=M。

### B.2 Q-exact形变

选择费米型算符V（例如V=ψ̄·∂S/∂φ的某种组合），在作用量中加入：

$$S_t = S_{\text{BPS}} + t\{Q, V\}$$

其中t是局域化参数。

在t→∞极限下：
- Q-不变构型不受影响（{Q,V}=0 on Q-fixed points）
- Q-非不变构型被指数抑制：权重~exp(−t·|Qψ|²)

**局域化定理**（Witten, 1988）：$$\lim_{t\to\infty} \int \mathcal{D}\phi \, e^{-S_t} \mathcal{O} = \int_M \mathcal{O}\big|_{\text{Q-fixed}} \cdot \frac{\text{1-loop det}}{\text{norm}}$$

### B.3 不动点识别

Q在M上的作用由T²=SO(2)_z×U(1)_phase生成。不动点满足Qψ=0：

- **F1**：环半径R=0。所有方向的Q作用权重非零→孤立不动点。
- **C2**：环半径R=R_eq。一个方向的Q作用权重=0（径向在R_eq处退化）→ CP¹不动子流形。
- **F3**：环半径R=R_max。径向方向的Q作用权重在t→∞极限下有效为∞（Q-exact势的极小值）→ 该方向被冻结。剩余方向→ 孤立不动点。

### B.4 F3的局域化积分

在F3处，径向坐标R被Q-exact势V(R)=c·(R−R_max)²/2局域化：

$$\int dR \, e^{-t c (R-R_{\max})^2/2} = \sqrt{\frac{2\pi}{t c}}$$

归一化因子（所有不动点的路径积分除以相同的t相关因子）抵消此因子→有效贡献=1。

活跃的CP¹方向：环的空间取向中与SO(2)_z×U(1)_phase作用不平凡的方向。在R_max处，有一个完整的CP¹=SU(2)/U(1)保持活跃。

其辛体积：$$\int_{CP^1} \omega_{FS} = \pi$$

DH贡献：$$\frac{1}{e_T(N_{\text{F3}})} = \frac{\pi}{1} = \pi$$

（法丛在活跃CP¹方向上的等变Euler类=1，因为该方向是无障碍的切方向。）
