## 0. 框架回顾

**SCVC 核心假设**：
- 真空 = F=1 旋量 BEC 超流体，序参量为三分量复标量场 ψ_m（m = -1, 0, +1）
- 真空流形 M_vac = (S² × S¹)/Z₂，具有非平凡拓扑
- 基本粒子 = 真空中的涡旋环（vortex ring）拓扑激发
- 涡旋环的绕数（通量量子数）N ∈ Z 决定费米子零模数目（Atiyah-Singer 指数定理）

**已知结果**：
| N | 费米子零模 | 物理对应 | 稳定性 |
|---|---|---|---|
| 1 | 1 | 电子（U(1) 荷，自旋 1/2） | 绝对稳定（拓扑保护 + 电荷守恒） |
| 3 | 3 | 三代费米子结构 | 可衰变至 N=1（弱相互作用） |
| — | — | 规范群 = SU(3)×SU(2)×U(1) | 从 M_vac 几何涌现 |

**DH 求和（N=1）**：
$$\alpha^{-1} = 4\pi^3 + \pi^2 + \pi = 137.036304$$

模空间不动点：F1 (R=0)、C2 (截锥棱上)、F3 (R=R_max)。

## 1. N=0 涡旋环（纯磁通环）

### 1.1 定义与存在性

N=0 涡旋环在拓扑分类中属于平凡扇区（trivial sector）。然而，在 F=1 旋量 BEC 的多分量结构中，"N=0"并不等同于真空。考虑三分量场 ψ_m 各自具有绕数 (n₊₁, n₀, n₋₁)，总绕数：

$$N = n_{+1} + n_0 + n_{-1}$$

N=0 包含以下非平凡配置：
- **(k, -k, 0)** 型：两个分量具有相反绕数，第三个分量为零
- **(k, -k/2, -k/2)** 型：绕数在三者间分配使总和为零
- 纯扭曲型：各分量绕数均为零，但有非平凡的相对相位缠绕（texture）

这些配置虽总绕数为零，但各分量携带非零的"部分通量"，因此具有非零的场能。

### 1.2 费米子零模：严格为零

由 Atiyah-Singer 指数定理：

$$\text{Index}(D) = \frac{1}{2\pi} \int F = N$$

其中 D 是涡旋背景上的狄拉克算符。N=0 ⇒ Index = 0，即**严格没有费米子零模**。这意味着 N=0 涡旋环不携带任何标准模型量子数——没有电荷、没有色荷、没有弱同位旋。它是"不可见"的激发，仅通过引力（能量-动量张量）和可能的拓扑耦合（见 1.5 节）与 SM 扇区相互作用。

### 1.3 质量估计

N=0 涡旋环没有费米子核心的贡献。其质量完全来自规范场（曲率）能量和旋量场的梯度能量。类比 Abelian Higgs 模型中涡旋环的半经典质量公式：

$$M_{\text{ring}} \approx 2\pi^2 R \cdot T(R)$$

其中 T(R) 是涡旋环的有效张力。对于 N=0 的 (k, -k) 型配置，各分量的部分通量在远场处相互抵消（类似电偶极子），张力随距离指数衰减：

$$T(R) \sim \mu \cdot e^{-R/\xi}$$

其中 ξ 为关联长度，μ 为特征能量标度。

这意味着 N=0 环的质量远小于 N=1 环：
$$M_{N=0} \ll M_{N=1} \approx m_e = 511\ \text{keV}$$

典型估计：$M_{N=0} \sim 10^{-3}\ \text{eV}$ 至 $1\ \text{eV}$ 范围。

### 1.4 稳定性分析

**拓扑保护：无。** N=0 属于 π₁ 的平凡元素，没有同伦保护。

**动力学稳定性：弱。** 三种可能衰变道：

| 衰变道 | 过程 | 时间标度 |
|---|---|---|
| 辐射衰变 | N=0 环 → 电磁辐射（光子） | τ ~ 10⁻²⁰ s |
| 碎裂 | (k, -k) → 两个分开的 (±k) 激发 | 取决于能垒 |
| 收缩湮灭 | 环半径 R → 0，能量耗散为声子 | τ ~ R/c_s ~ 10⁻²² s |

**结论：N=0 涡旋环不稳定，寿命远短于宇宙学时标。** 它不是可行的暗物质候选。

### 1.5 与 SM 粒子的耦合

虽无直接规范耦合（无费米子零模 → 无 SM 量子数），但存在间接耦合：

- **引力耦合**：能量-动量张量耦合（普适）
- **拓扑耦合**：若 N=0 环带有非平凡的轴子型耦合（aF̃F），可与光子混合
- **高阶算符**：通过虚的 N=±1 环对产生，诱导极弱的有效耦合

这些耦合极弱，但在早期宇宙中可能影响 N=0 环的热产生。

### 1.6 模空间比较：N=0 vs N=1

| 属性 | N=0 | N=1 |
|---|---|---|
| 模空间维度 | 3（位置）+ 内部自由度 | 3（位置）+ 3（方向）+ U(1) 相位 |
| 费米子零模 | 0 | 1（手征） |
| 拓扑保护 | 无 | π₁ ≠ 0 |
| 稳定性 | 不稳定 | 绝对稳定 |
| 质量 | ≪ m_e | ≈ m_e |

## 2. N > 3 大涡旋环

### 2.1 费米子零模数目

由指数定理，N 涡旋环携带 $|N|$ 个费米子零模。对于大 N：

$$n_F = N \quad (\text{若 } N > 0), \quad n_F = |N| \quad (\text{一般})$$

这些零模在环的世界面上形成 (0+1) 维费米子体系。对于 N > 3，多于 3 个零模意味着必须引入超越了标准模型三代费米子的额外自由度。

### 2.2 质量标度

涡旋环的总质量有两个竞争贡献：

$$M_N = E_{\text{gauge}}(N) + E_{\text{core}}(N) + E_{\text{Fermi}}(N)$$

**规范场能量**（主导项）：对 N 重通量的涡旋，磁场能量密度 $\sim B^2$，总通量 $\Phi = N\Phi_0$。类比超导体中的多量子涡旋：

$$E_{\text{gauge}} \approx \frac{\Phi_0^2}{8\pi^2 \lambda^2} \cdot N^2 \cdot \ln\left(\frac{\lambda}{\xi}\right) \cdot 2\pi R$$

其中 λ 是穿透深度，ξ 是关联长度。**关键观察**：规范场能量按 $N^2$ 标度（因为 $B \propto N$，$B^2 \propto N^2$）。

**核心能量**：$E_{\text{core}} \propto N$（每个通量量子一个核心区域）

**费米子能量**：填充 N 个零模的狄拉克海能量，$\propto N \cdot \Lambda$（Λ 为截断）

总质量标度：
$$M_N \approx a N^2 M_1 + b N M_1$$

对于大 N，$N^2$ 项主导。因此：

$$\boxed{M_N \propto N^2 \cdot M_1 \quad (\text{大 N 极限})}$$

### 2.3 稳定性：大 N → 小 N 的衰变

考虑衰变道 $N \to N_1 + N_2$（$N = N_1 + N_2$）：

$$\Delta M = M_N - (M_{N_1} + M_{N_2})$$

代入 $M_N \propto N^2$：

$$\Delta M \approx M_1 [N^2 - (N_1^2 + N_2^2)] = 2 M_1 N_1 N_2 > 0$$

**结论：大 N 涡旋环在能量上总是不利的**——它倾向于碎裂为多个小 N 环。N² 标度使大环内禀不稳定。

**拓扑约束**：但是否存在阻止碎裂的拓扑障碍？

M_vac = (S² × S¹)/Z₂ 中，同伦群 π₁ = Z。每个涡旋的绕数 N 是相加守恒的：
- $N \to N_1 + N_2$ 是允许的（只要 $N = N_1 + N_2$）
- 不存在额外的 Z₂ 障碍阻止此过程

唯一的潜在稳定性来自**动力学能垒**：分裂过程需要克服中间态的能量势垒。若势垒足够高（$\gg k_B T_{\text{universe}}$），大 N 环可具有宇宙学意义上的亚稳态。

势垒高度估计：
$$\Delta E_{\text{barrier}} \sim M_1 \cdot N^{3/2}$$

对于 N=4,5,6，此势垒可能使寿命超过宇宙年龄，使其成为**亚稳态暗物质候选**。

### 2.4 具体 N 值的分析

| N | 零模数 | 可能身份 | 稳定性 |
|---|---|---|---|
| 4 | 4 | 四代费米子 + 规范玻色子 | 可衰变至 3+1 或 2+2 |
| 5 | 5 | 五重态费米子 | 可衰变至 3+2 等 |
| 6 | 6 | — | 可衰变至 3+3（最对称衰变道） |
| N ≫ 1 | N | 宏观通量管环 | 经典稳定性（宇宙弦环类比） |

其中 **N=4** 和 **N=6** 值得特别关注：

- **N=4**：四零模。若 SCVC 框架允许四代费米子，N=4 环可能是第四代费米子的拓扑来源。由于 SM 中第四代手征费米子被 LEP 排除，N=4 环须有高质量（> m_Z/2）或非手征性质。若 N=4 环的零模在低能下被质量项耦合（类似 seesaw），其有效低能自由度减少，可作为仅通过引力耦合的暗物质。

- **N=6**：六零模，可衰变至 3+3（两个三代费米子结构）。这等价于两个 N=3 环的束缚态。若束缚能 $\Delta E_{\text{bind}} < 0$（即 $M_6 < 2M_3$），则 N=6 对比分裂是绝对稳定的。是否 $M_6 < 2M_3$ 取决于中间区域的非微扰动力学。

### 2.5 宇宙学丰度

若大 N 环在早期宇宙中通过 Kibble-Zurek 机制产生，初始丰度呈指数压低：

$$\Omega_N h^2 \sim e^{-c N} \cdot \Omega_1 h^2$$

实际上，在相变温度 $T_c \sim M_1 \sim \text{TeV}$ 时，关联长度 $\xi \sim 1/T_c$，产生 $N>3$ 环的概率指数级小。除非存在后期产生机制（如小环的融合），大 N 环的宇宙学丰度可忽略不计。

## 3. 纯磁通环（无费米子核心的涡旋）

### 3.1 物理图像

"纯磁通环"指涡旋环的规范场-希格斯部分（类比 Abelian Higgs 模型中的 ANO 涡旋），但**费米子核心未被填充**——即费米子零模是空的（未被量子激发占据）。

在 SCVC 超流体图像中：
- 涡旋环 = 环状的拓扑缺陷，环的核心处序参量为零
- 费米子零模局域在核心处
- 当零模未被占据时，环仅携带规范场/超流体能量

这与 N 的关系：
- 纯磁通环可具有任意 N（通量量子数）
- N ≠ 0 时有零模"空穴"——可被占据但未被占据
- N = 0 时天然无零模（见第 1 节）

### 3.2 存在性：裸涡旋的可能性

在 Abelian Higgs 模型中，裸 ANO 涡旋是严格存在的静态解。SCVC 的推广：

**有费米子的情况**：费米子场与涡旋的耦合通过 Yukawa 项 $\sim \bar{\psi} \phi \psi$。当序参量 ϕ 在核心处归零时，费米子获得零模。核心处费米子场的存在**反作用于涡旋解**（backreaction），修改其剖面和能量。

**裸涡旋（空零模）**：取费米子场期望值为零（⟨ψ⟩=0），求解纯玻色子涡旋方程。此时得到的是裸涡旋解。其存在性在数学上等价于纯规范-希格斯涡旋的存在性，已被 ANO 涡旋证明。

**关键物理问题**：裸涡旋是否是能量最低态？若填充零模可降低总能量（因费米子负能态），裸涡旋是激发态而非基态，会通过"吸收"周围费米子快速填充零模。

### 3.3 质量估计

裸涡旋的质量 = 纯规范-希格斯涡旋质量，不含费米子填充能。

$$M_{\text{bare}} \approx M_{\text{ANO}}(N) \approx \frac{v^2}{2} \cdot N \cdot f(\lambda/e^2) \cdot 2\pi R$$

与填充零模的环相比：
$$M_{\text{filled}} = M_{\text{bare}} + N \cdot \epsilon_F$$

其中 $\epsilon_F \sim \Lambda_{\text{UV}}$ 为零模填充能。

**裸涡旋更轻**，因此在高能碰撞中可能作为中间态产生。

### 3.4 稳定性

裸涡旋的稳定性与填充涡旋类似——完全由 N（通量数）的拓扑保护决定：
- N ≠ 0：拓扑保护，绝对稳定（不能连续变形到真空）
- N = 0：无拓扑保护（见第 1 节）

额外衰变道：裸涡旋可通过"费米子填充"衰变：
$$\text{裸涡旋}_N + \text{费米子} \rightarrow \text{填充涡旋}_N + \gamma$$

此过程的截面由费米子-涡旋耦合强度决定。在早期宇宙的热浴中，裸涡旋会被快速填充。

### 3.5 早期宇宙产生

裸涡旋可通过两种机制产生：

1. **Kibble-Zurek 机制**：相变中直接产生。由于相变时间 $\tau_Q$ 有限，因果关联区域内涡旋方向随机。初始产生的涡旋可能是裸的（费米子未占据零模）或部分填充的。裸涡旋占比取决于相变时费米子化学势。

2. **高能碰撞**：$\gamma + \gamma \rightarrow \text{裸涡旋}_N + \text{裸反涡旋}_{-N}$。此过程在早期宇宙高温阶段有效，截面对大 N 指数压低。

### 3.6 暗物质可行性

裸涡旋本身**不是好的暗物质候选**：在宇宙热历史中它们会被填充，变成携带 SM 量子数的普通涡旋（即电子等 SM 费米子）。唯一的例外是 N=0 裸环——但 N=0 环不稳定（第 1 节结论）。

## 4. 多环束缚态

### 4.1 物理动机

SCVC 中涡旋环的动力学由如下效应控制：
- **长程力**：类似流体力学中的涡旋-涡旋相互作用（比约克尼斯力）
- **短程力**：核心重叠产生的规范场/标量场介导的力
- **拓扑力**：来自 M_vac 的非平凡几何

在特定条件下，两个或更多涡旋环可形成束缚态。

### 4.2 N=1 + N=-1 涡旋-反涡旋对

这是最重要的多环配置。单独讨论它与希格斯机制的区别。

**规范量子数**：
- N=1 环：1 个费米子零模（电子型）
- N=-1 环：1 个反费米子零模（正电子型）
- 总绕数：N_tot = 0，总零模数 = 2（一个粒子 + 一个反粒子）

**自旋分析**：
两个零模在环的世界面上形成 2 维费米子体系。其基态由占据方式决定：
- 两个零模均占据：Dirac 海填充，总角动量 J = 0（标量）或 J = 1（矢量）
- 零模间的相对轨道角动量决定总自旋

最低能量态（S 波）：
$$\text{基态}: J^P = 0^+ \ \text{（标量）或} \ 1^- \ \text{（矢量）}$$

类比正负电子偶素（positronium），N=1+N=-1 束缚态按轨道角动量分为：
- **对 ortho-态**（S=1，三重态）：矢量，$J^{PC} = 1^{--}$
- **对 para-态**（S=0，单重态）：标量，$J^{PC} = 0^{-+}$

**与希格斯机制的关键区别**：

| 属性 | 涡旋-反涡旋对（束缚态） | 希格斯凝聚（多体效应） |
|---|---|---|
| 粒子数 | 精确 2 个环 | 宏观占据数（N → ∞） |
| 描述 | 量子力学二体问题 | 平均场 / 有效场论 |
| 基态 | 离散束缚态谱 | 连续对称破缺真空 |
| 激发 | 环-环相对运动 | Nambu-Goldstone 模 |
| 物理角色 | 可能暗物质候选 | 赋予 W/Z 质量 |

### 4.3 束缚态谱与稳定性

N=1+N=-1 束缚态的有效势（类比重夸克偶素）：

$$V(r) \approx -\frac{\alpha_{\text{eff}}}{r} + \sigma r + V_0$$

其中 α_eff 为有效耦合常数（来自规范场交换），σ 为涡旋环"弦张力"（防止环无限分离）。

束缚态质量：
$$M_{n\ell} = 2M_1 + E_{n\ell}^{\text{bind}}$$

束缚能 $E_{\text{bind}} < 0$。若 $|E_{\text{bind}}| > 2m_e$（正负电子对的阈值），则束缚态对湮灭至正负电子对是不允许的。

**稳定性分析**：

主要湮灭/衰变道：
1. **直接湮灭**：涡旋环 + 反涡旋环 → 规范玻色子（γ, Z）/ 引力子
   - 截面 $\sigma_{\text{ann}} v \sim \alpha_{\text{eff}}^2 / M_1^2$
   - 若 M_bound < 2m_e：此道关闭，束缚态绝对稳定
   
2. **量子隧穿**：环收缩至零半径 → 拓扑解开 → 纯辐射
   - 速率 $\Gamma \sim e^{-S_E}$，其中 $S_E$ 是欧几里得作用量
   - 若势垒足够高，寿命可超过宇宙年龄

**稳定条件**：
$$\boxed{M_{\text{bound}} < 2m_e \ \text{且} \ S_E \gg 1}$$

若满足，N=1+N=-1 束缚态是绝对稳定的暗物质候选。

### 4.4 其他多环束缚态

| 配置 | 总 N | 总零模 | 可能自旋 | 稳定性 |
|---|---|---|---|---|
| N=1 + N=1 | 2 | 2 | 0, 1 | 可衰变至 2×(N=1) 自由态 |
| N=1 + N=-2 | -1 | 3 | 1/2, 3/2 | 可衰变 |
| N=2 + N=-2 | 0 | 4 | 0, 1, 2 | 类 N=1+N=-1 增强版 |
| N=3 + N=-3 | 0 | 6 | 多种 | 重束缚态 |
| N=k + N=-k | 0 | 2k | 多种 | 对称性增强稳定性 |

**N=2+N=-2** 特别有趣：总 N=0（无净 SM 荷），但有 4 个零模。束缚态可能类似于 QCD 中的四夸克态（tetraquark）。质量约为 $4M_1 = 4m_e = 2.044\ \text{MeV}$，在暗物质质量的有趣范围内。

### 4.5 早期宇宙产生与丰度

束缚态通过两阶段产生：
1. **第一阶段**（$T \sim T_c$）：Kibble-Zurek 产生自由 N=±1 环
2. **第二阶段**（$T < T_{\text{bind}}$）：自由环配对形成束缚态

束缚态产率：
$$Y_{\text{bound}} \sim \langle \sigma v \rangle n_{\text{ring}}^2 / H$$

其中 $n_{\text{ring}}$ 为自由环数密度，$H$ 为哈勃率。

若环的初始丰度足够（∼ 暗物质丰度量级），配对后的束缚态自然具有正确的暗物质丰度。这类似于"不对称暗物质"范式——暗物质粒子和反粒子的不对称性由拓扑数守恒决定。

### 5.1 候选汇总

| 序号 | 候选 | 质量范围 | SM 荷 | 稳定性 | 丰度可计算性 | 总体评分 |

### 5.2 最终判定：🟡

**总体评估：🟡 — 存在候选但稳定性/丰度需要额外假设**

理由如下：

1. **最自然的候选是 N=1+N=-1 束缚态**（第 4 项）：总 N=0，无 SM 荷，若束缚能足够大使其质量低于 2m_e 阈值则绝对稳定。这是涡旋环领域中等价于"暗光子"或"暗标量"的候选。然而，其稳定性依赖于非微扰束缚动力学（束缚态是否确实存在且质量低于阈值），这是当前无法从第一原理严格证明的假设。

2. **大 N 涡旋环**（第 2 项）的 N² 质量标度使其内禀不稳定（倾向于分裂）。仅在动力学能垒足够高时才能获得宇宙学寿命，且 Kibble-Zurek 产生率指数压低。这需要精细调节早期宇宙参数。

3. **裸纯磁通环**（第 3 项）是激发态而非基态，在热浴中会被填充。除非存在某种对称性阻止填充（如费米子化学势为负且绝对值为大），否则不是可行的暗物质。

4. **N=0 涡旋环**（第 1 项）明确不可行。

### 5.3 如果目标是 🟢 需要什么？

要从 🟡 升级到 🟢，需要以下进展之一：

- **非微扰束缚态计算**：证明 N=1+N=-1 束缚态确实存在，且质量 < 2m_e（可通过格点模拟或全息对偶实现）
- **拓扑 Z₂ 保护**：证明 M_vac = (S²×S¹)/Z₂ 中的 Z₂ 商群为某些 N 值的环提供额外的拓扑保护（类似于超对称中的 R-宇称）
- **新衰变禁戒**：发现阻止 N > 3 环衰变的新守恒律（如高形式对称性 higher-form symmetry）

### 5.4 模型无关的总结

SCVC 涡旋环谱为暗物质提供了丰富的候选空间。在最保守的假设下（不引入额外对称性、不精细调节参数），最自然的候选是**涡旋-反涡旋束缚态**。这在概念上类似于"镜像暗物质"或"隐藏扇区暗物质"——SM 粒子的拓扑"影子"形成暗扇区。与常规 WIMP 模型不同，此候选的质量和耦合原则上完全由真空流形的几何决定，使模型具有独特的可预测性。
