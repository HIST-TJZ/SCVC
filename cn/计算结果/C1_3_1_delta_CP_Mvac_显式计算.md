## 1. M_vac 上费米子波函数的 Z₂ 奇偶性

### 1.1 覆盖空间 S²×S¹ 上的波函数

覆盖空间 S²×S¹ 上，费米子场 ψ̃(x, r, φ) 满足 Z₂ 等变条件：

\[
\boxed{\tilde{\psi}(x, -r, \varphi+\pi) = \eta \,\tilde{\psi}(x, r, \varphi), \quad \eta = \pm 1}
\]

其中 (-r) 是 S² 上的对径映射（antipodal map）。

### 1.2 S² 球谐函数在对径映射下的行为

S² 上标准球谐函数 Y_{lm}(θ, ϕ)（归一化半径 R_S²）：

\[
Y_{lm}(\pi-\theta, \phi+\pi) = (-1)^l \, Y_{lm}(\theta, \phi)
\]

这是对径映射的基本性质：l 为偶 → 偶宇称（+1），l 为奇 → 奇宇称（-1）。

### 1.3 S¹ 上的 Fourier 模

S¹ 上的正交基：e^{i n \varphi}, n ∈ ℤ。

在半周期平移 φ → φ+π 下：

\[
e^{i n (\varphi+\pi)} = (-1)^n e^{i n \varphi}
\]

### 1.4 Z₂ 奇偶性条件

组合波函数展开：

\[
\tilde{\psi}(x, r, \varphi) = \sum_{l,m,n} c_{lmn}(x) \, Y_{lm}(r/R_S²) \, e^{i n \varphi}
\]

在 Z₂ 下：

\[
\tilde{\psi}(x, -r, \varphi+\pi) = \sum_{l,m,n} c_{lmn}(x) \, (-1)^l Y_{lm}(r/R_S²) \, (-1)^n e^{i n \varphi}
\]

要求此等于 η ψ̃(x, r, φ)：

\[
\boxed{(-1)^{l+n} = \eta \quad \text{对每个非零模}}
\]

**关键结果**：

| 费米子类型 | η (Z₂ 宇称) | 允许的 (l+n) | 最低模例子 |
|:---|:---:|:---|:---|
| 上型 (up-type) | +1 | 偶数 | l=0, n=0 或 l=1, n=±1 |
| 下型 (down-type) | -1 | 奇数 | l=0, n=±1 或 l=1, n=0 |

## 2. CP 相位的几何来源分析

### 2.1 M_vac 上 Yukawa 耦合的结构

标准模型 Yukawa 项：

\[
\mathcal{L}_Y = y_{ij} H \bar{Q}_i d_j + \text{h.c.}
\]

在几何框架中，Yukawa 耦合来自 M_vac 上波函数的重叠积分：

\[
y_{ij} \propto \int_{M_{vac}} d^3y \sqrt{g_3} \; \psi_{Q_i}^\dagger(y) \, \Phi_H(y) \, \psi_{d_j}(y)
\]

其中 ψ_{Q_i} 是 SU(2)_L 双重态分量（上型），ψ_{d_j} 是下型单态，Φ_H 是 Higgs 场。

### 2.2 最低模的显式重叠积分

取最低允许模：
- 上型：l=0, n=0 → ψ_up ∝ Y_{00} · 1（常数）
- 下型：l=0, n=±1 → ψ_down ∝ Y_{00} · e^{±iφ}
- Higgs（必须是 Z₂-偶，η=+1）：最低模 l=0, n=0 → Φ_H ∝ Y_{00} · 1

在 M_vac 上积分。M_vac 上的积分 = (S²×S¹ 上的积分)/2：

\[
y \propto \frac{1}{2} \int_{0}^{2\pi} d\varphi \int_{S^2} d\Omega \, |Y_{00}|^2 \cdot 1 \cdot e^{\pm i\varphi} \cdot |Y_{00}|^2
\]

\[
= \frac{1}{2} \cdot \frac{1}{4\pi} \cdot \int_{0}^{2\pi} d\varphi \, e^{\pm i\varphi} = 0
\]

### 2.3 关键发现：最低模直接 Yukawa 耦合为零！

**直接的 tree-level Yukawa 耦合在 Z₂-对称的最低模截断下恒为零。** 这是因为上型和下型有不同的 S¹ Fourier 模数（n_up 偶，n_down 奇），而 Higgs 的最低模是 φ-常数。

### 2.4 非零 Yukawa 耦合需要什么？

有以下几种可能使 Yukawa 非零：

**(A) Higgs 的非平凡 φ-依赖**：如果 Higgs 获得 n_H ≠ 0 的 Fourier 分量（例如通过自发破缺或与 S¹ Wilson 线耦合），则选择定则变为：
\[
-n_{up} + n_H + n_{down} = 0 \quad (\text{模 S¹ 上的动量守恒})
\]
即 n_down - n_up = -n_H。若 n_H = ±1，则 n_down = n_up ± 1，此时 l_down + n_down 和 l_up + n_up 可以同时满足各自的 η 条件。

**(B) S² 角动量 mismatch**：上型和下型可以使用不同的 l 值。例如：
- 上型：l=0, n=0（η = +1 ✓）
- 下型：l=1, n=0（η = -1 ✓）
- Higgs: l=1, n=0（若 Higgs 也带 S² 角动量）

此时重叠积分涉及三个球谐函数的积分（Gaunt 系数）。

### 2.5 方案(A)的显式计算：S¹ 上的 CP 相位

假设 Higgs 的 S¹ 模数为 n_H = 1（最低非平凡模），且 Higgs 的 η = +1（需 l_H 为奇数以使 l_H + n_H 为偶数）：

取：
- 上型：l=0, n=0, η=+1
- 下型：l=0, n=1, η=-1
- Higgs: l=1, n=1, η=+1（1+1=2 偶数 ✓）

Yukawa 积分：

\[
y \propto \frac{1}{2} \int_{0}^{2\pi} d\varphi \int_{S^2} d\Omega \, Y_{00}^*(\Omega) \, Y_{1m_H}(\Omega) \, Y_{00}(\Omega) \cdot e^{-i\cdot 0 \cdot\varphi} \cdot e^{i\cdot 1 \cdot\varphi} \cdot e^{i\cdot 1 \cdot\varphi}
\]

S² 部分：∫ Y_{00} Y_{1m_H} Y_{00} ∝ δ_{l_H,0} = 0（l=1 与 l=0 球谐函数正交）。

❌ **此方案 S² 积分为零。**

### 2.6 方案(B)的显式计算：S² 角动量 mismatch

取：
- 上型：l=0, n=0, η=+1
- 下型：l=1, n=0, η=-1（1+0=1 奇数 ✓）
- Higgs: l=1, n=0, η=+1（1+0=1 奇数 → η应为-1 ❌）

Higgs 的 η=-1 不可接受——Higgs 必须与自身耦合（Higgs 势的 λ|H|⁴ 项在对角线中），要求 η_H = +1。

尝试 Higgs: l=0, n=1, η=+1（0+1=1 奇数 → η应为-1 ❌）
或 Higgs: l=2, n=0, η=+1（2+0=2 偶数 ✓）

取 Higgs: l=2, n=0, η=+1：
- 上型：l=0, n=0, η=+1
- 下型：l=1, n=0, η=-1

Yukawa 积分 S² 部分：
∫ Y_{00} Y_{2m} Y_{1m'}：Gaunt 系数 ∼ ⟨00; 2m | 1m'⟩。只有当 m' = m 时非零。具体值：∼ δ_{m,m'} / √(4π) · (来自 Y_{2m} Y_{1m} 与 Y_{00} 的正交性…)

实际上：∫ Y_{00} Y_{LM} Y_{lm} = δ_{Ll} δ_{Mm} / √(4π)（Y_{00} = 1/√(4π)，利用球谐函数的正交归一性）。

所以 ∫ Y_{00} Y_{2m} Y_{1m'} = 0（因 l=2 ≠ l=1）。

❌ **同样为零。**

## 3. 根本困难：M_vac 中 CP 相位的结构

### 3.1 核心问题

上述计算揭示了一个根本性问题：**在 M_vac = (S²×S¹)/Z₂ 中，Z₂ 仅给出离散的 ±1 宇称。** 上型和下型之间的相位差是一个 Z₂ 相位（即 0 或 π），而不是连续的 U(1) 相位（60° = π/3）。

### 3.2 为什么旧理论得到 60°？

旧理论使用 Hopf 纤维化 S¹↪S³→S²。关键区别：

| | Hopf 纤维化 (旧理论) | M_vac = (S²×S¹)/Z₂ (新框架) |
|:---|:---|:---|
| S¹ 丛的类型 | 非平凡，陈数 = 1 | 平凡乘积丛 |
| S¹ 坐标与 S² 的关系 | 纤维坐标 γ 通过 Hopf 映射耦合到 S² | φ 与 S² 独立（乘积） |
| Z₂ 如何作用 | γ → γ+π，与 Hopf 扭转结合 | φ → φ+π，与 S² 对径映射独立配对 |
| 连续相位来源 | Hopf 不变量 = 1 → 扭转角有连续的 2π 周期，Z₂ 将其离散化为两个值 {0, π}，但两个值之间的"差"在覆盖意义上为 π/3（因 Hopf 映射的几何） | 无连续扭转 → Z₂ 给出严格离散的 ±1 |

### 3.3 M_vac 中缺失的要素

要得到连续的 δ_CP ≠ 0 或 π，需要以下至少一项：

1. **S¹ 上的连续 Wilson 线**：如果 U(1)_Y 规范场在 S¹ 上有非平凡的 Wilson 线（真空期望值），则可产生连续相位。但这是动力学问题，非纯几何。

2. **S² 上的非平凡 SU(2) 联络**：如果 SU(2)_L 在 S² 上有瞬子（Instanton）或通量，连续相位可从 Chern-Simons 项产生。但这同样需要动力学输入。

3. **CP 破缺来自 4D 有效理论的复 Yukawa 矩阵**：这是标准路径——Yukawa 矩阵的相位不由紧化几何唯一确定，而是模场（moduli）的真空期望值。

## 4. 进一步的几何分析：M_vac 上的离散 Wilson 线

### 4.1 精确的 Z₂ 商化分析

M_vac 的 Z₂ 商化等价于在 S²×S¹ 上引入一个**离散 Z₂ Wilson 线**（沿 S¹ 方向）：

绕 S¹ 一周（φ: 0 → 2π），场不回到自身，而是被 S² 上的对径映射 g(r) = -r 旋转。绕 S¹ 两周才真正闭合。

### 4.2 离散 Wilson 线的群论

对径映射 g 在 SU(2) 中的提升：对径映射是 S² = SU(2)/U(1) 上的映射，来自 SU(2) 的 Weyl 群元素 w。在 SU(2) 的基本表示（旋量表示）中，w 作用为：

\[
w = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix} \quad (\text{在 SU(2) 中})
\]

w² = -I（在 SU(2) 中），对应 SO(3) 中的恒等元。

因此，沿 S¹ 绕行一周，SU(2) 双重态的**两个分量交换并带一个相位**。

### 4.3 上型和下型的相位差

设上型和下型分量为 SU(2) 双重态的两个分量：

\[
Q = \begin{pmatrix} u_L \\ d_L \end{pmatrix}
\]

在 M_vac 上，绕 S¹ 一周后：

\[
Q(\varphi+2\pi) = \rho(g) \, Q(\varphi)
\]

其中 ρ(g) 是对径映射在 SU(2) 双重态表示中的矩阵。ρ(g) 的作用是交换 u_L ↔ d_L 并乘以一个相位。

但 Q 本身在 M_vac 上当且仅当 η 条件满足时才是良定义的，而 u_L 和 d_L 的 η 值相反（+1 和 -1）。

这意味着 u_L 和 d_L 之间的"相对相位"由 ρ(g) 决定。如果 ρ(g) = ±iσ₂（交换上下分量并带 ±i 因子），则相位差为 90°，不是 60°。

但 ρ(g) 的具体形式取决于 SU(2) 表示的选取——这是约定问题而非几何预言。

### 4.4 M_vac 几何能否预言一个确定的角度？

**结论：不能。** M_vac 的 Z₂ 商化产生的是一个 Z₂ × (SU(2) 旋量相位) 结构，其中 SU(2) 旋量相位是约定的（与 SU(2) 表示的基选取有关）。
