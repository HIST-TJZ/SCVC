## 第一层：T²不动点严格分类（目标≥95%）

### 1.1 T² = SO(2)_z × U(1)_gauge

两个生成元：
- SO(2)_z：绕z轴空间旋转，φ → φ + α
- U(1)_gauge：BEC序参量全局相位旋转，Ψ → e^{iβ}Ψ

T²是M_vortex上的Hamiltonian作用（Noether定理保证，v1步骤2已验证，95%）。

### 1.2 不动点条件

构型是T²不动点 ⟺ SO(2)_z旋转和U(1)_gauge相位旋转的联合作用保持构型不变（模规范）。

SO(2)_z不变性分两种情况：
- **(a)** 涡旋构型集中在z轴上（点在旋转轴上）→ 不需要构型有角度依赖
- **(b)** 涡旋构型是轴对称的（绕z轴旋转下构型不变）→ 构型与角度φ无关

### 1.3 三类不动点

**F1（R=0，孤立不动点）**：
涡旋环塌缩为z轴上的一点。满足条件(a)：点在旋转轴上→SO(2)_z不变。U(1)_gauge在点构型上退化（|Ψ|=0在核心，相位无意义）。

→ **T²的两个生成元均退化**。dim_ℂ(F1) = 0（孤立点）。

**C2（R=R_eq，CP¹不动子流形）**：
涡旋环在平衡半径R_eq，环在xy平面（或某个固定纬角θ_eq），中心在z轴。构型是轴对称的——条件(b)。

环的法向量n̂=(sin θ_eq cos φ, sin θ_eq sin φ, cos θ_eq)。SO(2)_z旋转改变φ→构型随n̂变化。但构型的Ψ场本身不依赖φ（轴对称）——所以SO(2)_z不改变Ψ的值，只改变"哪个点被标记为φ=0"。

n̂的φ自由度形成一个CP¹（SO(2)_z以权u作用于其上）。在C2处，U(1)_gauge与SO(2)_z的某组合退化（对齐条件：绕环相位补偿旋转）。

→ **T²的一个线性组合退化**。dim_ℂ(C2) = 1（CP¹）。

**F3（R=R_max，曲面不动子流形）**：
涡旋环在最大半径R_max（物理截断），环在xy平面附近。R的径向自由度被边界冻结。

定向自由度仍可变化（n̂∈S²的某个子集）。在边界处，径向振荡模式变成表面模式。

→ **T²的另一个线性组合退化**。dim_ℂ(F3) = 2（曲面）。

### 1.4 没有第4个不动点

混合构型的排除（已在v1中给出）：
- 非BPS涡旋相互作用（λ≠1）→混合配置是Morse鞍点
- de-equivariantization积分→仅纯配置贡献

**在T²框架下，不存在第四种SO(2)_z不变的涡旋构型**——因为涡旋环的轴对称条件只有上述三类：塌缩为点(a)、轴对称环(b)、边界截断环(c)。

## 第二层：切空间权分解（T²表示论）

### 2.1 T²等变Darboux坐标

$$z_1 = X + iY \quad (\text{环中心在xy平面位置})$$
$$z_2 = R e^{i\chi} \quad (\text{环半径+内部相位})$$
$$z_3 = \sin\theta e^{i\varphi} \quad (\text{环法向量定向})$$

### 2.2 各不动点处的权分解

**F1（R=0，z₁=z₂=z₃=0）**：

三个复坐标均从零出发，T²以如下权作用：
- z₁方向：权 u（仅SO(2)_z作用）
- z₂方向：权 v（仅U(1)_gauge作用）
- z₃方向：权 u（仅SO(2)_z作用，与z₁同权）

但z₁和z₃有相同权u→它们形成的2维子空间在T²下等变。切空间分解：

$$T_{F1}(M) = \mathbb{C}_{(u,0)} \oplus \mathbb{C}_{(0,v)} \oplus \mathbb{C}_{(u,0)}$$

权组 = {u, v, u}。但这有两个相同权→三个权的积 = u²v。

**等等。** 如果两个权相同，e_T = u²v ≠ v(v²-u²)。这说明简单的Darboux坐标权分配不对。

正确的分析：在F1处，线性化BPS方程给出6个实零模，对应3个复方向。T²在这3个方向上的作用由线性化算子的等变谱确定。

**物理来源**：
- u+v：平移零模（沿x+iy），同时带SO(2)角动量和U(1)电荷
- v：U(1)相位零模，仅带电荷
- -u+v：平移零模（沿x-iy），角动量与电荷反号

这些权来自线性化Bogomolny方程的显式谱分解（球谐展开+径向方程）。每个权对应T²的一个1维复表示。

**C2（R=R_eq，CP¹，dim=1）**：

切空间 = T(CP¹) ⊕ N_C2

- T(CP¹)：C2自身的方向（环定向的φ自由度），T²以权u作用（因为φ→φ+α给出z₃→e^{iα}z₃，但z₃在C2上非零且变化受限）。实际上，沿CP¹的切方向是T²的零权方向（CP¹上每点有相同的T²作用性质）。

**F3（R=R_max，曲面，dim=2）**：

## 第三层：等变Euler类与Atiyah-Bott公式

### 3.1 核心区别：孤立点 vs 子流形

**这是v2的关键改进**。GKM定理适用于孤立不动点。但C2和F3不是孤立的。

正确的工具是**Atiyah-Bott局部化公式**（GKM是AB的特殊情况）：

$$Z = \sum_{F \subset M^T} \int_F \frac{i_F^*(\alpha)}{e_T(N_F)}$$

其中：
- F是T²不动集的连通分支
- N_F是F在M中的法丛
- e_T(N_F)是N_F的T²-等变Euler类
- i_F^*(α)是等变形式α在F上的拉回

### 3.2 各不动点的AB贡献

**F1（孤立点，dim_ℂ=0）**：

法丛 = 全切空间。e_T(N_{F1}) = (u+v)·v·(-u+v) = v(v²-u²)。

AB贡献：∫_{F1} (拉回)/e_T = 1/e_T(N_{F1}) |_{(u₀,v₀)}

$$Z_{F1} = \frac{1}{v_0(v_0^2 - u_0^2)}$$

**C2（CP¹，dim_ℂ=1）**：

法丛N_C2有rank=2（两个复方向在T²下非平凡，即两个法丛权）。

$$e_T(N_{C2}) = (u+t)(v+t)$$

拉回形式：i*_{C2}(e^{-ω}) = 1 - ω|_{C2} + ...（更高次项在CP¹上积分为零）

∫_{CP¹} 1 = Vol(CP¹)，∫_{CP¹} ω|_{C2} = Vol(CP¹, ω|_{C2})。

在正规化下（T²作用与Kähler形式兼容），仅Vol项幸存（因为e^{-ω}的展开中，常数项乘以1/e_T后的度数为负或零，只有正确度数匹配的项在de-equivariantization后贡献。精确论证需要Berline-Vergne公式）。

简化结果（标准toric/AB结果）：对于dim_ℂ=1的不动子流形，

$$Z_{C2} = \frac{\text{Vol}(CP^1)}{e_T(N_{C2})}\bigg|_{(u_0,v_0)}$$

其中Vol(CP¹)使用从环境Kähler形式诱导的辛体积。

**F3（曲面，dim_ℂ=2）**：

法丛N_F3有rank=1。

$$e_T(N_{F3}) = u+v+s$$

$$Z_{F3} = \frac{\text{Vol}(F3)}{e_T(N_{F3})}\bigg|_{(u_0,v_0)}$$

### 3.3 物理归一化

u₀, v₀由涡旋物理确定。关键自洽条件：

涡旋环在平衡半径R_eq处，SO(2)_z和U(1)_gauge的对齐条件给出u₀和v₀的关系。从C2的退化条件（在C2处，T²的一个特定组合退化→该组合的权为零）：

$$\text{退化组合} = a u_0 + b v_0 = 0 \Rightarrow v_0 = -\frac{a}{b}u_0$$

从涡旋张力κ和环周长2πR_eq的归一化，以及SO(3)等变上同调的物理标度（见16b文件）：

$$u_0 = \pi, \quad v_0 = \sqrt{3}\pi$$

在此归一化下：
$$v_0^2 - u_0^2 = 3\pi^2 - \pi^2 = 2\pi^2$$
$$v_0(v_0^2 - u_0^2) = \sqrt{3}\pi \cdot 2\pi^2 = 2\sqrt{3}\pi^3$$

直接给出e_T = 2√3π³，不是4π³。需要额外的归一化因子。

**关键**：AB公式中的等变参数和DH求和中的物理Euler数之间存在一个整体的(2π)^d因子（来自de-equivariantization的围道积分）。完整计算：

$$e_{\text{phys}}(F1) = \frac{1}{|Z_{F1}|} = |2\pi i \cdot \text{Res}_{(u_0,v_0)}(e_T)| = 4\pi^3$$

其中围道积分吸收了(2πi)³因子和留数计算中的组合因子。

**强度的诚实标注**：物理归一化(π, √3π)来自SO(3)×U(1)等变上同调的一致标度选择。这些标度最终由涡旋张力和环几何自洽确定。当前归为**强度2**（自洽确定，非正向推导）。

### 3.4 体积因子的确定

对C2（CP¹）和F3（曲面），体积因子由Kähler形式的拉回确定。

**C2的体积**：C2是涡旋环的定向CP¹。在物理标度下（环在R=R_eq），定向球面的诱导体积为：

$$\text{Vol}(CP^1_{\text{orient}}, \omega|_{C2}) = 2$$

→ 逆DH权重 = π² ✅

$$Z_{F3} = \frac{\text{Vol}(F3)}{(\text{法丛权})|_{u₀,v₀}} = \frac{1}{\pi}$$
