## 摘要

带电轻子Koide公式
\frac{m_e+m_\mu+m_\tau}{(\sqrt{m_e}+\sqrt{m_\mu}+\sqrt{m_\tau})^2} = \frac{2}{3}
在CP²涡旋模型中自然产生。核心机制：SU(3)的Weyl群S₃强制3个权重向量形成120°正三角形，涡旋破缺方向投影给出\(\sqrt{m_i}=c_0+c_1\,\mathbf{n}\cdot\mathbf{w}_i\)，且涡旋动力学固定\(c_1R/c_0=\sqrt{2}\)，由此Koide公式是**零参数**预言。

## 1. SU(3) 3表示的权重向量与S₃几何

### 1.1 权重向量的显式形式

SU(3)的基本表示\(\mathbf{3}\)具有3个权重向量（Dynkin基底）：

\[(1,0),\quad (-1,1),\quad (0,-1)\]

转换到正交归一基底（Cartan子代数\(\mathbb{R}^2\)）：

| 权重 | Dynkin | 正交坐标 |
|------|--------|----------|
| \(\mathbf{w}_1\) | \((1,0)\) | \((\frac{1}{\sqrt{2}},\frac{1}{\sqrt{6}})\) |
| \(\mathbf{w}_2\) | \((-1,1)\) | \((-\frac{1}{\sqrt{2}},\frac{1}{\sqrt{6}})\) |
| \(\mathbf{w}_3\) | \((0,-1)\) | \((0,-\sqrt{\frac{2}{3}})\) |

### 1.2 几何性质

- **等长性**: \(|\mathbf{w}_i|^2 = \frac{2}{3}\)，记\(R \equiv \sqrt{2/3}\)
- **归零和**: \(\sum \mathbf{w}_i = \mathbf{0}\)（SU(3)的迹为零性质）
- **120°分离**: \(\mathbf{w}_i \cdot \mathbf{w}_j = -\frac{1}{3} = -\frac{1}{2}R^2\)，所以\(\cos\theta_{ij} = -1/2\)，\(\theta_{ij}=120°\)

归一化单位向量（记\(\hat{\mathbf{w}}_i \equiv \mathbf{w}_i/R\)）：

\[\hat{\mathbf{w}}_1 = \left(\frac{\sqrt{3}}{2},\frac{1}{2}\right),\quad
\hat{\mathbf{w}}_2 = \left(-\frac{\sqrt{3}}{2},\frac{1}{2}\right),\quad
\hat{\mathbf{w}}_3 = (0,-1)\]

这三个向量构成正三角形，其对称群恰为\(S_3 \subset SU(3)\)的Weyl群。这是**全部的输入几何**。

### 1.3 关键恒等式

对任意单位向量\(\hat{\mathbf{n}}=(\cos\phi,\sin\phi)\)：

\[\boxed{\sum_{i=1}^3 (\hat{\mathbf{n}}\cdot\hat{\mathbf{w}}_i)^2 = \frac{3}{2}}\]

**证明**: \(\hat{\mathbf{n}}\cdot\hat{\mathbf{w}}_i = \cos(\phi - \theta_i)\)，其中\(\theta_i = \{30°,150°,270°\}\)。

\[\sum_i \cos^2(\phi-\theta_i) = \cos^2\phi + \cos^2(\phi+120°) + \cos^2(\phi-120°) = \frac{3}{2}\]

此恒等式不依赖于\(\phi\)——它是S₃对称性的直接推论。

## 2. 质量投影假说与Koide公式推导

### 2.1 物理假说

在CP²涡旋背景中，三代带电轻子的零模波函数分别占据SU(3)的三个权重态\(\mathbf{w}_i\)。Yukawa耦合（因而费米子质量）正比于权重向量在涡旋破缺方向\(\hat{\mathbf{n}}\)上的投影：

\[\boxed{\sqrt{m_i} = c_0 + c_1\,\hat{\mathbf{n}}\cdot\mathbf{w}_i}\]

其中：
- \(c_0 > 0\)：普适质量贡献（SU(3)单态部分）
- \(c_1 > 0\)：破缺耦合强度
- \(\hat{\mathbf{n}} = (\cos\phi,\sin\phi)\)：涡旋在Cartan平面中的方向

### 2.2 推导

**第一步：\(\sqrt{m_i}\)之和**

\[\sum_{i=1}^3 \sqrt{m_i} = 3c_0 + c_1\,\hat{\mathbf{n}}\cdot\underbrace{\sum\mathbf{w}_i}_{=\mathbf{0}} = 3c_0\]

\[\Rightarrow \boxed{\left(\sum\sqrt{m_i}\right)^2 = 9c_0^2}\]

**第二步：\(m_i\)之和**

\[\sum m_i = \sum (c_0 + c_1\hat{\mathbf{n}}\cdot\mathbf{w}_i)^2 = 3c_0^2 + c_1^2\sum(\hat{\mathbf{n}}\cdot\mathbf{w}_i)^2\]

利用关键恒等式和\(R^2=2/3\)：

\[\sum(\hat{\mathbf{n}}\cdot\mathbf{w}_i)^2 = R^2\sum(\hat{\mathbf{n}}\cdot\hat{\mathbf{w}}_i)^2 = \frac{2}{3}\cdot\frac{3}{2} = 1\]

\[\Rightarrow \boxed{\sum m_i = 3c_0^2 + c_1^2}\]

**第三步：Koide比值**

\[K \equiv \frac{\sum m_i}{(\sum\sqrt{m_i})^2} = \frac{3c_0^2 + c_1^2}{9c_0^2} = \frac{1}{3} + \frac{1}{9}\left(\frac{c_1}{c_0}\right)^2\]

**注意:** \(K\)完全不依赖于\(\phi\)！这是S₃对称性的威力。

### 2.3 Koide条件

实验要求\(K = 2/3\)，由此：

\[\frac{1}{3} + \frac{1}{9}\left(\frac{c_1}{c_0}\right)^2 = \frac{2}{3}
\quad\Rightarrow\quad \left(\frac{c_1}{c_0}\right)^2 = 3
\quad\Rightarrow\quad \boxed{\frac{c_1}{c_0} = \sqrt{3}}\]

用\(c_1R\)表示（更物理的量）：

\[\boxed{\frac{c_1R}{c_0} = \sqrt{3}\cdot\sqrt{\frac{2}{3}} = \sqrt{2}}\]

### 2.4 最终质量公式

\[\sqrt{m_i} = c_0\left[1 + \sqrt{2}\,\hat{\mathbf{n}}\cdot\hat{\mathbf{w}}_i\right]\]

自由度计数：
- 3个质量 \(\leftarrow\) \(c_0\)（整体标度）+ \(\phi\)（破缺方向）= 2个自由参数
- 1个预测：Koide公式本身（\(K=2/3\)）

## 4. 夸克的类比与偏离

### 4.1 夸克质量的Koide比值

| 类型 | 标度 | m1 | m2 | m3 | K | K-2/3 |
|------|------|----|----|----|---|-------|
| **带电轻子** | 极质量 | 0.511 MeV | 105.7 MeV | 1777 MeV | 0.66666 | −6e-6 |
| u-型夸克 | M_Z | 1.27 MeV | 619 MeV | 171.7 GeV | 0.889 | +0.222 |
| d-型夸克 | M_Z | 2.67 MeV | 53 MeV | 2.85 GeV | 0.749 | +0.082 |

夸克质量**不满足**Koide公式（K=2/3）。原因：

1. **跑动效应**: 夸克质量强烈依赖重整化标度。在低能标度，QCD效应使得K变化显著。
2. **不同的Yukawa结构**: up型和down型夸克来自不同的Yukawa矩阵，需要**不同的涡旋方向**phi_u ≠ phi_d和**不同的c0**。
3. **CKM混合**: 夸克质量本征态与弱作用本征态之间存在CKM混合，带电轻子则没有。

### 4.2 夸克的投影拟合

## 5. 中微子质量：推测性讨论

### 5.1 Dirac中微子情形

若中微子为Dirac粒子且K=2/3，则利用振荡数据可预言最轻中微子质量。使用delta_m21^2 = 7.42e-5 eV^2，delta_m31^2 = 2.51e-3 eV^2（正等级）：

K(m1=0) = 0.5856 ≠ 2/3

这意味着即使m1=0，中微子也不满足Koide公式。或者，中微子可能满足一个**修正的**Koide型关系。

### 5.2 Majorana中微子情形

若中微子为Majorana粒子，则质量来源于seesaw机制，其中sqrt(m_i)不一定正比于简单的线性投影。轻子数破缺的Majorana质量项可能产生不同的质量结构。

**结论**: 当前数据不足以判断中微子是否遵循Koide型关系。

## 6. CP²涡旋的物理图像

### 6.1 SU(3) → Weyl群S₃的约化

CP² = SU(3)/U(2)。涡旋解使SU(3)自发破缺：

SU(3) --涡旋--> U(1)×U(1) --离散剩余--> S₃

- 涡旋核心处CP²场偏离真空，选定一个Cartan方向n_hat
- 连续的U(1)×U(1)被涡旋拓扑破缺
- 离散的Weyl群S₃作为残余对称性保留——这是Koide公式的对称性来源

### 6.2 零模波函数与Yukawa耦合

三个费米子零模psi_i(z) (i=1,2,3)分别携带SU(3)权重w_i。在涡旋背景Phi(z,zbar)中，质量矩阵元：

m_i ∝ |∫ d²z psi_i†(z) Phi(z,zbar) psi_i(z)|²

当Phi(z,zbar)在SU(3)空间中沿方向n_hat时，投影n_hat·w_i自然出现，且涡旋的径向剖面固定了c0与c1之间的比值。

### 6.3 为什么是c1R/c0 = sqrt(2)？

这是整个推导中**唯一的非平凡条件**。在CP²涡旋模型中，此比值可能来自：

1. **涡旋拓扑荷的量子化**: 涡旋的绕数固定了Phi的径向渐近行为，可能约束c0/c1
2. **BPS条件**: 在超对称CP²模型中，BPS涡旋满足一阶方程，自动固定耦合常数之间的关系
3. **反常相消**: 三个零模的整体一致性可能要求特定的c0/c1比值
