## 一、F1 (R=0)：环塌缩为点

### 1.1 物理构型

涡旋环缩为一点。恢复完整SO(3)旋转对称。序参量Φ和规范场A具有球对称性。

### 1.2 切空间分解

M_vortex的6个实切方向在F1处按SO(3)×U(1)表示分解：

| 复方向 | 物理来源 | SO(2)_z权 | U(1)_phase权 | U(1)_helicity权 |
|:---|:---|:---:|:---:|:---:|
| z₁ | SO(3)/SO(2)_z ≅ S² | **±u** | 0 | 0 |
| z₂ | 全局相位旋转 | 0 | **+v** | 0 |
| z₃ | 径向呼吸(R缩放) | 0 | 0 | **+w** |

T² = SO(2)_z×U(1)_phase权：**{(±u,0), (0,+v), (0,0)}**

在u=v处：**{+v, +v, 0}** → T²下退化（一个零权）

**T³下非退化**：三权全非零。e_T(F1) = u·v·w。

## 二、F3 (R=R_max)：物理截断

### 2.1 F1-F3等价证明

**断言**：F3的T²权与F1相同。

**证明**：

F1处：S²切空间 = SO(3)/SO(2)_z的余切。SO(2)_z作用于其上为二维矢量表示 → 权±1。

F3处：涡旋在环横截面内的平移(x⊥, y⊥)。SO(2)_z（绕z轴旋转）使环上固定点处的横截面旋转 → (x⊥, y⊥) → (x⊥ cos θ − y⊥ sin θ, x⊥ sin θ + y⊥ cos θ)。这是SO(2)_z的**二维矢量表示** → 权±1。

两种情形都是SO(2)_z的同一个二维实表示 → **权完全相同**。

对比：
```
F1: S²切向 = SO(3)/SO(2)_z → SO(2)_z矢量 → 权±1
F3: 截面平移(x⊥,y⊥) → SO(2)_z矢量 → 权±1
                    ↓
              相同的SO(2)_z表示
                    ↓
              相同的T²权重
```

**F1-F3等价得证。** □

### 2.2 边界效应：为什么F3=π

F3在R=R_max处——这是物理边界（有限能量截断）。

APS边界公式：对带边流形，等变积分 =
$$\sum_F \int_F \frac{1}{e_T(N_F)} + \eta_{\text{boundary}}(0)$$

在F3处：
- 径向方向(R)是边界面法向 → **冻结**（Dirichlet边界条件）
- 剩下2个复方向中：
  - 截面平移方向 → 在边界处受约束 → 只有**1个活跃方向**
  - 相位方向 → 不受边界影响 → 活跃

**活跃方向计数：1（而非F1的3）**

$$\boxed{\text{F3贡献} = \pi \quad (\text{非}4\pi^3)}$$

η(0)=0来自边界法丛的对称权(±v→±w抵消)。

## 三、C2 (R=R_eq)：临界环

u=v处，e_T(F2) = 2v(v²−u²) = 0 → 呼吸模式权消失。

C2 ≅ CP¹（环的S²取向空间）为不动子流形。

法丛N_C2 = O(−1)⊕O(+1)（来自截锥多面体的面自交数s_lat=−1, s_trunc=+1）。

$$\boxed{\text{C2贡献} = \int_{\mathbb{CP}^1}\frac{1}{e_T(O(-1)\oplus O(+1))} = \pi^2}$$
