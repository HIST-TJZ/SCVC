
## G. SU(2) Dynkin指标比推导

SU(2)代数：生成元T^a = σ^a/2（基础表示）

基础表示：Tr(T^a T^b) = ½ δ^{ab}  →  T(fund) = 1/2

伴随表示：(T^a_adj)_{bc} = −i ε^{abc}
Tr(T^a_adj T^b_adj) = f^{acd} f^{bcd} = C_A δ^{ab} = 2 δ^{ab}  →  T(adj) = 2

规范动能项正则条件：
```
−½ Tr(F_μν F^{μν}) = −½ × T(R) × F^a F^a = −¼ F^a F^a (SM约定)
→ T(R) = 1/2
```

但SM约定是 −¼ F^a F^a（规范场动能项系数为1/4），且用基础表示：
```
−½ Tr_fund(F F) = −½ × (1/2) × F^a F^a = −¼ F^a F^a ✓
```

在伴随表示中：
```
−½ Tr_adj(F F) = −½ × 2 × F^a F^a = −F^a F^a = −4 × (−¼ F^a F^a)
```

所以伴随表示的自然动能项是基础表示的4倍。为了得到相同的正则归一化−¼F²：
```
1/g_fund² = 1/(4g_adj²)  →  g_fund = 2 × g_adj
```

等等，这里是g_fund = 2×g_adj，不是1/2。

仔细再推：
- 动能项系数C：S ⊃ −C ∫ F^a F^a，SM约定C=1/4
- KK给出S ⊃ −C_KK ∫ F^a F^a，C_KK = 1/(4g_KK²)
- SM约定：1/(4g_fund²) = 1/4 → g_fund = 1（这是生成元归一化的结果）
- 实际SM耦合g₂ ≠ 1，因为生成元的归一化被吸收进耦合常数的定义

更准确的说法：
SM拉氏量：L ⊃ −1/4 F_μν^a F^{a μν}
其中 F_μν^a = ∂_μ A_ν^a − ∂_ν A_μ^a + g₂ ε^{abc} A_μ^b A_ν^c

这里g₂是SU(2)规范耦合。生成元是T^a = σ^a/2。

KK约化给出：L ⊃ −C_KK ∫ F_μν^a(KK) F^{a μν}(KK)
其中 F_μν^a(KK) = ∂_μ A_ν^a − ∂_ν A_μ^a + f^{abc} A_μ^b A_ν^c
f^{abc}来自Killing矢量的李括号。

如果Killing矢量的归一化使得f^{abc} = ε^{abc}（标准su(2)结构常数），
那么对比SM：g₂ = 1（在KK归一化下）。

但SM的实际g₂(M_KK) = 0.514，所以需要因子。

关键：**SM的A_μ乘以生成元T^a = σ^a/2，而KK的A_μ乘以Killing矢量K^a。**
K^a和T^a有不同的归一化。

定义Killing矢量的归一化使得：
[K^a, K^b] = −ε^{abc} K^c（标准李括号）

SM生成元：[T^a, T^b] = i ε^{abc} T^c，T^a = σ^a/2

如果把A_μ^a(KK)与A_μ^a(SM)直接等同，则规范变换：
δA_μ^a(KK) = ∂_μ ε^a − ε^{abc} A_μ^b ε^c
δA_μ^a(SM)  = ∂_μ ε^a + g₂ ε^{abc} A_μ^b ε^c

所以g₂ = −1...但这只是符号约定。

动能项对比：KK给出的F_μν^a(KK)与SM的F_μν^a(SM)有不同系数。
详细推导见27号任务。

这里的关键结论是：**N₂=1/2**的严格群论推导需要在27号任务的基础上，
显式追踪KK规范场动能项的(2π)因子和生成元归一化。
