## 三、路径B：BPS涡旋解的直接论证（→ 95%+ 独立）

### 3.1 BPS方程的经典严格性

#### 3.1.1 Bogomolny分解

3D Abelian Higgs模型的静态能量泛函（在Weinberg极限e→0, β=λ/e²固定下）：

$$\mathcal{E} = \frac{1}{2}\int d^2x \left[ B^2 + |D_i\Phi|^2 + \frac{\lambda}{2}(|\Phi|^2 - v^2)^2 \right]$$

Bogomolny重排（使用恒等式|D_iΦ|² = |(D_1±iD_2)Φ|² ± B|Φ|² ∓ ε^{ij}∂_iJ_j）：

$$\mathcal{E} = \frac{1}{2}\int d^2x \left[ (B \mp (|\Phi|^2 - v^2))^2 + 2|(D_1 \pm iD_2)\Phi|^2 \right] + (\lambda-1)\int d^2x \frac{1}{2}(|\Phi|^2 - v^2)^2 \pm v^2 \int d^2x B$$

#### 3.1.2 λ=1时的能量下界

当λ=1时，第三项消失。前两项是正定的平方项。由此得到：

$$\mathcal{E} \geq \left| v^2 \int d^2x B \right| = |n| \cdot 2\pi v^2$$

其中n = (1/2π)∫B d²x 是涡旋的拓扑绕数。当且仅当以下BPS方程成立时等号成立：

$$(D_1 \pm iD_2)\Phi = 0, \quad B = \pm(|\Phi|^2 - v^2)$$

#### 3.1.3 λ≠1时BPS方程不存在

关键观察：二阶Euler-Lagrange方程

$$D_i D_i \Phi = \lambda \Phi(|\Phi|^2 - v^2)$$

能分解为一阶BPS方程当且仅当λ=1。证明：

将BPS方程(D_1±iD_2)Φ=0作用D_1∓iD_2得到：
$$(D_1 \mp iD_2)(D_1 \pm iD_2)\Phi = D_i D_i \Phi \mp i[D_1,D_2]\Phi = D_i D_i \Phi \mp B\Phi = 0$$

使用第二个BPS方程B=±(|Φ|²−v²)代入：
$$D_i D_i \Phi = \pm B \Phi = (|\Phi|^2 - v^2)\Phi$$

这正是λ=1时的二阶方程。当λ≠1时，右边是λΦ(|Φ|²−v²)，无法匹配。

**这是经典层面的严格数学结果：BPS方程→λ=1是充要条件。**

### 3.2 量子层面的BPS保护

核心问题：经典λ=1在量子修正下是否稳定？

#### 3.2.1 超对称保护机制

在具有N=2超对称的3D AH模型中，BPS涡旋是1/2 BPS态。超对称代数在中心荷扩展下：

$$\{Q_\alpha, Q_\beta^\dagger\} = 2\delta_{\alpha\beta}H + 2(\sigma_i)_{\alpha\beta} Z_i$$

其中Z_i是中心荷。对于涡旋态|n⟩：

$$\langle n| \{Q, Q^\dagger\} |n\rangle = 2\langle H \rangle_n - 2|Z|_n \geq 0$$

⇒ M_n ≥ |Z_n|。BPS态饱和此界：M_n = |Z_n|。

**质量=中心荷**不是近似——它是超对称代数的精确结果。只要超对称不被显式破缺，这个关系在量子层面精确成立。

#### 3.2.2 BPS质量与λ的关系

涡旋的BPS质量：

$$M_{\text{BPS}} = |n| \cdot 2\pi v^2$$

其中v²是Higgs VEV。在超对称理论中，v²由D-term势决定：

$$V_D = \frac{e^2}{2}\left(|\Phi|^2 - \xi\right)^2$$

其中ξ是Fayet-Iliopoulos参数。比较Standard型势：

$$\frac{\lambda}{2}(|\Phi|^2 - v^2)^2 \longleftrightarrow \frac{e^2}{2}(|\Phi|^2 - \xi)^2$$

⇒ λ = e²（即κ² = λ/e² = 1）是N=2 SUSY的自动结果——超对称要求规范耦合与标量自耦合通过D-term关联。

#### 3.2.3 非重整化定理

N=2超对称的非重整化定理保护以下关系：
- **D-term不被重整化**：D-term势 V_D 不受量子修正（除了一般的波函数重整化）
- **FI参数ξ不受重整化**（在Abelian情形中）
- **规范耦合e的跑动仅来自一圈**（在N=2中，高圈β函数为零）

这些定理的直接推论：**λ/e² = 1 在量子层面保持不变。** 任何对κ²=1的偏离都意味着超对称的显式破缺。

### 3.3 SCVC中的超对称来源

SCVC框架中的超对称来自7D超引力在M_vac = CP²×S¹上的紧致化：

- **7D N=1 SUGRA** → 紧致化在CP²×S¹上 → 4D N=1 SUSY（保形为N=2的部分子扇区）
- **涡旋扇区**保留了增强的超对称：涡旋世界体理论有(2,2)超对称（在1+1维），横向方向贡献额外超荷

涡旋环的BPS性质继承了母理论的超对称保护。具体地：
- 7D作用量中的Chern-Simons项和动能项在紧致化后产生涡旋的Wess-Zumino型耦合
- 该耦合在超对称变换下不变
- BPS条件λ=1是超对称Ward恒等式的直接推论

### 3.4 即使没有严格超对称：受保护的BPS

即使SCVC中不存在严格的N=2超对称（例如如果超对称在M_KK标度破缺），BPS条件仍有残存保护：

#### 3.4.1 软破缺情形

如果超对称在M_SUSY ≪ M_KK处软破缺，对λ的修正为：

$$\delta\lambda \sim \left(\frac{M_{\text{SUSY}}}{M_{KK}}\right)^2 \sim \left(\frac{10^3}{10^{17}}\right)^2 \sim 10^{-28}$$

但此时RG流（路径A）仍然保证IR收敛到κ²=1。

#### 3.4.2 技术自然性

即使没有超对称，λ=1具有增强的技术自然性：

## 四、路径C：超对称论证的完整展开（→ 95%+ 独立）

### 4.1 N=2超对称代数中的中心荷

3D N=2超对称代数（从4D N=1维度约化）：

$$\{Q_\alpha, \bar{Q}_\beta\} = 2\gamma^\mu_{\alpha\beta} P_\mu + 2i\epsilon_{\alpha\beta} Z$$

其中Z是中心荷——在涡旋扇区中Z = ∫T_{0i} dS^i（拓扑荷×VEV）。

对于单位绕数BPS涡旋：**Z = 2πv²**。

### 4.2 BPS界的超代数推导

从超对称代数直接推出BPS界：

$$\langle\psi| \{Q, Q^\dagger\} |\psi\rangle = \sum_\alpha |Q_\alpha|\psi\rangle|^2 \geq 0$$

代入{Q,Q†}=2H−2|Z|（对BPS表示）：
$$2\langle H \rangle - 2|Z| \geq 0 \Longrightarrow M \geq |Z| = 2\pi v^2$$

**这是代数的结果，不是动力学的。**它不依赖于作用量的具体形式、不依赖于微扰阶数、不依赖于耦合常数的大小。

### 4.3 λ=1 作为代数的自动推论

在N=2超对称规范理论中，标量势由D-term和F-term组成：

$$V = V_D + V_F = \frac{e^2}{2}(D^a)^2 + |F^i|^2$$

对于Abelian Higgs模型的超对称扩展（两个手征多重态Q和\tilde{Q}，带电荷±1），D-term为：

$$D = |Q|^2 - |\tilde{Q}|^2 - \xi$$

在对称破缺真空中\tilde{Q}=0, |Q|²=ξ，给出：

$$V_D = \frac{e^2}{2}(|Q|^2 - \xi)^2$$

比较非超对称的势能：

$$V_{\text{non-SUSY}} = \frac{\lambda}{2}(|\Phi|^2 - v^2)^2$$

这两个势能形式**完全一致**当且仅当λ = e²。在超对称理论中，这不是巧合——D-term的系数e²/2来自规范动能项−1/(4e²)F²的归一化，而标量动能项|DΦ|²的归一化固定了规范耦合在标量扇区也是e。因此：

$$\lambda = e^2 \quad \Longleftrightarrow \quad \kappa^2 \equiv \frac{2\lambda}{e^2} \cdot \frac{1}{2} \cdot 2 = 1$$

在N=2中，D-term的归一化是**刚性**的——它由超空间测度∫d⁴θ的约定唯一确定。没有可调参数。

### 4.4 非重整化定理的严格形式

#### 4.4.1 N=2的Seiberg-Witten型非重整化

N=2超对称规范理论的关键非微扰结果：

1. **低能有效作用量完全由预备势（prepotential）F(A)决定**
2. **预备势在微扰论中只能有一圈贡献**——所有高圈修正为零
3. **非微扰贡献（瞬子）由Seiberg-Witten曲线约束**

对于Abelian情形（N=2 SQED），预备势的精确形式为：

$$\mathcal{F}(A) = \frac{1}{2}\tau_0 A^2 + \frac{i}{4\pi} A^2 \ln\frac{A^2}{\Lambda^2}$$

其中τ₀=θ/(2π)+4πi/e₀²是裸耦合。**没有高圈修正**——这是N=2超对称的严格定理。

#### 4.4.2 D-term非重整化

D-term不被重整化是N=1超对称的标准结果。在N=2中该保护更强：
- FI参数ξ在Abelian情形中完全不被重整化
- D-term的二次形式不因量子修正而改变
- 因此λ/e²=1在量子层面是**精确的**

#### 4.4.3 对3D AH模型的具体应用

3D N=2 AH模型（从4D N=1维度约化）继承了非重整化定理：
- 规范耦合e₃²在1圈处有有限重整化（来自KK塔的积出）
- λ₃获得与e₃²精确相同的有限重整化（超对称要求）
- 比值κ²=2λ₃/e₃²在量子层面保持不变

这就是路径A的RG固定点论证的超对称版本：不是因为RG流推动κ²→1，而是因为**超对称从一开始就强制κ²=1，且量子修正保持此关系**。
