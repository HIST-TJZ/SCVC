## 第一部分：λ=1 的自洽性论证

### 1.1 传统困境

此前λ=1被视为独立假设：BPS极限需要λ恰好等于1，而C线论证只给了~75%置信度。问题是：为什么自然界会选择这个精确值？

### 1.2 新论证：框架自洽性强制的λ=1

**逻辑链**：

```
(1) SCVC框架成立的前提：α⁻¹ = DH求和(4π³+π²+π)
(2) DH求和要求模空间路径积分可以局域化
(3) 局域化要求Q-exact超对称结构
(4) Q-exact结构要求模空间是Kähler流形
(5) 涡旋模空间是Kähler ⇔ BPS极限 ⇔ λ=1
                    ↓
(6) 因此：λ=1不是额外假设——是(1)的自洽性条件
```

**详细论证**：

**(1)→(2)**：α⁻¹ = DH求和是SCVC框架的核心预言。DH求和通过AB局域化公式从不动点数据计算。

**(2)→(3)**：AB局域化要求路径积分可以被Q-exact形变。没有这个，路径积分不能简化为不动点处代数数据的求和。

**(3)→(4)**：Q-exact形变{ Q, V }的存在要求：
- 存在费米型超荷Q满足Q² = 0（或Q² = 玻色对称性生成元）
- 玻色作用量可以写为{ Q, … } + 拓扑项的形式
- 这要求靶空间是Kähler流形（复结构+辛结构+黎曼结构的兼容性）

对于涡旋模空间，Kähler结构等价于BPS条件。这来自BPS涡旋理论的标准结果：Manton的L²度规是Kähler的**当且仅当**涡旋满足一阶BPS方程。

**(4)→(5)**：在Abelian Higgs模型中，BPS方程存在**当且仅当**λ=1。这不仅是充分条件——是必要条件。如果在势能V(|Φ|) = (λ/2)(|Φ|²−v²)²中λ≠1，二阶场方程不能分解为一阶BPS方程，模空间失去Kähler结构。

**(5)→(6)**：因此，从SCVC框架的自洽性出发，λ=1是强制的。如果λ≠1，整个α推导链断裂——而这与框架的其他成功预言（fermion masses, Koide, sin²θ_W, δ_CP等）矛盾。

### 1.3 数学表达

Abelian Higgs模型的Bogomolny分解：

$$E = \int d^3x \left[ \frac{1}{2}B^2 + |D_i\Phi|^2 + \frac{\lambda}{2}(|\Phi|^2 - v^2)^2 \right]$$

$$= \int d^3x \left[ \frac{1}{2}(B \mp (|\Phi|^2 - v^2))^2 + |(D_1 \pm iD_2)\Phi|^2 \right] + (\lambda - 1)\int d^3x \frac{1}{2}(|\Phi|^2 - v^2)^2 + \text{边界项}$$

当λ=1时，最后一个积分消失。能量取下界（Bogomolny界）：E ≥ |Φ_top|·v²。BPS涡旋精确饱和此界。

当λ≠1时，(λ−1)∫(…)²项≠0。模空间上的L²度规获得非Kähler修正：

$$g_{i\bar{j}}^{(\lambda)} = g_{i\bar{j}}^{\text{BPS}} + (\lambda-1) \cdot \delta g_{i\bar{j}}$$

其中δg破坏了Kähler条件：dω ≠ 0。

这意味着超对称代数{ Q, Q† } = H不再成立——H获得非零的基态能量，Q²不再闭。

**结论：λ=1是模空间为Kähler的充要条件，而Kähler性是Q-exact局域化的前提。**

### 1.5 λ=1的"技术自然性"

即使λ=1在经典层面是自洽性条件，量子修正是否破坏它？答案：**不破坏**，因为λ=1处于BPS极限，具有增强的超对称保护。

在λ=1处，涡旋的玻色和费米激发谱配对，一圈量子修正抵消（非重整化定理）。这在超对称规范理论中有大量先例（Seiberg-Witten, Nekrasov等）。

具体地，涡旋有效作用量的圈展开：

$$S_{\text{eff}} = S_{\text{BPS}} + \hbar \cdot 0 + \hbar^2 \cdot (\text{可能的修正}) + \cdots$$

一圈修正为零因为玻色-费米抵消。二圈及以上修正被超对称Ward恒等式约束。在具有足够超对称的理论中，BPS条件在量子层面保持不变。

**SCVC中的超对称来源**：7D超引力在M_vac上紧致化后，保留了4D N=1（或更多）的超对称。涡旋扇区作为BPS孤子，继承了母理论的超对称保护。

## 第二部分：BPS涡旋环的线性化分析

### 2.1 涡旋环几何与坐标系

考虑3+1维Abelian Higgs模型（λ=1）中的静态BPS涡旋环。选取柱坐标系(ρ, φ, z)，环在ρ=R₀, z=0处。

在环的邻域使用局部环坐标系(s, r, θ)：
- s：沿环的弧长参数（环向坐标）
- r：环横截面内的径向距离
- θ：环横截面内的角向

度规：ds² = dt² + ds² + dr² + r²dθ²（在环的邻域近似，忽略了1/R₀的曲率修正）。

### 2.2 BPS方程

在环坐标系中，假设环向平移对称（∂/∂s = 0，对长直涡旋近似，当R₀远大于涡旋核尺寸时成立）：

$$(D_r \pm \frac{i}{r}D_\theta)\Phi = 0$$

$$B_s = \mp(|\Phi|^2 - v^2)$$

其中B_s是沿环向的磁场（环内的磁通方向）。

边界条件：
- r→∞：|Φ| → v, B_s → 0
- r→0：Φ → 0, B_s → 有限

这是标准的二维BPS涡旋方程。环向曲率引入1/R₀的修正。

### 2.3 三个不动点的线性化分析

#### F1：R₀→0（环塌缩）

当环半径趋于零，涡旋环退化为一个点。此时环向和径向的区分消失——系统具有**完全旋转对称SO(3)**。

在R₀=0处线性化BPS方程：

扰动δΦ = δΦ(r, θ, φ)（球坐标）。将δΦ展开为球谐函数。对T² = SO(2)_z × U(1)_phase作用：

- m_z = 0, ±1角动量模式（绕z轴）：对应的T²权重为±u
- 相位模式（全局U(1)旋转）：T²权重为+v
- 径向呼吸模式：T²权重为0（无角动量或相位变化）

在SO(3)对称下，m_z = ±1的两个模式简并。加上相位模式(+v)和径向模式(0)，我们得到3个复模方向（6个实方向）。

**T²权重在F1处：{+u, −u, +v, −v, 0, 0}**（取u=v的物理极限后变为{+v, −v, +v, −v, 0, 0}）。

等变Euler类（模去SO(3)增强因子4π）：

$$e_T(F1) \propto v \cdot (-v) \cdot v = -v^3$$

对应DH贡献 4π³。

#### C2：R₀=R_eq（临界环）

在R₀=R_eq处，环的呼吸模式（径向缩放）的T²权重消失。这是因为在临界耦合u=v下，空间旋转和相位旋转对环的净效应抵消。

线性化分析：
- 环向平移（沿s）：零模，对应CP¹方向
- 横截面内涡旋位置（两个方向）：有限的T²权重±2v
- 呼吸模式（R缩放）：在u=v处权重为零 → **这正是CP¹不动子流形的特征**

法丛N_C2的T²权重：{+2v, −2v}。

法丛分解：两个线丛L₁和L₂，分别带有权重+2v和−2v。在CP¹上，L₁ ≅ O(−1)，L₂ ≅ O(+1)（由环半径R沿CP¹的依赖关系决定：环的"倾斜"模式对应O(−1)，环的"平移"模式对应O(+1)）。

**确认：N_C2 = O(−1) ⊕ O(+1)。**

对应DH贡献 π²。

#### F3：R₀=R_max（大环极限）

在R₀ ≫ 涡旋核尺寸时，环曲率可以忽略。涡旋退化为标准二维BPS涡旋。模空间方向：
- 涡旋在横截面内的位置（2个实方向）：有限权重
- 全局相位：有限权重
- 环半径：在边界处"冻结"（物理截断）

在物理截断下，F3的3个法方向中只有1个保持"活跃"（其余两个被边界条件约束）。T²权重结构与F1**局部相同**（因为大环的局部几何与塌缩环在适当的对偶下等价——两者都是锥顶点），但矩映射值不同。

**确认：F1和F3在局部toric结构中等价（相同权集，不同矩映射位置）。**

对应DH贡献 π（截断面正则化后）。

### 2.4 线性化分析的数值框架

虽然完整非线性BPS涡旋环的数值解需要2D PDE求解器，但**线性化分析不需要完整解**——只需要在每个不动点附近展开到二阶。这是可以解析完成的。

#### F1处的二阶展开（R₀→0极限）

在R₀=0处，涡旋环塌缩。此时序参量具有球对称性：Φ = f(r)e^{iθ}（其中r是球坐标中的径向坐标）。

BPS方程简化为1D径向ODE：

$$f'' + \frac{2}{r}f' - \frac{1}{r^2}f = f(f^2 - 1)$$

（在v=1的归一化下）。这是标准的球对称涡旋方程（与't Hooft-Polyakov monopole类似，但标量场是带电的）。

线性化：f(r) = f₀(r) + δf(r)。零模δf对应径向呼吸模式（T²权重0）。角向激发（球谐Y_{1m}）给出权重±v的非零模。

#### C2处的二阶展开（R₀=R_eq极限）

在R=R_eq处，环是稳定的。线性化BPS方程在环坐标系下的展开：

令Φ = Φ₀(r,θ) + δΦ(r,θ,s)，其中Φ₀是BPS涡旋解。

δΦ的环向Fourier展开：δΦ = Σ_n e^{ins/R_eq} δΦ_n(r,θ)。

- n=0：环向均匀模式。包含横截面内位置模、相位模、呼吸模。
- n=±1：环向弯曲模式（倾斜）。在u=v处获得零权重。
- n≥2：高阶环向模式。非零权重。

**n=±1模式在u=v处权重为零** → 这精确对应CP¹不动子流形。法向（n=0截面模式）保持有限权重±2v。

### 2.5 提取T²权重

T² = SO(2)_z × U(1)_phase 的生成元在模空间上有明确的作用：

- SO(2)_z生成元：ξ_z = ∂/∂φ（绕z轴旋转）
- U(1)_phase生成元：ξ_ph = i(Φ∂/∂Φ − Φ*∂/∂Φ*)

在F1（R₀=0）处，模空间的6个实方向按T²权重分解：

| 模式 | SO(2)_z权重 | U(1)_phase权重 | 组合 |
|:---|:---:|:---:|:---|
| 环的方位角方向 (S²上的θ) | +1 | 0 | +u |
| 环的方位角方向 (S²上的φ) | −1 | 0 | −u |
| 全局相位旋转 | 0 | +1 | +v |
| 径向呼吸 | 0 | 0 | 0 |
| 环的位置 (x方向) | +1 | 0 | +u |
| 环的位置 (y方向) | −1 | 0 | −u |

在u=v的物理极限下：非零权重为{+v, −v, +v, +v, −v, 0}。三个复权重：{+v, −v, +v}。

在C2（R₀=R_eq）处，n=±1环向弯曲模式权重为零。保留的法向模式权重为{±2v}。

在F3（R₀=R_max）处，局部权重结构与F1相同（{+v, −v, +v}），但呼吸模式在边界处被约束。

## 第三部分：数值验证路线图

### 3.1 方程设置

BPS涡旋环的完整3D方程（柱坐标，轴对称）：

$$\frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\frac{\partial\Phi}{\partial\rho}\right) + \frac{\partial^2\Phi}{\partial z^2} - \frac{1}{\rho^2}\Phi = \Phi(|\Phi|^2 - 1)$$

（在v=1, e=1的归一化下，其中ρ是到z轴的距离）。

磁场：B_φ = (|Φ|² − 1)。

边界条件：
- 在环核心（ρ=R₀, z=0）：Φ=0（涡旋中心的零点）
- 在z轴（ρ=0）：∂Φ/∂ρ=0（正则性）
- 远场（r→∞）：|Φ|→1
- z→±∞：|Φ|→1

### 3.2 数值策略

1. **离散化**：柱坐标(ρ,z)的2D网格，在环核心附近加密
2. **迭代求解**：用逐次超松弛(SOR)或Newton-Krylov方法
3. **模空间扫描**：对每个R₀值求解Φ_{R₀}(ρ,z)，计算能量E(R₀)和L²度规g(R₀)
4. **不动点识别**：找dE/dR₀=0的点（C2），确认R₀→0（F1）和R₀最大（F3）的极限行为
5. **权重提取**：在C2附近计算Hessian矩阵的特征值，提取T²权重

### 3.3 可完成的解析验证

即使没有完整数值解，以下验证是可行的：

| 验证项 | 方法 | 状态 |
|:---|:---|:---|
| F1的SO(3)增强 | 球对称极限分析 | ✅ 解析可证 |
| C2的T²权重退化 | n=±1模式在u=v处的零权重 | ✅ 线性化分析可证 |
| F1和F3的局部等价 | 大R₀极限与R₀→0极限的线性化比较 | ✅ 渐近分析 |
| N_C2 = O(−1)⊕O(+1) | CP¹上法丛的Chern类计算 | ✅ 拓扑论证 |
