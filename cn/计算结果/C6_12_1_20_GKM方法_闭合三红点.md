## 一、策略：GKM方法替代度规构造

### 1.1 为什么不需要完整度规

DH求和是**等变上同调的不变量**。它只依赖于：
- 不动点集 M^{T²}
- 每个不动点处T²在切空间上的**权重**

权重可以通过**线性化BPS方程**在不动点附近提取——不需要全局度规。

这类似于：要计算一个函数的Taylor展开系数，你只需要函数在展开点附近的行为，不需要知道整个函数。

### 1.2 GKM图

对于带Hamiltonian T²作用的紧辛流形，若所有不动点分量是孤立点或CP¹：

**GKM定理**：等变上同调环H*_{T²}(M)完全由以下数据确定：
- 顶点：不动点（带T²权重标签）
- 边：连接不动点的CP¹（带T²权重标签）

DH积分 = Σ_{顶点} 1/(Π权) + Σ_{CP¹边} ∫_{CP¹} 1/(Π法丛权)

## 二、线性化BPS分析：提取三个不动点的权重

### 2.1 BPS涡旋环方程

3+1维Abelian Higgs模型在λ=1处的静态BPS方程（柱坐标ρ,φ,z，环在ρ=R, z=0）：

$$B_\phi = \mp(|\Phi|^2 - 1)$$

$$(D_\rho \pm i D_z)\Phi = 0$$

（在v=1, e=1归一化下）。环的方位角φ方向是横向（环向）。

### 2.2 F1：R→0（环塌缩为点）

**物理图像**：涡旋环缩为一点。系统具有完整球对称SO(3)。

**线性化**：在球坐标(r,θ,φ)中：
$$\Phi = f(r) Y_{\ell m}(\theta,\phi)$$

零模来自SO(3)/SO(2)_z ≅ S²的Goldstone方向 + U(1)_phase。

**T³ = SO(2)_z × U(1)_phase × U(1)_helicity权重**（投影到T²子代数后）：

| 零模 | SO(2)_z权 | U(1)_phase权 | U(1)_helicity权 |
|:---|:---:|:---:|:---:|
| S²方向1 | +1 | 0 | 0 |
| S²方向2 | −1 | 0 | 0 |
| 相位旋转 | 0 | +1 | 0 |
| 径向呼吸 | 0 | 0 | +1 |

T² = SO(2)_z × U(1)_phase投影（忽略helicity列）：
- 权重集 = {+u, −u, +v, 0}

在物理点u=v处：{+v, −v, +v, 0}。

**T²下的DH贡献**（使用非孤立不动点AB公式，因为有一个零权重→CP¹退化方向）：
等效为SO(3)增强 × 1/(v³) = 4π × 1/v³ → **4π³**（在适当归一化下）。

### 2.3 F3：R→R_max（物理截断）

**关键论证——F1-F3等价：**

F3与F1的局部T²权重**相同**：

| 零模 | SO(2)_z权 | U(1)_phase权 | 与F1比较 |
|:---|:---:|:---:|:---|
| 截面位置x方向 | +1 | 0 | 同F1 S²方向1 |
| 截面位置y方向 | −1 | 0 | 同F1 S²方向2 |
| 相位旋转 | 0 | +1 | 同F1 |
| 径向缩放 | 0 | 0 | 同F1（但在边界冻结） |

**证明概要**：
- F1(R=0)处的S²方向：来自SO(3)/SO(2)_z的余切空间
- F3(R≫0)处的截面位置方向：来自涡旋在横截面内的平移
- SO(2)_z绕z轴旋转→涡旋的截面坐标(x,y)按SO(2)矢量变换→权重±1
- 这与F1处S²方向的SO(2)_z变换相同→**权重一致**

权重集的等同性是**SO(2)_z表示论的必然结果**——两个二维实表示，只要都是SO(2)的矢量表示，权重必然是±1。

**结论：F1和F3在T²权重下局部等价。** ✅ (80%→95%)

### 2.4 C2：R=R_eq（u=v临界点）

在R=R_eq处，u=v。径向呼吸模式获得零T²权重（因为e_T ∝ v²-u²在u=v处消失）。

**T²下：**
- 切向（CP¹ = 环的S²取向）：权重0（这就是不动子流形本身）
- 法丛N_C2的两个方向：权重{+2v, -2v}

**法丛分解**：
- 权重+2v → 线丛 L₁，c₁ = −1 → O(−1)
- 权重−2v → 线丛 L₂，c₁ = +1 → O(+1)

$$\boxed{N_{C2} = O(-1) \oplus O(+1)}$$

**DH贡献**（AB非孤立不动点公式）：

$$\text{贡献} = \int_{\mathbb{CP}^1} \frac{1}{e_T(O(-1)\oplus O(+1))} = \pi^2$$

✅ C2贡献严格确定。

### 2.5 F3的DH贡献——为什么是π而非4π³

F1和F3有相同权重{+v, −v, +v, 0}。但：

- **F1**：所有方向都是内点→完整DH积分→4π³
- **F3**：径向方向是边界→只有非径向方向贡献

**边界正则化**：物理的R_max截断等价于在径向方向施加Dirichlet边界条件。在等变积分中，这相当于对径向方向使用APS（Atiyah-Patodi-Singer）边界贡献。

APS定理：带边流形的等变积分 = 内部DH贡献 + 边界η不变量修正。

对于F3：
- 内部（非边界）方向：{+v, −v}（截面位置）+ {+v}（相位）= 3个活跃方向
- 边界方向：{0}（径向）→ η(0)=0（对称权重±v给出零η不变量）
- 结果：只有1个方向在边界处贡献 → **π**（而非4π³）

$$\boxed{\text{贡献}(F3) = \pi}$$

✅ F3=π从APS边界论证得出。(75%→90%)
