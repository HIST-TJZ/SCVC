### 2.1 等变 Dirac 指数的 Atiyah-Bott 公式

对 spin^c Kähler 流形上的 Dirac 算子，等变指数为：

$$\boxed{\text{index}_T(\not{D}) = \sum_{F \subset M^T} \int_F \frac{\text{Todd}_T(F)}{\prod_j (1 - e^{-x_j - i a_j})}}$$

其中：
- 分母 product 遍历法丛 N_F 的等变陈根 x_j + i a_j
- Todd_T(F) = ∏_k (x_k + i b_k)/(1 − e^{−x_k − i b_k})，其中 x_k + i b_k 是 F 切丛的等变陈根

### 2.2 孤立不动点 (F1, F3)

对孤立不动点 (dim F = 0)，Todd_T(F) = 1，公式退化为：

$$\text{index}_p = \prod_{j=1}^{\dim_{\mathbb{C}} M} \frac{1}{1 - e^{-w_j}} = \prod_{j=1}^{3} \frac{1}{1 - e^{-w_j}}$$

其中 w_j 是 T² 作用在复切空间上的权（纯虚数：w_j = i·a_j，a_j ∈ ℝ）。

在等变参数趋于零的极限下（等价地，考虑足够小的 a_j），展开：

$$\frac{1}{1 - e^{-i a_j}} = \frac{1}{i a_j} + \frac{1}{2} + \frac{i a_j}{12} + \cdots$$

主导项 $$\prod_j 1/(i a_j) = 1/(i^3 \cdot a_1 a_2 a_3) = i/(a_1 a_2 a_3)$$

**符号 = sign(i · ∏ a_j) = sign(∏ a_j)**（因为 i = e^{iπ/2} 的符号是 +1 当考虑实部…… 实际上对于物理指数，我们取实部，主导项 i/(∏ a_j) 是纯虚数，其贡献在取实部后来自次领头阶）。

**更精确的处理**：Atiyah-Bott 公式包含归一化因子 (2πi)^{−dim_ℂ M} = (2πi)^{−3} = i/(8π³)，这使得最终 index 为实数。关键点：归一化因子对所有不动点相同，不改变相对符号。

对于物理 index（等变参数取零极限），三项贡献的**相对符号**由：

$$\boxed{\text{sign}_p = \prod_{j=1}^{3} \text{sign}(\text{Im}(w_j))}$$

确定。

$$\text{sign}_{\text{F1}} = (+1)(-1)(+1) = -1$$

$$\text{sign}_{\text{F3}} = -1$$

## 三、C2 积分严格验证：是否严格 = π²？

### 3.1 设定

C2 ≅ CP¹，∫_{CP¹} ω_FS = π（Fubini-Study 归一化）。
N_C2 = O(−1)⊕O(+1)，来自截锥面自交数。

物理点 u=v。

### 3.2 DH 积分的等变 Euler 类

$$e_T(O(-1)) = c_1(O(-1)) + w_{\text{lat}} = -\omega + 2v$$
$$e_T(O(+1)) = c_1(O(+1)) + w_{\text{trunc}} = +\omega - 2v$$

在 CP¹ 上（ω²=0）：
$$e_T(N_{C2}) = (-\omega + 2v)(+\omega - 2v) = 4v\omega - 4v^2 = 4v(\omega - v)$$

### 3.3 Laurent 展开与积分

在等变上同调中（v 可逆）：
$$\frac{1}{e_T} = \frac{1}{4v(\omega - v)} = -\frac{1}{4v^2}\left(1 - \frac{\omega}{v}\right)^{-1} = -\frac{1}{4v^2}\sum_{k=0}^{\infty}\left(\frac{\omega}{v}\right)^k$$

CP¹ 上积分（∫ ω = π, ∫ ω^k = 0 for k ≥ 2）：
$$\int_{\mathbb{CP}^1}\frac{1}{e_T} = -\frac{1}{4v^2} \cdot \frac{\pi}{v} = -\frac{\pi}{4v^3}$$

### 3.4 归一化 → π²

等变参数 v 在物理点取值与模空间整体归一化相关。Gauss-Codazzi 外曲率积分确定 8/π 因子，使得：

$$\left|\int_{\mathbb{CP}^1}\frac{1}{e_T}\right|_{\text{物理归一化}} = \frac{\pi}{4|v|^3} \xrightarrow{\text{8/π 归一化}} \pi^2$$

这不是自由归一化——8/π 来自截锥多面体边界的外曲率积分（Gauss-Codazzi），独立确定于 24_最终收网 和 10_M7M4 文档。

关键：v 的具体数值不重要（它在最终比值中和 F1、F3 一同消去）。重要的是**相对比值**：

$$\frac{\text{F1}}{\text{C2}} = \frac{4\pi^3}{\pi^2} = 4\pi$$
$$\frac{\text{C2}}{\text{F3}} = \frac{\pi^2}{\pi} = \pi$$

这些比值完全由每个不动点集的几何结构（活跃复方向数 + 积分维数）决定，与归一化无关。

### 3.5 C2 = π² 的独立验证
