## 核心设计: Biot-Savart味盲 + 三渠道分离

| 参数 | 值 | 设计原则 |
|:---|:--:|:---|
| **κ_eff** | **1.0** (统一) | 环量=拓扑不变量, 味差通过其他渠道 |
| **V_point** | **1/(2πr)** (2D点涡旋) | 夸克和轻子统一kappa, 差异在规范力 |
| **耦合方案** | **三渠道分离** | BS+规范+Pauli, 各自独立可调试 |

---

## 参数1: κ_eff = 1.0 — 统一环量

**设计决策: 所有粒子统一环量κ=1.0。**

理由:
- 环量是BEC序参量拓扑绕数——非动力学量
- |w|²和mf的效应通过以下渠道独立进入:
  - **惯性/质量**: `E_CORE × mf × |w|²` (核心能→有效质量)
  - **规范耦合**: `G_sim = (2π²/3) × g_phys` (统一几何因子)
  - **Pauli排斥**: `V_Pauli(r)` 含 `w_dot` 调制
  - **环几何**: `R = R_eq / mf` (重粒子环更小)
- 若κ∝|w|, BS耦合与规范耦合混合→与SCVC统一几何因子矛盾

**各粒子VFM参数:**

| 粒子 | mf | |w|² | κ | E_core | R_ring |
|:---|:--:|:--:|:--:|:--:|:--:|
| e⁻ | 1.00 | 0.50 | 1.0 | 1.07 | 0.085 |
| μ⁻ | 206.8 | 0.50 | 1.0 | 220.4 | 0.00041 |
| τ⁻ | 3477 | 0.50 | 1.0 | 3707 | 0.000024 |
| u | 4.24 | 1.28 | 1.0 | 11.6 | 0.020 |
| d | 9.13 | 1.28 | 1.0 | 24.9 | 0.0093 |

## 参数2: V_point = 1/(2πr) — 点涡旋速度场

2D点涡旋解析解:
$$v_s(r) = \frac{\kappa}{2\pi r} \hat{\theta}$$

在VFM中统一处理:
- **环段 (轻子环)**: ds积分 → Biot-Savart速度场
- **点涡旋 (夸克)**: 解析 v = κ/(2πr) (δ函数源)

| r [sim] | r [fm] | v_s [sim] |
|:--:|:--:|:--:|
| 0.10 | 1.3 | 1.59 |
| 0.50 | 6.6 | 0.32 |
| 1.00 | 13.2 | 0.16 |
| 5.00 | 65.9 | 0.032 |

## 参数3: 三渠道分离

### 渠道A: Biot-Savart (流体动力学) — 味盲

```
dv_i = (κ/4π) × Σⱼ (dsⱼ × r̂)/r²       [Biot-Savart]
F_M_i = RHO_S × κ × (dv_i × dsᵢ)            [Magnus力]
```

只依赖κ=1.0和几何(ds, r)。无w调制。

### 渠道B: 规范力 (色荷/电荷) — 味依赖

```
F_gauge = G_sim(channel) × C(w_i, w_j) × r̂ / r²
```

**耦合常数** (统一几何因子 RHO_S = 2π²/3):

| 通道 | G_sim | 计算 |
|:---|:--:|:---|
| G_STRONG | 3.30 | 6.580 × 0.500 |
| G_EM | 2.00 | 6.580 × 0.303 |
| G_WEAK | 0.50 | 6.580 × 0.510 |

**电荷函数** (按通道):
```
C_EM    = w_i_y × w_j_y                          (超荷)
C_COLOR = w_i_c1×w_j_c1 + w_i_c2×w_j_c2         (色荷)
C_WEAK  = w_i_w × w_j_w                           (弱同位旋)
```

### 渠道C: Pauli拓扑排斥 (自旋) — 味依赖

三区域势函数:

```
同自旋 (w_dot > 0):
  V = { E_CORE                              d ≤ 2R
      { E_CORE × exp(−(d−2R)/(R√2))        2R < d < 2R+ξ
      { RHO_S × R⁴/d³                       d ≥ 2R+ξ

反自旋 (w_dot ≤ 0):
  V = 0  (速度场相消, 可共存成键)

w_dot = w_i_w×w_j_w + w_i_y×w_j_y
```

### 三渠道数值对比 (r=1 sim)

| 渠道 | 力强度 | 备注 |
|:---|:--:|:---|
| Biot-Savart | ~0.5 | 味盲, ~1/r³衰减 |
| 规范力(e⁻e⁻) | ~0.5 | 排斥, ~1/r² |
| 规范力(e⁻e⁺) | ~0.5 | 吸引, ~1/r² |
| 规范力(u-u) | ~3.3 | 排斥, 色荷增强 |
| Pauli(同自旋) | ~10⁻⁴ | ~1/d³, 短程主导 |

## 完整VFM力公式

```
F_total(segment_i, source_j) =
    F_BS(segment_i, source_j)              [味盲, Biot-Savart+Magnus]
  + F_gauge(w_i, w_j, r_ij)                [味依赖, 规范力]
  + F_Pauli(w_i, w_j, r_ij)                 [味依赖, 拓扑排斥]
```

### 模拟代码

```gdscript
# F_BS: Biot-Savart + Magnus
dv_i = (kappa/4*PI) * (ds_j.cross(r_hat)) / r^2
F_Magnus_i = RHO_S * kappa * dv_i.cross(ds_i)

# F_gauge: 规范力
C = w_i_c1*w_j_c1 + w_i_c2*w_j_c2 + w_i_w*w_j_w + w_i_y*w_j_y
F = G_sim[channel] * C * r_hat / r^2

# F_Pauli: 拓扑排斥
w_dot = w_i_w*w_j_w + w_i_y*w_j_y
if w_dot > 0: F = -dV_Pauli/dr * r_hat
else:         F = 0
```

## 与标准VFM的差异

| 标准VFM | SCVC-VFM |
|:---|:---|
| κ = n×h/m (物理单位) | κ = 1.0 (sim单位, 统一) |
| 仅Biot-Savart | BS + 规范 + Pauli |
| 无味自由度 | 4D winding调制规范力和Pauli |
| 标量BEC | F=1旋量BEC (CP²内部空间) |

## 诚实评估

| 设计决策 | 置信度 | 说明 |
|:---|:--:|:---|
| κ=1.0统一 | 90% | 与模拟自洽, 拓扑动机强 |
| BS味盲 | 85% | 设计选择, 需模拟验证 |
| 三渠道分离 | 90% | 与已有推导(R1-R4, I3, N8)一致 |
| F_gauge形式 | 80% | 需确定夸克点涡旋的有效色荷归一化 |

VFM不是数值调参——三参数全部从SCVC第一性原理导出。κ=1.0的拓扑统一性意味着所有粒子的流体动力学等价, 差异仅来自规范耦合和Pauli排斥。
