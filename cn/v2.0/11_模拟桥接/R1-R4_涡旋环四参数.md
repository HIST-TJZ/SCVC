## R1: RHO_S = 2π²/3 (Magnus力系数)

```
GP Magnus力/单位线长 = 2π × v_rel
模拟耦合转换: G_sim/G_GP = (2π²/3)/(2π) = π/3
→ RHO_S = 2π × π/3 = 2π²/3 = 6.580

三重自洽:
  G_STRONG = 6.580 × 0.500 = 3.290 (sim: 3.30, 0.3%)
  G_EM     = 6.580 × 0.303 = 1.993 (sim: 2.00, 0.3%)
  RHO_S    = 6.580              (旧: 5.0, 31.6%)
```

**旧值5.0偏31.6%。** 同一个几何因子修复三个量。

## R2: R_eq = 0.085 (环平衡半径)

环自能: E_ring(R) = RHO_S×R/(4π) × [log(8R/ξ)−2]

dE/dR=0 → log(8R/ξ)=1 → **R_eq = ξ×e/8 = 0.0849**

物理: R_eq ≈ 1.12 fm。**环是局域拓扑缺陷（~1 fm），轨道半径~Bohr半径（~4000 sim单位）。**

## R3: Pauli排斥 = BEC动能重叠

旧: V=5.0×(R−d)/R (线性，数值凑的)

新: **V = RHO_S × exp(−(d−2R)/ξ) / (d/ξ+ε)**

物理：两涡旋环靠近→超流速度场叠加→动能密度升高→排斥。
指数衰减 = ξ尺度Yukawa屏蔽。

Pauli力 ~ RHO_S×e⁻¹/ξ ≈ 9.68 — 约G_EM的4.8倍。
**解释了为什么壳层排斥 ≫ 电磁吸引。**

在R_eq处: E_ring = RHO_S×R_eq/(4π)×(1−2) = **−0.0445**

## 模拟代码修改 (推荐)

```gdscript
const RHO_S = 6.5797    # 2π²/3
const R_EQ  = 0.0849    # ξ×e/8

func pauli_potential(r1, r2, d):
    var overlap = max(d − 2*max(r1.R, r2.R), 0)
    return RHO_S * w_dot * exp(−overlap/XI) / max(d/XI, 0.1)

func ring_self_energy():
    var lg = max(8*R/XI, 1.01)
    return RHO_S * R/(4π) * (log(lg) − 2 − 1/3)  # −1/3: CP²修正
```
