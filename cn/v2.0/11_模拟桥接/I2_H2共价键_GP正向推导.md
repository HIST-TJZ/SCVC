## §1: SCVC设定化学标度

化学键的两个全局标度:

```
Ry = ½α²m_ec² = 13.606 eV
a₀ = ℏ/(α m_e c) = 0.529 Å
```

SCVC不把这些当作实验输入:

```
α⁻¹ = 4π³+π²+π = 137.036  (DH求和, 2.2 ppm)
m_e = H₀^(1/3)涡旋标度 → 0.511 MeV (−0.4%)
```

**化学键为什么是eV量级而非MeV或neV？因为α来自π³的多项式。**
**化学键为什么是Å量级而非fm或mm？因为a₀来自α和m_e的比值。**

这是SCVC对化学最深刻的贡献 — 不是替代量子化学, 而是解释其参数。

## §2: 经典GP模型的边界

### 能做到的

1. **标度正确**: 经典Coulomb势 `V_C = −G_EM/r` 给出能量 ~G_EM/a₀ ~ Ry ~ eV
2. **键长量级**: 不确定原理 `Δx·Δp ~ ℏ` + Coulomb → d ~ ℏ²/(m e²) ~ a₀ ~ Å
3. **定性趋势**: 更强的核电荷 → 更短的键 → 更大的键能

### 做不到的

1. **精确键长**: 经典模型无稳定极小 (H原子会塌缩)
2. **结合能**: 需要量子动能 + 交换关联
3. **Ampère力的直接应用**: 在Å距离上, 经典点环Ampère力 ~10⁻¹³ eV (完全可忽略)

### Ampère力的正确角色

```
经典 (点环, 0.74 Å): V_Amp ≈ −10⁻¹³ eV  → 无影响
量子 (离域化环):    V_Amp ~ −(1-5) eV  → 替代交换积分
```

Ampère力在SCVC中替代QM交换积分的机制不是经典点环力, 而是两个离域化环的量子相干耦合。这需要环波函数的显式重叠积分 — 等价于QM的交换积分计算。

## §3: 经典标度论证 — 键长~1 Å

不确定原理 + Coulomb 极小化:

```
E(d) ~ −G_EM/d + ℏ²/(2m_eff d²)
dE/dd = 0 → d_eq = ℏ²/(m_eff × G_EM)
```

在sim单位中, 有效质量从Magnus动力学确定:
```
m_eff = F_M_per_v × r/v = F_M_per_v² × r³/G_EM
     = 79.0² × r³/2.00 = 3.12×10³ × r³
```

对r ~ a₀_sim = 4017 sim:
```
m_eff ~ 2.0×10¹⁴ sim
d_eq ~ ℏ_sim²/(m_eff × G_EM) ~ 0.17/(2×10¹⁴×2) ~ 4×10⁻¹⁶ sim
```

这个结果(~10⁻¹⁶ sim ~ 10⁻³⁰ Å)显然不对。问题在于m_eff在sim单位中的定义与ℏ_sim的标度不匹配。这反映了经典Magnus动力学在原子尺度上的不自洽 — 需要完整的量子处理。

**正确结论: 标度论证给出键长~a₀~Å, 但经典模型不能自洽地达到这个结果。** 需要量子力学 (或等效的GP波函数方法)。

## §4: 模拟场景参数 — 替代手写值

### 标度假定

模拟世界需要定义1 sim unit对应的物理长度。两个选项:

**选项A: 环核心尺度 (1 sim ≈ 1.3×10⁻¹⁴ m ≈ 13 fm)**
```
Bohr半径 a₀ ≈ 4017 sim
H₂键长 0.741 Å ≈ 5626 sim
```
→ 单屏幕可见原子轨道, 但环核心 (~0.085 sim) 为一个像素点

**选项B: 原子尺度 (1 sim ≈ 0.01 Å = 10⁻¹² m)**
```
Bohr半径 a₀ ≈ 53 sim
H₂键长 0.741 Å ≈ 74 sim
```
→ 原子轨道和键长在方便尺度, 但环核心 (~8.5×10⁻⁴ sim) 不可见

### 推荐参数 (选项B)

```
// H₂场景初始化
const D_H2: float = 74.1          // 质子间距 (0.741 Å)
const PROTON_CHARGE: float = 2.0  // G_EM (sim Coulomb单位)

// 质子位置
proton1.position = Vector3(0, 0, -D_H2/2)
proton2.position = Vector3(0, 0, +D_H2/2)

// 电子环 (反自旋单重态)
ring1.mass_factor = 1.0
ring1.radius = 0.085             // 环核心半径 (fm尺度, 量子离域化~26 sim)
ring1.normal = Vector3(0, 0, 1) // 环面∥xy, 法向量∥键轴
ring1.w_w = -0.5; ring1.w_y = -0.5  // 自旋↑
ring1.position = Vector3(0, 0, -D_H2/4)  // 环中心在质子间

ring2.mass_factor = 1.0
ring2.radius = 0.085
ring2.normal = Vector3(0, 0, 1)
ring2.w_w = +0.5; ring2.w_y = +0.5  // 自旋↓ (反平行)
ring2.position = Vector3(0, 0, +D_H2/4)

// 环速度: 约束轨道 (非实时Magnus, 轨道周期10¹³-10¹⁷ sim帧)
ring1.center_velocity = Vector3.ZERO  // 准静态
ring2.center_velocity = Vector3.ZERO

// Pauli排斥: w_dot = ring1.w_w*ring2.w_w + ring1.w_y*ring2.w_y
// = 0.25 + 0.25 = 0.5 (同向?), 不对:
// ring1: (-0.5,-0.5), ring2: (+0.5,+0.5) → w_dot = -0.5 (反向!)
// 反向 → V_Pauli = 0 (无排斥, 可共存) ✓
```
