## 推导

### CP²的等距群

CP² = SU(3)/U(2)是齐性Kähler流形。其等距群=SU(3)，最大紧子群。

Fubini-Study度规在SU(3)作用下不变。

### KK约化 → 规范对称性

7D → 4D KK约化后，CP²的等距群成为4D规范群:

```
Isom(CP²) = SU(3) → 4D SU(3) 规范理论
```

### 自发破缺: SU(3) → SU(2)×U(1)

涡旋-反涡旋对凝聚 → 类似希格斯机制破缺:

- 凝聚选择CP²中的一个点(真空方向)
- 该点的稳定子群 = SU(2)×U(1)
- 对应标准模型电弱规范群
