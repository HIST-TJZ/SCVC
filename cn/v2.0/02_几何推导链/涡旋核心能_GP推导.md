## 推导

### GP方程

SCVC真空=F=1旋量BEC。涡旋的径向序参量f(r)满足无量纲GP方程:

$$f'' + \frac{f'}{r} - \frac{f}{r^2} + f - f^3 = 0$$

边界条件:
- f(0) = 0（涡旋核心：序参量消失）
- f(∞) = 1（远离核心：恢复到基态）

唯一自由参数c₁ = f'(0)。通过打靶法确定:

```

```

### 涡旋能量泛函

n=1涡旋 ψ(r,θ) = f(r)·e^(iθ) 的GP自由能（每单位长度）:

$$E_{\text{vortex}} = 2\pi \int_0^\infty r\,dr \left[ \frac{1}{2}(f')^2 + \frac{f^2}{2r^2} + \frac{1}{4}(1-f^2)^2 \right]$$

| 项 | 物理含义 | 占比 |
|:---|:---|:--:|
| ½(f')² | 序参量跨核心变化的梯度能 | 25.2% |
| f²/(2r²) | 超流环量~1/r的运动能 | 35.9% |
| ¼(1−f²)² | 偏离基态|ψ|=1的势能代价 | 38.9% |

### 核心 vs 尾部分离

离心项在大r处: f²/(2r²) → 1/(2r²)，积分 → π·log(R_max/R_min)——对数发散。

**这是2D涡旋的经典对数发散，代表涡旋间的长程相互作用。** 模拟中的成对项 `G·(wᵢ·wⱼ)·log(1+r²/ξ²)` 已经捕捉了这部分。因此E_CORE仅取r<ξ的核心区:
