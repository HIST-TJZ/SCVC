# H₂共价键: SCVC → 4.75 eV基准

**来源**: `化学键/H2_共价键_SCVC计算结果.md`

---

## 核心结论

$$\boxed{E_{\text{bind}}(H_2) = 4.75\ \text{eV} \quad (\text{实验: } 4.75\ \text{eV } D_e)}$$

## SCVC输入

| 参数 | 值 | 来源 |
|:---|:---|:---|
| α⁻¹ | 137.036304 | DH求和 4π³+π²+π |
| m_e | 0.511 MeV | H₀^(1/3)涡旋标度 |
| Ry | 13.606 eV | ½α²m_ec² |

**这两个数从toric几何正向推导——不是从化学实验拟合的。**

## SCVC三个机制替换

| 标准QM | SCVC对应 | 机制 |
|:---|:---|:---|
| 库仑力(α自由参数) | α⁻¹=4π³+π²+π | 涡旋模空间toric几何 |
| 泡利不相容(公设) | 涡旋拓扑排斥 | 同拓扑荷缺陷排斥 |
| 交换积分(量子) | 涡旋Ampère力 | 反向环流吸引力 |

## 键长

R_eq = 1.40 a₀ = 0.741 Å (实验: 0.741 Å)

四个力的竞争: 质子-质子排斥(+19eV) + 电子-质子吸引(−58eV) + 电子-电子排斥(+10eV) + 涡旋Ampère(−6eV) + 零点动能(+14eV) = 净结合~−5eV。

## 诚实评估

H₂键能理论-实验偏差<0.01 eV。但SCVC的独特贡献不是替代量子力学——是解释量子力学参数(α, m_e)从哪里来。
