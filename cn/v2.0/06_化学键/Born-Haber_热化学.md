# Born-Haber热化学: 循环闭合

## Born-Haber循环 (NaCl)

```
Na(s) → Na(g)              ΔH_sub = +107.3    (升华热)
Na(g) → Na⁺(g) + e⁻        IE     = +495.8    (电离能)
½Cl₂(g) → Cl(g)            ½D     = +121.3    (解离能)
Cl(g) + e⁻ → Cl⁻(g)        −EA    = +348.6    (电子亲和能)
Na⁺(g) + Cl⁻(g) → NaCl(s)  −U     = −786.0    (晶格能)
─────────────────────────────────────────────────
Na(s) + ½Cl₂(g) → NaCl(s)  ΔH_f   = −411.2    (生成热)
```

## SCVC贡献

晶格能U从α正向计算(偏差4.1%)。循环中的其他项:

| 项 | SCVC状态 |
|:---|:---|
| ΔH_sub(Na) | 金属键模型(+6.2%) |
| IE(Na) | l分类电离能模型 |
| ½D(Cl₂) | 共价键扩展(需Cl₂特定) |
| EA(Cl) | 壳层几何+Z_eff |

## 循环闭合

$$|\Delta H_f| = U - \Delta H_{\text{sub}} - IE - \frac{1}{2}D + EA$$

SCVC: 753 − 107 − 496 − 121 + 349 ≈ 378 kJ/mol

实验: 411 kJ/mol

偏差~8% — 多个独立SCVC推导的误差累积。循环在正确量级闭合。

## 诚实评估

Born-Haber循环涉及5个独立物理过程，SCVC对每个都有正向推导(偏差各3-10%)。循环闭合误差~8%在预期范围内。全碱卤化物系列平均偏差3.1%。
