# Slater屏蔽常数: 几何推导 → σ=0.3477 (−0.67%)

**来源**: `化学键/09_Slater常数_几何推导结果.md`

---

## 核心结论

| Slater常数 | 经验值 | SCVC几何值 | 偏差 |
|:---|:--:|:--:|:--:|
| σ_1s | 0.30 | **0.3125** = 5/16 | +4.2% |
| σ_same (n=2) | 0.35 | **0.3477** | **−0.67%** |
| σ_same (n=3) | 0.35 | **0.3561** | +1.8% |
| σ_n-1 (Na) | 0.85 | **0.915** | +7.7% |
| σ_n-1 (K) | 0.85 | **0.862** | +1.4% |

**Slater规则(1930)不是经验的——它们是氢原子Coulomb积分的几何必然。**

## 核心公式

从变分原理:

$$\sigma_{\text{same}}(nl) = \frac{F^0(nl,nl)}{2 \cdot \langle 1/r \rangle_{nl}}$$

其中F⁰(nl,nl)是单极Coulomb积分。

## 关键结果

| 轨道 | F⁰(Z=1) | ⟨1/r⟩ | σ_same |
|:---|:--:|:--:|:--:|
| 1s | 0.6250 | 1.000 | **0.3125** = 5/16 |
| 2s | 0.1504 | 0.250 | 0.3008 |
| 2p | 0.1816 | 0.250 | 0.3633 |
| **n=2加权平均** | | | **0.3477** (2s²+2p⁶) |

Slater用0.35覆盖n=2。SCVC几何: 0.3477。偏差仅−0.67%。

## 物理意义

SCVC不需要任何经验屏蔽常数。所有Slater规则从类氢波函数+Coulomb积分正向推导。唯一的输入是α(设定库仑力强度)和m_e(设定长度标度)——两者均从SCVC几何推导。
