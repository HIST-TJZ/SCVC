## 二、任务A：ρ_s 与 m_constituent 的关系

### 出发方程

假设电子质量全部来自涡旋能量：

```
m_e c² = (1/2) ρ_s κ² r_e |ln(r_e/a)|
```

其中环流量子 κ = h/m_c = 2πℏ/m_c。

### 推导

```
m_e c² = (1/2) × ρ_s × (4π²ℏ²/m_c²) × r_e × |ln|
ρ_s = (m_e c² × m_c²) / (2π²ℏ² r_e |ln|)
```

代入 r_e = αℏ/(m_e c)：

```
ρ_s = m_e² c³ m_c² / (2π² α ℏ³ |ln(r_e/a)|)
```

### 最终公式

> **ρ_s = K × m_c²**

其中 K = m_e² c³ / (2π² α ℏ³ |ln|) = **8.699×10⁶⁸ (SI)**

### 核心发现：涡旋刚度 S 的不变性

定义涡旋刚度 S ≡ ρ_s κ²：

```
S = ρ_s κ² = K m_c² × 4π²ℏ²/m_c² = 4π²ℏ² K = 2 m_e c² / (r_e |ln|)
```

> **S = 3.819×10² J/m（在所有 m_c 选择下保持不变）**

这意味着：**m_e 和 r_e 共同确定的不是 ρ_s，而是组合量 S = ρ_s κ²**。ρ_s 和 κ 各自的取值可以任意变化，只要它们的乘积等于 S。这是一个简并自由度。

| m_e | 9.06×10⁻³¹ | 7.14×10⁸ | 1.39×10⁻⁸⁸ | 7.31×10⁻⁴ | 1.86×10¹² | 5.64×10²⁶ |

| 10×M_Pl | 2.18×10⁻⁷ | 4.12×10⁵⁵ | 7.99×10⁻⁴² | 3.04×10⁻²⁷ | 7.73×10⁻¹² | 2.35×10³ |
| 10²×M_Pl | 2.18×10⁻⁶ | 4.12×10⁵⁷ | 7.99×10⁻⁴⁰ | 3.04×10⁻²⁸ | 7.73×10⁻¹³ | 2.35×10² |

| **~0.5g** | **5.11×10⁻⁴** | **2.17×10⁶²** | **4.22×10⁻³⁵** | **1.33×10⁻³⁰** | **3.29×10⁻¹⁵** | **≈1.0** |

> 注：ξ = 1/√(8πρ_s ℓ_Pl)，假设散射长度 a_s ~ ℓ_Pl，此时 ξ 与 m_c 无关（仅由 ρ_s 决定）。

## 三、任务B：检验 ρ_s = k × H₀²/G_N

若假设 ρ_s 正比于宇宙学密度标度：

| m_c | ρ_s (kg/m³) | k = ρ_s / (H₀²/G_N) |
|:---|:---|:---|
| M_Pl | 4.12×10⁵³ | **5.72×10⁷⁸** |
| m_e | 7.14×10⁸ | **9.91×10³³** |
| 10⁴×M_Pl | 4.12×10⁶¹ | **5.72×10⁸⁶** |

**k 在任何自然的 m_c 选择下都不可能是 O(1)。**

这是因为 ρ_s 单独不是由 m_e 确定的物理量——只有 S = ρ_s κ² 是固定的。将 ρ_s 与 H₀²/G_N 比较是类别错误：H₀²/G_N 是质量密度 [M/L³]，而 S = ρ_s κ² 是力 = 能量/长度 [ML/T²]。SCVC 的"刚度"不是 ρ_s 而是 S。

## 四、任务C：SCVC 的 "G₀ 等价物"

### 弹性理论的放大机制

```
G₀ = ρ_c × c²
  ρ_c ~ 3H₀²/(8πG_N) ~ 9×10⁻²⁷ kg/m³
  c² ~ 9×10¹⁶ (m/s)²
  → G₀ ~ 8×10⁻¹⁰ Pa
```

c² 因子将微小的宇宙学质量密度放大了 ~10¹⁷ 倍成为剪切模量。

### SCVC 中的对应量

在 SCVC 中，涡旋能量的形式是：

```
E = (1/2) × (ρ_s κ²) × r_e × ln(a/r_e)
      ^^^^^^^^
      这就是刚度参数
```

> **S = ρ_s κ² = 3.819×10² J/m**

S 的量纲是 [能量/长度] = [力] = [ML/T²]，与弹性理论的 G₀ × L²（力）类似。

**关键区别**：弹性理论中 G₀ = ρ_c c²，ρ_c 是固定的（由宇宙学决定），c² 是放大因子。SCVC 中 S = ρ_s κ² 是固定的（由 m_e 决定），ρ_s 和 κ² 通过 m_c 相互"分配"这个刚度。

```
S = ρ_s × κ²
  = (K m_c²) × (4π²ℏ²/m_c²)
  = 4π²ℏ² K = 常数
```

**不存在单一的"放大因子"**：如果 m_c 大，则 ρ_s 大但 κ² 小；如果 m_c 小，则 ρ_s 小但 κ² 大。乘积恒定。

### 候选物理量

| 量 | 表达式 | 量纲 | 是否由 m_e 固定？ |
|:---|:---|:---|:---|
| 涡旋刚度 S | ρ_s κ² | [ML/T²] | ✅ 固定 |
| 能量密度 | ρ_s c² | [M/(LT²)] | ❌ 取决于 m_c |
| 愈合长度 ξ | 1/√(8πρ_s a_s) | [L] | ❌ 取决于 m_c |
| 声速 c_s | √(S a_s/(πm_c)) | [L/T] | ❌ 取决于 m_c |

## 五、任务D：量纲分析

### 推导

要求从 E_vortex ∝ ρ_s κ² r_e 和 m_e ∝ H₀^(1/3) 确定 ρ_s ∝ H₀^p ℏ^q G_N^r c^s 的指数。

步骤：
1. m_e ∝ H₀^(1/3) ℏ^(2/3) G_N^(-1/3) c^(-1/3)
2. r_e ∝ ℏ/(m_e c) ∝ H₀^(-1/3) ℏ^(1/3) G_N^(1/3) c^(-2/3)
3. m_e c² ∝ (1/2) ρ_s κ² r_e → ρ_s ∝ m_e² m_c² c⁵ / (ℏ³)

代入 m_e 的表达式：

```
ρ_s ∝ m_c² × H₀^(2/3) ℏ^(4/3) G_N^(-2/3) c^(-2/3) × c³ / ℏ³
     ∝ m_c² × H₀^(2/3) ℏ^(-5/3) G_N^(-2/3) c^(7/3)
```

### 一般情况（m_c 未指定）

| 指数 | 值 |
|:---|:---|
| **p (H₀)** | **2/3** |
| **q (ℏ)** | **−5/3** |
| **r (G_N)** | **−2/3** |
| **s (c)** | **7/3** |

完整表达式（含数值系数）：

> **ρ_s = 0.003319 × m_c² × H₀^(2/3) ℏ^(-5/3) G_N^(-2/3) c^(7/3)**

### 特例：若 m_c = M_Pl

M_Pl = √(ℏc/G_N) → m_c² = ℏ G_N^(-1) c

```
ρ_s ∝ ℏ G_N^(-1) c × H₀^(2/3) ℏ^(-5/3) G_N^(-2/3) c^(7/3)
     = H₀^(2/3) ℏ^(-2/3) G_N^(-5/3) c^(10/3)
```

| 指数 | 值 |
|:---|:---|
| **p** | **2/3** |
| **q** | **−2/3** |
| **r** | **−5/3** |
| **s** | **10/3** |

量纲验证：
- [H₀^(2/3)] = T^(-2/3)
- [ℏ^(-2/3)] = M^(-2/3) L^(-4/3) T^(2/3)
- [G_N^(-5/3)] = M^(5/3) L^(-15/3) T^(10/3) = M^(5/3) L^(-5) T^(10/3)
- [c^(10/3)] = L^(10/3) T^(-10/3)
- 乘积：M¹ L^(-4/3-5+10/3) = M¹ L^(-9/3) = M¹ L⁻³ ✓

数值（m_c = M_Pl 时）：ρ_s = **4.12×10⁵³ kg/m³**

## 六、物理约束分析

假设散射长度 a_s ~ ℓ_Pl，则：

### 约束 1：愈合长度 ξ < 涡旋核心 a（涡旋图像成立）

```
ξ = 1/√(8π ρ_s ℓ_Pl) < a = 3.293×10⁻¹⁵ m
→ ρ_s > 1/(8π ℓ_Pl a²) = 2.27×10⁶² kg/m³
→ m_c > √(ρ_min/K) = 5.11×10⁻⁴ kg ≈ 2.35×10⁴ M_Pl
```

### 约束 2：声速亚光速

```
c_s = √(S ℓ_Pl/(π m_c)) < c
→ m_c > S ℓ_Pl/(π c²) = 2.19×10⁻⁵⁰ kg
```

此约束极弱，不构成实际限制。

### 约束 3：超流密度低于普朗克密度

```
ρ_s < ρ_Pl
→ m_c < √(ρ_Pl/K) = 7.70×10¹³ kg ≈ 3.54×10²¹ M_Pl
```

### 合并约束

```
5.11×10⁻⁴ kg < m_c < 7.70×10¹³ kg
2.35×10⁴ M_Pl < m_c < 3.54×10²¹ M_Pl
```

**m_c 存在允许范围，但下界是宏观量级（≈0.5克）。**

## 七、最终答案

### ρ_s 最可能的物理表达式

> **ρ_s = 3.319×10⁻³ × m_c² × H₀^(2/3) ℏ^(-5/3) G_N^(-2/3) c^(7/3)**

这个公式是**严格的**——所有指数由量纲分析和 m_e ∝ H₀^(1/3) 确定，数值系数由代数推导给出。

### 但是：ρ_s 没有被单独确定

核心结论：
- **S = ρ_s κ² = 3.819×10² J/m 是由电子质量和涡旋几何唯一确定的物理量**
- **ρ_s 和 κ 各自没有被确定，存在一个简并自由度 m_c**
- 打破简并需要独立确定 m_constituent 或 κ

### 物理上自洽的选择

如果采用 ξ < a 作为涡旋图像的最低要求，m_c **必须大于 ~0.5 克**（≈2.35×10⁴ M_Pl）：

| 参数 | 数值（ξ=a 阈值） | 量级评价 |
|:---|:---|:---|
| m_c | 5.11×10⁻⁴ kg | 宏观（半克量级） |
| ρ_s | 2.17×10⁶² kg/m³ | 远低于 ρ_Pl（~10⁻³⁵ ρ_Pl） |
| κ | 1.33×10⁻³⁰ m²/s | 极小的环流量子 |
| ξ | 3.29×10⁻¹⁵ m | = a，涡旋图像临界成立 |
| c_s | 1.98×10⁻¹⁵ m/s | 极度亚光速（~10⁻²³ c） |

### 与弹性理论的对比

| | 弹性理论 | SCVC |
|:---|:---|:---|
| 刚度参数 | G₀ = ρ_c c² | S = ρ_s κ² |
| 由什么固定 | ρ_c 由宇宙学固定，c² 是放大因子 | S 由 m_e 固定 |
| ρ 是否被单独确定 | ✅ ρ_c 单独已知 | ❌ ρ_s 与 κ 耦合 |
| 需要额外输入 | 无 | m_c（或 κ） |

## 八、诚实声明

- ✅ 从 m_e 反推 ρ_s 是直接代数运算，推导无误
- ✅ 量纲分析自洽，所有指数经显式验证
- ✅ 核心发现 S = 3.819×10² J/m 的"不变性"是此前未被注意的关键结果
- ❓ ρ_s 未被单独确定——这是理论的固有简并，非推导缺陷
- ❓ ξ < a 约束导出 m_c > 0.5g——如果 a_s ≠ ℓ_Pl 则约束会移动（若 a_s 更大，ξ 更小，约束放松）
- ❓ SCVC 不需要继承弹性理论的公式——S = ρ_s κ² 是它自己的刚度参数
