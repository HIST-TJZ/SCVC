# D线：N=3 T²权结构推导

**纯数学推导。基于 N=1 已知数据与标度律假设。**

---

## 1. 标度律 dim_ℂ M_vortex(N) = 3N 的数学基础

### 1.1 BPS涡旋模空间的标准结果

在Abelian Higgs模型中，N个BPS涡旋在Riemann面Σ上的模空间为：

\[
\mathcal{M}_N \cong \mathrm{Sym}^N(\Sigma) = \Sigma^N / S_N
\]

每个涡旋贡献1个复维度（其在Σ上的位置），因此 dim_ℂ = N。这是经典结果（Taubes, Jaffe-Taubes）。

在本问题中，单个涡旋模空间 M_vortex(1) 已是6实维（3复维），包含：
- 涡旋环中心位置（2复维：xy平面位置）
- 涡旋环朝向（1复维：S²上的方向）
- 涡旋环半径（隐含在自旋纹理中）

### 1.2 标度律成立的条件

dim_ℂ M_vortex(N) = k·N 成立当且仅当：

1. **涡旋间无拓扑相互作用**：在BPS极限（临界耦合λ=1）下，涡旋间力恰好抵消，N个涡旋的位置可独立选择。
2. **每个涡旋携带完整的内部自由度**：每个涡旋环独立贡献其中心、朝向、半径。
3. **无全局约束**：不存在消除整个模自由度的全局条件（如总动量为零等）。

在此条件下，N涡旋模空间自然模型为：

\[
\boxed{M_{\mathrm{vortex}}(N) \cong \mathrm{Sym}^N\left(M_{\mathrm{vortex}}(1)\right)}
\]

每个涡旋贡献 dim_ℂ M_vortex(1) = 3 个复维度，因此 dim_ℂ M_vortex(N) = 3N。

> **注**：Sym^N(X) 在涡旋重合处有商奇点（orbifold singularities）。在 dim_ℂ X = 3 时无可规范光滑分解（Hilbert概型仅适用于dim ≤ 2）。但T²等变上同调层面不受影响——奇点集是T²不变的闭子簇，余维数≥2，等变Euler类的局部化公式仍然成立。

### 1.3 N→∞ 时不动点数量的变化

设 M_vortex(1) 有 m = 3 个孤立不动点：F₀, F₁, F₂。

M_vortex(N) = Sym^N(M_vortex(1)) 的T²不动点 = 来自 {F₀, F₁, F₂} 的大小为N的多重集。

不动点总数：

\[
\boxed{|M_{\mathrm{vortex}}(N)^{T^2}| = \binom{N + m - 1}{m - 1} = \binom{N+2}{2} = \frac{(N+2)(N+1)}{2}}
\]

- N=1: 3 ✓（与已知数据一致）
- N=3: 10
- N→∞: ~N²/2，**不再是3个**

## 2. N=3 的 T² 权结构

### 2.1 符号约定

T² = SO(2)_z × U(1)，Cartan生成元 (u, v) ∈ H²_T(pt)。

N=1 已知数据（来自参考文件）：

| 不动点 | 几何 | 权 (m,n) | e_T |
|:---|:---|:---|:---|
| **F₀** | R=0 (真空) | (+1,0), (-1,0), (0,+1) | −u²v |
| **F₁** | R=R_eq (自旋纹理) | (+1,+1), (-1,+1), (0,−2) | 2v(v²−u²) |
| **F₂** | R=R_max (边界) | (+1,0), (-1,0), (0,+1) | −u²v |

**关键事实**：F₀ 和 F₂ 具有相同的等变Euler类（即相同的局部T²表示），但具有不同的moment map值——它们是不同的不动点。

### 2.2 Sym^N 模型的权结构

对于配置 (k₀, k₁, k₂)（k₀ 个涡旋在 F₀，k₁ 个在 F₁，k₂ 个在 F₂，k₀+k₁+k₂=N），切空间的T²表示为各组分切空间的直和：

\[
T_{[(k_0,k_1,k_2)]} M_{\mathrm{vortex}}(N) = \underbrace{T_{F_0}M_1 \oplus \cdots \oplus T_{F_0}M_1}_{k_0} \oplus \underbrace{T_{F_1}M_1 \oplus \cdots \oplus T_{F_1}M_1}_{k_1} \oplus \underbrace{T_{F_2}M_1 \oplus \cdots \oplus T_{F_2}M_1}_{k_2}
\]

每个T_{F_i}M_1 贡献其3个权。S_k 商作用仅置换相同权方向的副本——作为T²表示，同构于各权的直接倍乘。

等变Euler类：

\[
\boxed{e_T(k_0, k_1, k_2) = (-u^2 v)^{k_0 + k_2} \cdot \left[2v(v^2 - u^2)\right]^{k_1}}
\]

### 2.3 N=3 完整分类

N=3 时，(k₀, k₁, k₂) ∈ ℕ³，k₀+k₁+k₂=3。

#### 类型 A：全部在 F₀/F₂ (k₁ = 0)

n₀₂ = k₀+k₂ = 3，k₁ = 0

\[
e_T^{(A)} = (-u^2 v)^3 = -u^6 v^3
\]

| (k₀,k₁,k₂) | 说明 | e_T |
|:---|:---|:---|
| (3,0,0) | 3涡旋全在真空不动点 | −u⁶v³ |
| (2,0,1) | 2在真空，1在边界 | −u⁶v³ |
| (1,0,2) | 1在真空，2在边界 | −u⁶v³ |
| (0,0,3) | 3涡旋全在边界不动点 | −u⁶v³ |

**4个不动点**，具有相同的等变Euler类（但有4个不同的moment map值）。

#### 类型 B：一个在 F₁，两个在 F₀/F₂ (k₁ = 1)

n₀₂ = 2，k₁ = 1

\[
e_T^{(B)} = (-u^2 v)^2 \cdot 2v(v^2 - u^2) = 2u^4 v^3 (v^2 - u^2) = \boxed{2u^4 v^5 - 2u^6 v^3}
\]

| (k₀,k₁,k₂) | e_T |
|:---|:---|
| (2,1,0) | 2u⁴v⁵ − 2u⁶v³ |
| (1,1,1) | 2u⁴v⁵ − 2u⁶v³ |
| (0,1,2) | 2u⁴v⁵ − 2u⁶v³ |

**3个不动点**，相同的 e_T。

#### 类型 C：两个在 F₁，一个在 F₀/F₂ (k₁ = 2)

n₀₂ = 1，k₁ = 2

\[
e_T^{(C)} = (-u^2 v) \cdot [2v(v^2 - u^2)]^2 = -4u^2 v^3 (v^4 - 2u^2 v^2 + u^4) = \boxed{-4u^2 v^7 + 8u^4 v^5 - 4u^6 v^3}
\]

| (k₀,k₁,k₂) | e_T |
|:---|:---|
| (1,2,0) | −4u²v⁷ + 8u⁴v⁵ − 4u⁶v³ |
| (0,2,1) | −4u²v⁷ + 8u⁴v⁵ − 4u⁶v³ |

**2个不动点**，相同的 e_T。

#### 类型 D：全部在 F₁ (k₁ = 3)

n₀₂ = 0，k₁ = 3

\[
e_T^{(D)} = [2v(v^2 - u^2)]^3 = 8v^3(v^6 - 3u^2 v^4 + 3u^4 v^2 - u^6) = \boxed{8v^9 - 24u^2 v^7 + 24u^4 v^5 - 8u^6 v^3}
\]

| (k₀,k₁,k₂) | e_T |
|:---|:---|
| (0,3,0) | 8v⁹ − 24u²v⁷ + 24u⁴v⁵ − 8u⁶v³ |

**1个不动点**。

### 2.4 汇总

| 类型 | k₁ | 不动点数 | e_T (deg=9齐次多项式) |
|:---|:---|:---|:---|
| A | 0 | 4 | −u⁶v³ |
| B | 1 | 3 | 2u⁴v⁵ − 2u⁶v³ |
| C | 2 | 2 | −4u²v⁷ + 8u⁴v⁵ − 4u⁶v³ |
| D | 3 | 1 | 8v⁹ − 24u²v⁷ + 24u⁴v⁵ − 8u⁶v³ |
| **总计** | | **10** | |

## 3. 权结构分析：简单直和 vs. 新混合权

### 3.1 在 Sym^N 模型下：无新混合权

N=3 的每个不动点处，切空间权**完全由 N=1 基本权的多重集构成**：

- 权 (+1,0)：来自 F₀ 或 F₂，多重度 = k₀+k₂
- 权 (−1,0)：来自 F₀ 或 F₂，多重度 = k₀+k₂
- 权 (0,+1)：来自 F₀ 或 F₂，多重度 = k₀+k₂
- 权 (+1,+1)：来自 F₁，多重度 = k₁
- 权 (−1,+1)：来自 F₁，多重度 = k₁
- 权 (0,−2)：来自 F₁，多重度 = k₁

**结论**：在 Sym^N 模型下，N=3 的权结构是 N=1 权结构的**简单多重集并**，不出现新的混合权（即不出现 N=1 中不存在的 (m,n) 组合）。

### 3.2 偏离 Sym^N 的可能性

若 M_vortex(N) **不是**严格的 Sym^N，涡旋间存在不可忽略的相互作用，则可能出现：

1. **涡旋重合处的谱分裂**：当 k 个涡旋在 F₁ 重合时，自旋纹理的耦合可能导致权重发生非平凡的重新组合。例如，k₁=3 时，(0,−2) 权重可能出现分裂为 (0,−1)+(0,−1) 或其他组合。

2. **全局约束导致的权重消除**：若存在"总角动量为零"等全局约束，某些权方向可能被消除，dim_ℂ < 3N。

**当前缺口**：未提供涡旋间相互作用的微观理论（Gross-Pitaevskii方程的N涡旋解），无法严格排除上述可能性。以下均基于 Sym^N 模型进行。

## 4. 关于因子 3^{3/4} ≈ 1.682

### 4.1 直接计算检查

在 Sym^N 模型中，N=3 时的自然量：

- 不动点数：10（vs N=1: 3）
- 不动点数之比：10/3 ≈ 3.333
- 不同 e_T 类型数：4（vs N=1: 2）
- 类型数之比：2

| 量 | N=1 | N=3 | 比值 |
|:---|:---|:---|:---|
| 不动点数 | 3 | 10 | 10/3 ≈ 3.33 |
| e_T 类型数 | 2 | 4 | 2 |
| dim_ℂ | 3 | 9 | 3 |
| 最大 |e_T| 系数 | 2 | 8 | 4 |

3^{3/4} ≈ 1.682 不直接匹配上述任何比值。

### 4.2 可能来源的推测

1. **F₁ 权向量的几何均值**：F₁ 的三个权向量在 ℤ² 中的范数为 (√2, √2, 2)，乘积为 4。对于 F₁ 占主导的构型 (0,3,0)：(4)³ 的某种归一化 N^{-3/4} = 3^{-3/4} 可能出现 3^{3/4} 的倒数相关因子。

2. **矩多面体的缩放**：N=3 的矩多面体是 N=1 矩多面体（三角形）的3倍缩放。面积比为 9。若考虑某种特征长度的 (dim)^(1/4) 缩放，dim=9 → 9^{1/4} = 3^{1/2} ≈ 1.732，接近但不等。

3. **vol(M(N)) 的特征尺度**：
   \[
   \mathrm{vol}(M(N)) \propto \frac{V^N}{N!} \quad\Rightarrow\quad \text{特征长度} \propto \left(\frac{V^N}{N!}\right)^{1/(3N)} \approx \frac{V^{1/3} e^{1/3}}{N^{1/3}}
   \]
   N=3 vs N=1 的特征长度比 ≈ 3^{-1/3} ≈ 0.693，非 1.682。

**结论**：在当前对称积模型下，**3^{3/4} 不自然出现**。需额外假设（如涡旋间相互作用导致的非平凡谱结构）方可导出。标记为**开放问题**。

## 5. 欧拉数 e(N_i, N) 随 N 的变化

### 5.1 普通欧拉数

在 Sym^N 模型中，每个不动点 (k₀,k₁,k₂) 处的普通（非等变）欧拉数为：

\[
e(N_{(k_0,k_1,k_2)}) = [e(N_{F_0})]^{k_0} \cdot [e(N_{F_1})]^{k_1} \cdot [e(N_{F_2})]^{k_2}
\]

来自参考文件的 N=1 普通欧拉数（取绝对值的符号约定后）：

\[
e(N_{F_0}) = +\frac{\pi^3}{4}, \quad e(N_{F_1}) = -\frac{\pi^3}{2}, \quad e(N_{F_2}) = +\pi^3
\]

因此对于 N=3：

| 类型 | (k₀,k₁,k₂) | e(N) / π⁹ |
|:---|:---|:---|
| (3,0,0) | (3,0,0) | (1/4)³ = 1/64 |
| (2,0,1) | (2,0,1) | (1/4)² · 1 = 1/16 |
| (1,0,2) | (1,0,2) | (1/4) · 1² = 1/4 |
| (0,0,3) | (0,0,3) | 1³ = 1 |
| (2,1,0) | (2,1,0) | (1/4)² · (−1/2) = −1/32 |
| (1,1,1) | (1,1,1) | (1/4) · (−1/2) · 1 = −1/8 |
| (0,1,2) | (0,1,2) | (−1/2) · 1² = −1/2 |
| (1,2,0) | (1,2,0) | (1/4) · (−1/2)² = 1/16 |
| (0,2,1) | (0,2,1) | (−1/2)² · 1 = 1/4 |
| (0,3,0) | (0,3,0) | (−1/2)³ = −1/8 |

### 5.2 欧拉示性数

T²等变局部化给出：

\[
\chi(M_{\mathrm{vortex}}(N)) = \sum_{p \in M^T} \mathrm{sign}(e(N_p))
\]

在 Sym^N 模型下，正负号依赖于 k₁ 的奇偶性（因 e(N_{F₁}) < 0）：

- k₁ 偶 → sign = +（因 e(N_{F₀})>0, e(N_{F₂})>0, [e(N_{F₁})]^{even}>0）
- k₁ 奇 → sign = −

计数：

N=3时，k₁ ∈ {0,1,2,3}：
- k₁=0: C(3+0+2-1, 2-1) = C(4,1) = 4 个不动点，sign = +1
- k₁=1: C(2+1,1) = 3 个，sign = −1
- k₁=2: C(1+1,1) = 2 个，sign = +1
- k₁=3: C(0+1,1) = 1 个，sign = −1

\[
\chi(M_{\mathrm{vortex}}(3)) = 4 - 3 + 2 - 1 = 2
\]

**通项公式**（Sym^N 模型）：

\[
\boxed{\chi(M_{\mathrm{vortex}}(N)) = \sum_{k_1=0}^{N} (-1)^{k_1} \binom{N - k_1 + 1}{1} = \sum_{k_1=0}^{N} (-1)^{k_1} (N - k_1 + 1)}
\]

计算部分和：
- N=1: 2 − 1 = 1
- N=2: 3 − 2 + 1 = 2
- N=3: 4 − 3 + 2 − 1 = 2
- N=4: 5 − 4 + 3 − 2 + 1 = 3

欧拉示性数交替增长，约 ~N/2（非随N线性增长，而是以1/2的斜率振荡增长）。

## 6. 对费米子质量标度的影响

### 6.1 涡旋零模与费米子耦合

在涡旋背景中，费米子零模通过指数定理与涡旋模空间相关联。dim_ℂ M_vortex(N) = 3N 意味着存在 3N 个复费米子零模（每个复维度对应一对实零模）。

### 6.2 质量算符的矩阵元

费米子质量矩阵的矩阵元 ∼ 零模波函数的重叠积分。在 Sym^N 图像中，零模局域在涡旋位置。对于N个分离良好的涡旋，质量矩阵近似块对角，每个涡旋贡献独立的质量 ∼ m_F（N=1时的费米子质量）。

因此：

\[
m_{\mathrm{fermion}}(N) \sim m_{\mathrm{fermion}}(1) \quad (\text{不随N显著变化})
\]

### 6.3 涡旋重合时的修正

当涡旋重合时（如 (0,3,0) 构型），零模波函数重叠导致质量矩阵的混合。混合角 ∼ O(1) 可产生谱的展宽，但特征标度不改变一个数量级。

**结论**（Sym^N 模型下）：费米子质量标度对 N 不敏感。若存在超越 Sym^N 的相互作用，则可能出现 ∼ N^α 的标度依赖性，但当前缺乏微观输入。

### 开放问题 / 缺口

1. **涡旋间相互作用**：未提供N涡旋Gross-Pitaevskii方程的解。若存在不可忽略的相互作用，权结构可能偏离简单直和。
2. **F₀与F₂的区别**：它们具有相同的 e_T 但不同的 moment map 值。在什么条件下它们可被视为"等变等价"？需要明确等价关系的定义。
3. **3^{3/4}的来源**：需要额外的微观假设方可导出。
4. **普通欧拉数的精确值**：来自参考文件的 N=1 欧拉数是"从目标反推"的——严格的一阶原理计算需要完整的谱分析。
