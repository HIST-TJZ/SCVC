# 计算任务：涡旋模空间 = toric Kähler 3-fold × 3 孤立不动点的严格证明

## 背景

SCVC 的核心几何对象是"BPS 涡旋环模空间 M_vortex"。框架声称：
1. M_vortex 是 toric Kähler 3-fold（复三维紧致环簇）
2. 在 T² 作用下恰好有 3 个孤立不动点：F1(R=0), C2(R=R_eq), F3(R=R_max)
3. 三个不动点的等变 Euler 类分别为：e_T(F1) = 4π³, e_T(C2) = π², e_T(F3) = π

批评者指出：这个模空间从未被显式构造。上述性质是**假定**的，不是证明的。

**本任务目标：从 BPS 涡旋方程出发，尽可能严格地证明 M_vortex 具有上述三个性质。**

## 第一步：BPS 涡旋方程回顾

SCVC 中，BPS 涡旋环满足：

$$F_{ab} = \frac{1}{2}\epsilon_{abc} D_c (|\Psi|^2 - 1)$$

或等价的一阶方程（在 3D 空间 R³ 中，柱坐标 (r, θ, z)）：

$$\partial_r f = \frac{1}{r}(1 - f^2)$$

其中 f(r) 是涡旋剖面函数，边界条件 f(0)=0, f(∞)=1。

对于涡旋环（而非涡旋线），还有环本身的半径 R 作为模参数。

## 第二步：M_vortex 是什么？

涡旋环的模空间 M_vortex = { 所有 BPS 涡旋环构型 } / 规范等价。

一个涡旋环构型由以下参数确定：
- **环半径 R**（连续参数，R ∈ [0, R_max]）
- **环平面方向** n̂ ∈ S²（两个角度 θ, φ）
- **环上的自旋织构**：CP¹ 上的 U(1) 缠绕（一个 S¹ 相角）
- **整体 U(1) 相位**（规范零模）

总自由度计数：
- R：1 个实参数
- n̂：2 个实参数（S²）
- 自旋织构相角：1 个周期参数（S¹）
- 整体相位：1 个周期参数（S¹）

→ M_vortex 的（实）维数 = 1 + 2 + 1 + 1 = 5，或复维数 ≈ 2.5？

**等等。** 环上的 S¹ 相位和整体 U(1) 相位可能不是独立的。需要仔细分析。

## 第三步：T² 作用与不动点

环面作用 T²（两个 U(1) 因子，来自涡旋的平移和内部旋转）作用于 M_vortex。

在 T² 作用下，哪些构型是**不动的**（即在 T² 作用下只改变相位，几何不变）？

**F1 (R→0)**：环半径趋于零 → 涡旋环退化为点状涡旋。此时：
- n̂ 方向失去意义（点没有方向）
- 自旋织构塌缩
- T² 作用退化为整体相位旋转
- **这是一个孤立不动点**

**F2 (R=R_eq, 特定方向)**：环在某平衡半径 R_eq，环面方向与涡旋内部相位旋转对齐。
- 此时 T² 的一个 U(1) 子群保持构型不变（对齐条件）
- 另一个 U(1) 产生环的刚体旋转，但环是轴对称的
- **这是不动点还是一个不动子流形？** 需要判定。

**F3 (R=R_max, 边界)**：环达到最大半径（由介质截断决定）。
- 环面方向仍可变化？还是被边界条件锁定？
- **这是孤立不动点还是边界上的不动集？**

**关键问题**：C2 是孤立不动点还是不动子流形？如果是子流形，DH 求和需要用 Atiyah-Bott 公式而非标准 DH 公式。目前框架对此的处理存在模糊性。

## 第四步：等变 Euler 类的计算

对每个不动点 F（假设是孤立的），切空间 T_F(M_vortex) 是 T² 的表示，分解为权空间：

$$T_F(M_{\text{vortex}}) = \bigoplus_{i=1}^{3} \mathbb{C}_{\omega_i}$$

其中 ω_i 是 T² 在切空间上的权（线性函数 on Lie(T²)）。

等变 Euler 类：

$$e_T(F) = \prod_{i=1}^{3} \omega_i$$

在物理归一化（涡旋张力 κ 和环半径 R 为基本标度）下，权 ω_i 取值于 {π 的多项式}。

**要求**：从 M_vortex 的几何结构出发，显式计算 F1、C2、F3 处的三个权 ω_i，然后计算 e_T = ∏ ω_i。

**候选计算路线**：
- **F1 (R=0)**：这是涡旋环空间的"原点"。切空间来自环半径 R（一个方向）、环面方向 n̂（S² 的两个方向）。三个权 ω_i 如何分配？
- **F3 (R=R_max)**：在边界处，R 的自由度被冻结（R 不再变化），切空间只有环面方向 S²。但模空间仍有三个复维？还是退化为两个？
- **C2 (R=R_eq)**：中间的不动点。权结构最复杂。

## 第五步：toric 性质的证明

M_vortex 是 toric 的，意味着：
1. 存在 T³ 的有效哈密顿作用（而不仅仅是 T²）
2. 存在 moment 映射 μ: M_vortex → ℝ³，其像 Δ 是凸多面体
3. 不动点恰好对应 Δ 的顶点

**需要证明**：第三个 U(1) 从哪里来？三个独立守恒荷是什么？

候选：涡旋环的三个守恒量——
1. 环的刚体旋转角动量（对应 S¹ 平移）
2. 涡旋内部 U(1) 相位旋转（对应自旋织构）
3. 环的"呼吸"模式？（对应 R 的变化）

这三个是否构成 T³ 作用？

## 第六步：多面体几何

如果 M_vortex 是 toric 3-fold，其 moment 多面体 Δ 是 ℝ³ 中的凸多面体。

三个不动点 F1, C2, F3 对应 Δ 的三个顶点。通过顶点处的边方向（权 ω_i），可以唯一确定 Δ 的 combinatorics（扇结构）。

**任务**：从权数据 {F1: (ω₁, ω₂, ω₃), C2: (ω'₁, ω'₂, ω'₃), F3: (ω''₁, ω''₂, ω''₃)}，重构 Δ 的扇。证明这是一个截锥（truncated cone），且截锥的 combinatorics 强制 e_T(F1) = 4π³, e_T(C2) = π², e_T(F3) = π。

## 诚实要求

1. 如果某一步不能严格证明（如"T² 扩展为 T³"），诚实标注为假设。
2. 如果 C2 实际上是 1 维不动子流形而非孤立不动点，诚实说，并改用 Atiyah-Bott 公式重新计算。
3. 如果 3 个不动点的 Euler 类值（4π³, π², π）中的任何一个不能从模空间几何正向算出，诚实标注。
4. 自评置信度：对上述六步中的每一步给 %。
