# 计算任务：T² GKM 框架下的涡旋模空间严格证明（v2，不依赖 T³）

## 为什么 v2

v1 证明的综合置信度 ~88%，最弱环节是步骤 6：T³ toric 扩展（80%）。

**本 v2 的核心策略：GKM 定理只需要 T²。DH 求和不需要 T³。** 

T³（"toric 3-fold"标签）是优雅的几何描述，但不是必须的前提。本任务明确区分：
- **核心层**（T² GKM）：不动点分类 + 切空间权分解 + Euler 类 → DH 求和。目标置信度 ≥92%。
- **标签层**（T³ toric）：几何优雅描述。非必须，置信度 80% 可接受。

## 起点

SCVC 中，费米子 = BPS 涡旋环。M_vortex = BPS 涡旋环构型模空间。

已知 T² = SO(2)_z × U(1)_gauge 是 M_vortex 上的 Hamiltonian 作用（v1 步骤 2，95%）。

## 任务：在 T² 框架下完成全部核心论证

### 第一层：不动点分类（目标 ≥95%）

T² 作用下的不动点条件：
- SO(2)_z 不变性 → 构型绕 z 轴旋转对称
- U(1)_gauge 不变性 → 相位旋转可被规范变换吸收

**要求**：在 T² 作用的框架下，严格分类所有不动点。证明恰好 3 个孤立不动点集：

| 不动点 | 几何 | T² 退化情况 |
|---|---|---|
| F1 (R=0) | 涡旋环塌缩为点，在 z 轴上 | SO(2)_z 退化（点在轴上）+ U(1)_gauge 退化 → d=3 退化 |
| C2 (R=R_eq) | 涡旋环在平衡半径 | 1 个 T² 子群退化（对齐条件）→ d=2 退化 |
| F3 (R=R_max) | 涡旋环在最大半径（截断） | 1 个 T² 子群退化（边界冻结）→ d=1 退化 |

**关键**：证明没有第 4 个不动点。排除混合构型（如环不平行于 xy 平面但具有 SO(2)_z 对称性）。

### 第二层：切空间权分解（目标 ≥92%）

在每个不动点 F，T² 线性作用于切空间 T_F(M_vortex)。根据等变 Darboux 定理，存在复坐标使得：

$$T_F(M_{\text{vortex}}) \cong \bigoplus_{i=1}^{d_F} \mathbb{C}_{\omega_i}$$

其中 d_F = dim_ℂ(M_vortex) = 3（对所有 F），权 ω_i 是 Lie(T²) 上的线性函数（整数线性组合）。

**任务**：从涡旋环构型的物理自由度出发，显式推导三个不动点处的权：

- **F1 (R=0)**：环塌缩为点。三个自由度：(1) 环半径 R 的呼吸模式，(2,3) 环面方向 n̂ ∈ S²。在 T² 作用下，这些自由度的变换性质 → 权 ω₁^(F1), ω₂^(F1), ω₃^(F1)。

- **C2 (R=R_eq)**：环在平衡半径，环面与涡旋内部相位对齐。三个自由度：(1) 环面方向的微调，(2) 环上自旋相位，(3) 整体平移。T² 作用下 → 权 ω₁^(C2), ω₂^(C2), ω₃^(C2)。

- **F3 (R=R_max)**：环在最大半径，边界截断。R 的自由度被冻结。剩余自由度：(1,2) 环面方向，(3) 自旋相位或边界模式。→ 权 ω₁^(F3), ω₂^(F3), ω₃^(F3)。

**要求**：显式写出每个不动点的三个权，作为 SO(2)_z 和 U(1)_gauge 生成元的线性组合。用 (u, v) 表示 Lie(T²) 的坐标，其中 u 对应 SO(2)_z，v 对应 U(1)_gauge。

期望结果（来自 v1 和已有计算）：
- F1: 权 = {v, v+u, v-u}（或等价形式）
- C2: 权 = {u, v+u, v-u}（或其他排列，其中一个权退化）
- F3: 权 = {v, u, something}（两个权非退化）

如果实际计算结果不同，诚实地报告实际结果。

### 第三层：等变 Euler 类计算（目标 ≥90%）

$$e_T(F) = \prod_{i=1}^{3} \omega_i(F)$$

将第二层的权代入，得到 e_T(F) 作为 u, v 的多项式。

然后代入**物理点** u = u₀, v = v₀（涡旋环的自洽参数：u₀ = π, v₀ = √3π 或等价归一化），计算数值：

$$e_T(F1) = 4\pi^3, \quad e_T(C2) = \pi^2, \quad e_T(F3) = \pi$$

**要求**：不假设结果。从权的显式表达式出发，代入 u₀, v₀，显式计算 e_T 的值。如果得到的结果不是 4π³, π², π，诚实地报告实际结果。

**物理归一化**：u₀ 和 v₀ 的值从涡旋张力 κ 和环半径 R_eq, R_max 确定。如果这一步无法正向推导（需要解完整 BPS 方程），诚实标注为"强度 2：自洽确定"。

### 第四层：GKM 定理 → 等变上同调环（目标 ≥95%）

**GKM 定理**：对于具有 T² Hamiltonian 作用的紧辛流形 M，若所有不动点集是孤立点或 CP¹，则：

1. 等变上同调环 H*_T(M) 完全由 **1-骨架**（不动点 + 连接它们的 T²-不变 CP¹）决定。
2. 不需要显式知道 M 上的度规。1-骨架的 combinatorics（哪些不动点被哪些 CP¹ 连接，以及在连接处的权）唯一确定 H*_T(M)。

**任务**：检查 M_vortex 是否满足 GKM 定理的条件。
- 紧性：v1 标注了假设 H1（可紧化）。评估此假设的合理性。
- 不动点集类型：F1 是孤立点，C2 可能是 CP¹ 丛，F3 可能是曲面。GKM 要求孤立不动点。如果 C2 和 F3 不是孤立点，GKM 如何调整？

**如果 GKM 条件不完全满足**：
- Atiyah-Bott 公式仍适用（对非孤立不动点集，用等变法丛的 Euler 类 + 不动点集上的积分）
- v1 中 C2 标记为"CP¹ 不动子流形"——这需要用 Atiyah-Bott（非 DH）公式。但在 SCVC 的计算中，C2 被当作孤立不动点处理，贡献 π²。这是否一致？

**这是 v2 需要解决的核心歧义**。

### 第五层：DH 求和 → α⁻¹（目标 ≥97%）

假设前三层成立，DH 求和给出：

$$\alpha^{-1} = \sum_{F} \frac{1}{e_T(F)} = \frac{1}{4\pi^3} + \frac{1}{\pi^2} + \frac{1}{\pi}$$

乘以 C_total = 1（已在 01 号证明中闭合），得到：

$$\alpha^{-1} = 4\pi^3 + \pi^2 + \pi = 137.036304$$
