计算任务：从涡旋环拓扑推导元素周期表结构

背景
SCVC已推导：α=1/137, m_e=0.511 MeV, m_p≈938 MeV。
涡旋环的环量量子化来自BEC拓扑：∮v·dl = nh/m_e。
泡利不相容在SCVC中不是额外公设——两个涡旋环是拓扑缺陷，
它们不能占据同一个量子态（和超流涡旋不能重叠同理）。

本任务目标：从上述已知量出发，定量推导元素周期表的壳层结构、
惰性气体位置、碱金属位置、原子半径和电离能的趋势。

第一部分：壳层填充顺序

步骤1.1：从涡旋环量子化到主量子数
环量量子化 → 驻波条件 2πr = nλ_dB → 角动量 L = nħ
→ 主量子数 n = 1, 2, 3, ...

步骤1.2：角量子数和磁量子数
涡旋环不仅可以绕质子做圆周运动（l=0），还可以倾斜环面（l>0）。
在3D空间中，环面的法向量 n̂ 可以取 (2l+1) 个方向 → m = -l, ..., +l。
l 的最大值受什么约束？是 n-1 吗？从涡旋环几何出发推导，不假设量子力学结果。

步骤1.3：自旋
涡旋环的绕行方向有两种（顺时针/逆时针）→ 每个轨道态可以容纳2个电子。
这在BEC涡旋中称为极化（polarization）→ 对应自旋向上/向下。

步骤1.4：填充顺序
n=1: l=0 → 1个轨道 → 2个电子 → He(2)
n=2: l=0,1 → 1+3=4个轨道 → 8个电子 → Ne(2+8=10)
n=3: l=0,1,2 → 1+3+5=9个轨道 → 18个电子 → Ar(2+8+8=18)
n=4: ... → Kr(2+8+18+8=36)

问题：4s为什么在3d之前填？SCVC能否解释这一偏离？

第二部分：原子半径

步骤2.1：氢原子半径
a₀ = ħ/(α m_e c) = 0.529 Å（已从SCVC的α和m_e+ħ输入确定）

步骤2.2：多电子原子的半径
外层电子的有效轨道半径 r_n ≈ n² a₀ / Z_eff
其中Z_eff是有效核电荷（核电荷Z减去内层电子的屏蔽）。
屏蔽由内层涡旋环的拓扑排斥决定。
给出惰性气体原子半径的理论值与实验值的对比表：
He(0.31 Å), Ne(0.38 Å), Ar(0.71 Å), Kr(0.88 Å)

第三部分：电离能

步骤3.1：氢原子电离能
E₁ = ½ α² m_e c² = 13.6 eV（直接从α和m_e计算）

步骤3.2：碱金属电离能
最外层单个电子，有效核电荷 ≈ +1（内层完全屏蔽）。
E_ion ≈ 13.6 eV / n² → Li(5.4 eV理论, 5.39 eV实验), Na(5.1 eV, 5.14 eV)
给出碱金属电离能的趋势线。

步骤3.3：惰性气体电离能
满壳层，最外层电子感受到较大的Z_eff。
给出He, Ne, Ar, Kr的实验电离能，SCVC是否能解释其数量级和趋势？

第四部分：成功标准

不需要每个元素的电离能精确到1%。判据是：
✅ 惰性气体出现在 Z = 2, 10, 18, 36（位置对）
✅ 碱金属在惰性气体+1处：Z = 3, 11, 19, 37（位置对）
✅ 原子半径趋势：同族递增，同周期递减（趋势对）
✅ 电离能趋势：惰性气体>卤素>氧族>...>碱金属（趋势对）
✅ 壳层容量 2, 8, 18, 32 从涡旋拓扑自然推出

第五部分：诚实标注

标注哪些是SCVC正向推导的，哪些借用了量子力学的已知结果：
- 壳层量子数 n,l,m 的起源：SCVC（涡旋环几何）
- 壳层容量 2(2l+1)：SCVC（环面方向自由度+极化）
- 能级顺序 1s→2s→2p→3s→3p→4s→3d：？SCVC能否解释4s先于3d？
- 有效核电荷 Z_eff 的定量值：需要涡旋环间排斥力的显式计算

如某一步SCVC给不出独立推导，诚实说"目前借用了量子力学的结果"。
