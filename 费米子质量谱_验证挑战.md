# 费米子质量谱的 π 多项式 — 独立验证挑战

> **只需要计算器。不需要判断物理。10 分钟就能自己验证。**

## 理论声称

标准模型全部费米子的质量比 + 中微子振荡 + 质量总和，用 5 个几何/群论基元表达：

```
π, π², π³, π⁴, DH(=4π³+π²+π), |S₃|=6, N_c=3, 5/3
```

**"0 个自由参数" 的意思是：每一个整数系数，改了就不是唯一解——偏差断崖式崩塌。而且即使随机删掉一半公式，剩下的公式仍然指向同一个整数集。**

**中微子为什么不是简单的 π 多项式？** 因为它们是 Majorana 费米子，质量来自跷跷板：

```
g₃² = 4π/(16π) = 1/4            [精确有理数！]
S_inst = π²/(6g₃²) = 2π²/3      [π 多项式——分母 3 = N_c！]
M_R = M_KK × e^(-2π²/3)         [指数桥，~M_KK/720]
m_D ∝ 1/4π³ : 1/π² : 1/π        [DH 不动点权重]
m_ν = m_D²/M_R                   [跷跷板]
```

> **Dirac 费米子（1-10）：π^n ——纯多项式**
> **Majorana 费米子（11-12）：π^(-n) × e^(2π²/3) ——多项式 × e^多项式**
> 
> 框架在 π 结构级别自动区分两种费米子。这不是 bug——这是特征。

## 验证任务二：整数唯一性（"没有老二"）

**对以下每个公式，检查相邻整数能否替代：**

```
m_c/m_u = 6π⁴     → 算 5π⁴, 6π⁴, 7π⁴。5 和 7 能否替代 6？
m_τ/m_e = 36π⁴    → 算 35π⁴, 36π⁴, 37π⁴。35 和 37 能否替代 36？
m_s/m_d = 2π²     → 算 1π², 2π², 3π²。1 和 3 能否替代 2？
m_b/m_d = 9π⁴     → 算 8π⁴, 9π⁴, 10π⁴。8 和 10 能否替代 9？
m_u/m_e = 3√2     → 算 2√2, 3√2, 4√2。2 和 4 能否替代 3？
m_H/m_W = π/2     → π/3 或 2π/3 能否替代 π/2？
m_d/m_u = (5/3)^(3/2) → (4/3)^(3/2) 或 (6/3)^(3/2) 能否替代？
```

**"唯一解" 的定义：最优整数是解，相邻整数偏差断崖式恶化。**

## 验证任务三：跨公式交叉锁定

**|S₃|=6 出现在 4 个公式里：**
- #4 m_τ/m_e = 36π⁴ (= 6²π⁴)
- #6 m_c/m_u = 6π⁴
- #7 m_t/m_u = 6π⁴×DH
- #11-12 中微子 m_D 权重来自同一 DH 不动点

**把 6→5，#4 #6 #7 偏差皆崩。中微子 m_D 比值也变。**

**N_c=3 出现在 4 个公式里：**
- #5 m_u/m_e = 3√2
- #10 m_b/m_d = 9π⁴ (= 3²π⁴)
- N_gen = 3（实验值，不能是 4）
- #11-12 中微子 S_inst 分母 = **3**

**把 3→4，#5 #10 偏差崩，N_gen 直接矛盾，中微子 S_inst 从 2π²/3→2π²/4，M_R 变 e^(π²/2)/e^(2π²/3) ≈ 140×。**

**比例 5/3 出现在 2 个公式里：**
- #3 m_μ/m_e = 4π³×(5/3)
- #8 m_d/m_u = (5/3)^(3/2)

## 🆕 验证任务四：子集唯一性（"删掉一半还成立吗？"）

**如果公式是拟合的，随机删掉一些，剩下的应该无法确定原参数。**

```
情景A：只用带电轻子（#1, #3, #4）
  算 #3 确定 5/3，算 #4 确定 6（36=6²），算 #1 确定 DH
  → 3 个整数从 3 个公式确定。是拟合？不可能——拟合需要 N_data > N_params。

情景B：只用上型夸克（#5, #6, #7）
  算 #5 确定 3，算 #6 确定 6，算 #7 验证 6+DH
  → 同上

情景C：只用下型夸克（#8, #9, #10）
  算 #8 确定 5/3，算 #9 确定 2，算 #10 确定 3（9=3²）
  → 同上

情景D：随机选 6 个公式（丢一半）
  每个整数出现在 2-4 个公式里。丢一半仍有幸存公式。
```

**结论：5 个整数参数，5 个公式就能全覆盖。其余 7 个是冗余交叉验证。像 QR 码——撕掉一角还能扫。拟合没有这种性质。**

## 验证任务五：DH 连锁

DH = 4π³+π²+π 不只出现在 #1 α⁻¹。其整数三元组 (4,1,1) 锁死 α⁻¹ 到 2.22 ppm。同时：
- #7 m_t/m_u 用 DH 作为乘积因子
- #11-12 中微子 M_R 的 α_s=1/(16π) 来自 DH→α⁻¹ 的扩展

**试 (2,6,5)：** DH_alt = 2π³+6π²+5π = 136.94（偏差 −0.07%，看似还行）。但检查 #7 和 #11-12 崩多少。
