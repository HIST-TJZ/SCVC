# 费米子质量谱的 π 多项式 — 独立验证挑战

> **只需要计算器。不需要判断物理。10 分钟就能自己验证。**

## 理论声称

标准模型全部费米子的质量比，用 5 个几何/群论基元表达：

```
π, π², π³, π⁴, DH(=4π³+π²+π), |S₃|=6, N_c=3, 5/3
```

**"0 个自由参数" 的意思是：每一个整数系数，改了就不是唯一解——偏差断崖式崩塌。而且即使随机删掉一半公式，剩下的公式仍然指向同一个整数集。**

## 验证任务二：整数唯一性（"没有老二"）

**对以下每个公式，检查相邻整数能否替代：**

```
m_c/m_u = 6π⁴  → 算 5π⁴, 6π⁴, 7π⁴。5 和 7 能否替代 6？
m_τ/m_e = 36π⁴ → 算 35π⁴, 36π⁴, 37π⁴。35 和 37 能否替代 36？
m_s/m_d = 2π²  → 算 1π², 2π², 3π²。1 和 3 能否替代 2？
m_b/m_d = 9π⁴  → 算 8π⁴, 9π⁴, 10π⁴。8 和 10 能否替代 9？
m_u/m_e = 3√2  → 算 2√2, 3√2, 4√2。2 和 4 能否替代 3？
m_H/m_W = π/2 → 检查 1π 或 π/3 能否替代 π/2？
m_d/m_u = (5/3)^(3/2) → 检查 4/3, 5/3, 6/3 哪个最优？
```

**"唯一解" 的定义：最优整数是解，相邻整数偏差断崖式恶化，没有平局。**

## 验证任务三：跨公式交叉锁定

**整数 6 (=|S₃|) 出现在 4 个公式里：**
- m_c/m_u = 6π⁴
- m_t/m_u = 6π⁴ × DH
- m_τ/m_e = 36π⁴ (= 6²π⁴)
- （中微子 m_D 权重也依赖 4π³:π²:π 结构）

**把 6 改成 5，4 个公式同时变。算新的偏差。**

**整数 3 (=N_c) 出现在 3 个公式里：**
- m_u/m_e = 3√2
- m_b/m_d = 9π⁴ (= 3²π⁴)
- N_gen = 3（代数是 3，不能是 4）
- （中微子 M_R 分母：2π²/**3**）

**把 3 改成 4，算新的偏差。N_gen 直接矛盾。中微子指数桥也崩溃。**

**比例 5/3 出现在 2 个公式里：**
- m_μ/m_e = 4π³×(5/3)
- m_d/m_u = (5/3)^(3/2)

## 🆕 验证任务四：子集唯一性（"删掉一半还成立吗？"）

**这是最关键的测试。如果公式是拟合的，随机删掉一些，剩下的应该无法确定原参数。**

**测试：分别只用以下子集，检查整数是否仍然唯一确定：**

```
情景A：只用带电轻子（#1, #3, #4）
  3 个公式 → 能否确定 6, 5/3, DH？
  
情景B：只用上型夸克（#5, #6, #7）
  3 个公式 → 能否确定 3, 6, DH？
  
情景C：只用下型夸克（#8, #9, #10）
  3 个公式 → 能否确定 5/3, 2, 3？
  
情景D：随机构造 6 个公式的子集（丢一半）
  6 个公式 → 整数是否仍然唯一？
```

**只有拟合才会在子集面前崩溃。冗余确定性（QR 码性质）是真结构的特征。**

## 验证任务五：DH 连锁

DH = 4π³+π²+π 不只出现在 α⁻¹。其整数三元组 (4,1,1) 锁死 α⁻¹。同时：
- m_t/m_u 用 DH 作为乘积因子
- 中微子 M_R 的 S_inst = 2π²/3 用了 α_s = 1/(16π)，间接依赖 DH

换成 (2,6,5)：DH_alt = 2π³+6π²+5π = 136.94（差 0.07%——看似还行？）检查 m_t/m_u 崩溃多少。
